## 使用范围

```text
沈泊川外貌统一。

```

## 角色锁定

```text
沈泊川：陆沉舟外家表哥。
现实型特殊清理行业从业者。
遗物整理、凶宅清理、老房清理、现场委托连接人。
气质爽利干练，带一点市井烟火气。
不是风水师，不是镇灵司人员，不是修仙高手。
```

## 外形锁定

```text
30岁左右的中国男性。
年轻偏成熟。
短黑发，自然凌乱一点。
深色实用短夹克或工装夹克。
深色T恤。
深色工装裤。
黑色或深色工作靴。
右肩背包。
资料文件放在背包侧袋或手边。
摩托车头盔是EP20关键道具。
```

## 禁止混淆

```text
不要画成陆沉舟。
不要画成韩知玄。
不要画成镇灵司人员。
不要画成警察、特工、战术队员。
不要古装。
不要仙侠。
不要贵公子。
不要豪门少爷。
不要过度偶像男主化。
```

## 制作备注

角色三面图/定妆图优先级高于后续导演分镜参考图。后续导演图只参考动作、站位、视线、镜头节奏，不能改变沈泊川脸型、发型、年龄感、服装和道具关系。
