# 《棺中檀色》第13集：阵心入口 — 감독형 연기 지도 이미지 분할표

## 사용 목적

이 문서는 사용자가 제공한 제13화 분진 내용을 바탕으로, 각 분진을 **연기 지도 이미지 생성용 분할 프롬프트**로 재구성한 감독형 제작표다.

- 원본 사건 순서와 대사는 유지한다.
- 각 분진은 **시작 이미지 → 진행 이미지 → 종료 이미지** 중심으로 분할한다.
- 각 분진의 마지막 분할은 다음 분진 첫 이미지와 이어지도록 **자세, 시선, 위치, 광원, 안개, 문동, 지면 균열, 카메라 방향**을 명시한다.
- 실제 생성 시에는 한 분할당 한 장의 연기 지도 이미지를 만들고, 직전 이미지의 마지막 상태를 다음 이미지 참조로 넣는다.
- 총 길이는 사용자가 수정한 기준인 **128초 / 2분 08초**에 맞춘다.

## 고정 지시 적용

이 문서는 반드시 [`FIXED_DIRECTOR_GENERATION_RULES.md`](FIXED_DIRECTOR_GENERATION_RULES.md)의 **角色三面图锁定 + 连续性演技分镜生成规则**을 최상위 규칙으로 따른다.

- 역할 삼면도 / 정장도 우선순위가 가장 높다.
- 감독 참고도는 동작, 카메라, 표정, 리듬만 참고한다.
- 감독 참고도나 배경 참고도의 임시 인물 얼굴·옷은 최종 캐릭터 외형으로 쓰지 않는다.
- 모든 CUT은 이전 CUT 종료 상태와 다음 CUT 연결 상태를 명확히 유지해야 한다.

## 인물 삼면도 절대 준수 — 얼굴·옷 100% 고정

이미지 생성 시 인물이 나오는 모든 분할은 **반드시 제공한 인물 삼면도와 같은 얼굴, 같은 머리, 같은 의상, 같은 신발, 같은 체형 비율**로 만들어야 한다. “비슷한 분위기”가 아니라 **같은 인물**이어야 한다.

```text
角色三视图是最高优先级参考。必须一比一保持三视图中的脸型、五官比例、眼睛形状、发型、发量、服装结构、衣领、袖口、腰带、裤裙长度、鞋子、身体比例。只允许改变动作、表情、视线、光影、湿润程度和镜头角度。禁止换脸，禁止换发型，禁止换衣服，禁止换鞋，禁止改变年龄，禁止改变体型，禁止重新设计角色。
```

실제 입력 순서：

```text
1. 对应人物三视图
2. 上一张连续性参考图
3. 本CUT分割提示词
4. 禁止换脸/换衣/换发型/换鞋/改变体型
```

## 삼면도 일치 필수 운용 — 인물 안 먹히게 하는 방법

이미지 생성할 때는 **반드시 위에 올린 인물 삼면도와 일치하게 만들어야 한다.** 텍스트로만 “陆沉舟”, “白清檀”이라고 쓰면 생성기가 매 컷마다 얼굴과 옷을 새로 해석해서 인물이 바뀐다. 그래서 모든 인물 컷은 아래 순서로 넣는다.

### 1. 참조 이미지 넣는 순서

1. **해당 인물 삼면도 이미지**를 먼저 넣는다.
2. 여러 인물이 나오면 화면에서 중요한 순서대로 넣는다. 예: 陆沉舟 중심 컷이면 `陆沉舟 삼면도 → 白清檀 삼면도 → 韩知玄/黑七 삼면도`.
3. 黑七는 반드시 검은 여우 삼면도를 넣는다. 안 넣으면 사람형 여우요괴나 늑대처럼 바뀔 수 있다.
4. 인물이 작게 나와도 삼면도는 넣는다. 작게 보여도 실루엣과 의상 고정에 필요하다.

### 2. 모든 인물 컷 첫머리에 붙일 고정 문장

```text
严格以随附角色三视图为最高优先级。人物脸型、五官、发型、服装结构、鞋子、身体比例必须与三视图一致。本镜头只允许改变姿势、表情、湿润程度、光影和镜头角度，不允许重新设计人物。
```

한국어 의미:

```text
첨부한 삼면도를 최우선 기준으로 한다. 얼굴형, 이목구비, 헤어, 의상 구조, 신발, 체형 비율은 삼면도와 반드시 일치해야 한다. 이번 컷에서는 자세, 표정, 젖은 정도, 조명, 카메라 각도만 바꾼다. 인물을 새로 디자인하지 않는다.
```

### 3. 캐릭터별 “절대 유지 / 절대 금지” 압축문

#### 陆沉舟

```text
陆沉舟必须与三视图一致：黑色短款皮夹克、黑T、深色长裤、黑色军靴、黑色微乱短发、冷静坚定眼神。禁止长风衣、古装、甲胄、修仙服、换脸、换发型。
```

#### 白清檀

```text
白清檀必须与三视图一致：长黑发、黑色旗袍式长裙或黑色长裙、黑色低跟鞋、清冷疲惫的脸。禁止白衣、广袖古装、女鬼化、半透明、换成幽魂。
```

#### 韩知玄

```text
韩知玄必须与三视图一致：黑色短发、黑衬衫、黑色长款镇灵司制服外套、黑裤、黑靴、胸前镇灵司标识。禁止轻甲、铠甲、修仙服、改掉标识。
```

#### 黑七

```text
黑七必须与三视图一致：黑狐本体、四足、橙黄色狐瞳、黑色湿毛、蓬松狐尾、尖耳。禁止人形、狼人、狐妖青年、直立成人。
```

### 4. 실제 한 컷 프롬프트 조립 순서

```text
[1. 삼면도 최우선 고정문]
严格以随附角色三视图为最高优先级……不允许重新设计人物。

[2. 해당 인물 압축문]
예: 陆沉舟必须与三视图一致……禁止长风衣……

[3. 연결성 고정문]
上一张图的最后姿势就是本张图的开始姿势……

[4. 이번 분할 연기 지도]
표에 있는 해당 분할 내용을 붙인다.

[5. 금지]
不要换脸，不要换衣服，不要改变发型，不要改变人物年龄，不要重新设计角色。
```

### 5. 그래도 인물이 안 맞을 때

- 프롬프트를 길게 쓰지 말고 **인물 고정문 + 이번 동작 한 줄**만 남긴다.
- 얼굴이 중요한 컷은 얼굴 클로즈업 삼면도를 참조로 더 강하게 둔다.
- 옷이 틀어지는 컷은 정면/측면/후면 전신 삼면도를 같이 넣는다.
- 여러 인물이 섞여 얼굴이 바뀌면 한 장에 모두 넣지 말고 **주인공 1명 중심 + 나머지는 흐린 실루엣**으로 만든다.
- 이전 컷 이미지와 삼면도를 동시에 넣는다. 이전 컷은 연결용, 삼면도는 인물 고정용이다.

## 镜1｜0–7秒｜全景下移运镜｜崖台继续下沉，门洞幽光翻涌

| 분할 | 시간 | 연기 지도 이미지 | 종료 연결점 |
|---|---:|---|---|
| 1-1 | 0-2s | 全景下移开始. 断崖台面已经开裂，镜头从上方向下缓慢压低。砂砾随震动滚向崖边，漆黑门洞在裂缝中心半开。 | 崖台还在下沉，裂缝向四周继续扩张。 |
| 1-2 | 2-5s | 门洞内幽光流转，阴冷黑雾向外吐出。狂风卷动水雾，但人物只做远处小剪影，重点是空间压迫感。 | 黑雾从门洞边缘向外翻涌，风压推向众人。 |
| 1-3 | 5-7s | 镜头继续下移，末日感加重。裂缝、碎石、门洞幽光形成画面中心。 | 下一镜第一张图中，陆沉舟从众人前方稳步走向门洞。 |

**핵심 프롬프트**

```text
断崖台面持续下沉，全景下移运镜，地面裂缝扩张，砂砾滚落崖边，漆黑门洞内幽光流转，阴冷黑雾向外翻涌，狂风卷动水雾，压抑末日感，无台词。
```

**다음 분진 연결**: 镜1 마지막의 문洞 위치와 균열 방향을 그대로 유지하고, 카메라는 陆沉舟 뒤/측면 중경跟拍으로 좁힌다.

## 镜2｜7–14秒｜中景跟拍｜陆沉舟走向门洞

| 분할 | 시간 | 연기 지도 이미지 | 종료 연결점 |
|---|---:|---|---|
| 2-1 | 7-9s | 陆沉舟 중경跟拍. 그는 검은 군화로 갈라진崖台를 밟으며漆黑门洞 쪽으로 걷는다. 손에는漆黑锁链. 걸음은 흔들림 없이 평온하다. | 시선은 문洞에 고정, 발걸음은 멈추지 않는다. |
| 2-2 | 9-12s | 대사 전반: “门外全是障眼法……” 주변은 흔들리고 갈라지지만, 陆沉舟의 호흡과 어깨는 평온하다. | 쇠사슬이 손안에서 아주 낮게 흔들린다. |
| 2-3 | 12-14s | 대사 후반: “真正的阵局核心，就在这道门之后。” 그는 말하면서도 보폭을 유지한다. | 다음 镜3 첫 이미지에서 白清檀은 이 등을 바라보다가 멈춰 선다. |

**핵심 프롬프트**

```text
陆沉舟中景跟拍，黑色皮夹克黑T深色长裤黑军靴，手握漆黑锁链，步伐沉稳走向漆黑门洞，衣摆随风猎猎，周围崩裂震动而他平静如常，视线锁定门洞。
```

**다음 분진 연결**: 陆沉舟의 등과 문洞 방향 시선을 白清檀의 시선으로 이어받는다.

## 镜3｜14–22秒｜近景侧推｜白清檀担忧劝阻

| 분할 | 시간 | 연기 지도 이미지 | 종료 연결점 |
|---|---:|---|---|
| 3-1 | 14-16s | 白清檀 근경侧推. 그녀는 陆沉舟의 말을 듣고 멈춰 선다. 눈은 문洞과 陆沉舟 등을 번갈아 본다. | 손이 검은 치마자락을 잡기 시작한다. |
| 3-2 | 16-19s | 双手가 치마자락을 꽉 쥔다. 指节이 희게 질린다. 발은 한 번 들렸다가 다시 내려온다. | 입술이 열리고 欲言又止가 끝난다. |
| 3-3 | 19-22s | 대사: “你明知道里面杀机四伏，进去就再也回不来，何必非要冒这种险？” 눈빛은 그의 등을 따라 움직인다. | 다음 镜4 첫 이미지에서 韩知玄이 이 걱정을 이어받아 앞으로 뛰어나온다. |

**핵심 프롬프트**

```text
白清檀近景侧推，黑色旗袍式长裙，长黑发被风雾吹动，双手攥紧衣摆，指节泛白，眼神紧盯陆沉舟背影与漆黑门洞，纠结担忧，欲言又止后轻声劝阻。
```

**다음 분진 연결**: 白清檀의 말끝과 걱정스러운 시선 앞으로 韩知玄이 빠르게 들어와 길을 막는다.

## 镜4｜22–31秒｜中景正反切｜韩知玄张臂拦路

**마지막 연결**: 화면을 닫힌门洞의 암흑과 黑七의 복잡한 눈빛에 오래 머물게 두어 다음 화의 阵心 내부 장면으로 연결할 여지를 남긴다.

## 실제 이미지 생성 순서

1. 한 镜 전체를 한 번에 만들지 말고, 표의 **분할 하나당 이미지 한 장**으로 만든다.
2. 예: `镜1-1 → 镜1-2 → 镜1-3 → 镜2-1` 순서로 생성한다.
3. 매번 직전 결과 이미지를 다음 분할의 참조 이미지로 넣는다.
4. 각 프롬프트 첫 줄에는 공통 연결 문장을 반드시 넣는다.
5. 인물이 나오면 해당 인물 삼면도와 캐릭터 고정 문장을 함께 넣는다.
6. 한 이미지에 사람이 많아지면 **주연 1명 선명 + 나머지는 배경 실루엣**으로 단순화한다.
7. 문洞, 균열, 세 눈, 바람 방향은 컷마다 절대 재배치하지 않는다.
