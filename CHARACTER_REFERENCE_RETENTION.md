# 角色三面图 / 定妆参考保留规则

## 固定原则

即使删除某一集分镜资料、导演演技分割稿、分진文档或临时制作稿，也不得删除角色三面图、角色定妆图、服装参考与角色参考说明。

角色参考资料属于全项目长期资产，不属于单集分镜资料。

## 删除范围区分

### 可以按用户要求删除

- 单集分镜资料。
- 单集导演演技分割稿。
- 单集 AI 视频提示词草稿。
- 临时 CUT 分割文档。
- 已废弃的剧集制作稿。

### 不得随分镜一起删除

- 角色三面图。
- 角色定妆图。
- 角色服装参考。
- 角色外形锁定说明。
- 角色声音 / 气质 / 禁止变形规则。
- `character_reference_*.md` 类型文件。
- `FIXED_DIRECTOR_GENERATION_RULES.md` 中的角色三面图锁定规则。

## 当前必须保留的角色参考

- 陆沉舟三面图 / 定妆参考。
- 白清檀三面图 / 定妆参考。
- 韩知玄三面图 / 定妆参考。

- 檀缺三面图 / 定妆参考。
- 檀缺换装后深色日常装参考：`character_reference_tanque.md`。
- 后续新增角色的三面图 / 定妆参考。

## 执行规则

当用户说“删除第几集”“删除分镜”“删除分진”“不用这集资料”时，只删除对应剧集制作文档。

除非用户明确说“删除某个角色三面图 / 删除某个角色参考”，否则不得删除任何角色参考文件或角色锁定规则。

如果删除操作可能影响角色参考，必须先保留角色参考，再删除剧集资料。
