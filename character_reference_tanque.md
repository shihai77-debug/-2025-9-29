# 角色定妆｜檀缺｜出棺后深色日常装

## 使用范围

此文件记录用户最新提供的檀缺服装参考。后续如果剧情发生在第十六集服装店换装之后，檀缺优先使用本套深色日常装。

- 第十五集：檀缺刚出棺，仍使用白衣出棺造型。
- 第十六集 CUT-01～CUT-11店内换装前：檀缺仍使用白衣出棺造型。
- 第十六集 CUT-11店内画外音“……行，就这件吧。”之后，以及第十七集开始：檀缺使用本文件记录的深色日常装。

## 外形细节

| 部位 | 必须保持 |
|---|---|
| 脸 | 年轻女性，小脸，苍白柔和肤色，灰褐色眼睛，清冷但比白衣出棺状态更有活人气。 |
| 发型 | 长黑发，半披半束，头顶自然蓬松，后背长发厚重微卷，额前与脸侧有碎发。 |
| 上衣内搭 | 深灰黑色圆领针织/打底上衣，贴合但不紧绷，领口简洁。 |
| 外套 | 深灰褐色宽松针织开衫，长袖，前襟自然垂落，有低调纽扣，不能换成白外衫或现代西装。 |
| 下装 | 黑色中长款百褶裙，裙摆到小腿附近，褶皱清楚，不能换成短裙、裤装或白色长裙。 |
| 鞋 | 黑色低跟皮鞋 / 系带皮鞋，简洁日常，不能换成浅灰白短靴、赤脚、高跟鞋。 |
| 色调 | 整体深灰、黑、深褐灰，低调日常，适合“出门没人多看的那种”。 |
| 气质 | 刚出棺后学习日常生活，认真、迟钝、嘴硬，但不装傻；比白衣造型更融入街道。 |
