# Self Update Protocol

Self update does not mean the AI may freely change canon.

```text
Self update = review real results, record repeated success/failure, propose candidates, then promote only approved rules.
```

## Required Review Files

After each episode or major generated result, create:

```text
episodes/epXX/review/epXX_story_logic_check.md
episodes/epXX/review/epXX_prompt_result_review.md
episodes/epXX/review/epXX_update_candidates.md
```

## Can Be Updated After Evidence

```text
three-second hook patterns
CUT length guidelines
dialogue rhythm and breath guidelines
storyboard table format
video prompt failure-prevention wording
CapCut timeline table format
subtitle timing rules
regen/edit decision criteria
recurring failure blacklist
```

## Must Not Auto-Update

```text
worldview core rules
character identity, age, appearance, personality, voice
relationships
contract / regulation / prohibition rules
confirmed episode events
major prop reveal order
death, survival, identity, and major reversal facts
```

These require explicit user approval before being changed.

## Promotion Rule

```text
review result -> update candidate -> user approval -> production_bible update
```

Never promote `update_candidates` directly into official rules without approval.
