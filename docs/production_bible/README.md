# Production Bible

This folder is the high-level production control layer for `《棺中檀色》`.

It does not replace the existing repository rules in `AGENTS.md`, root setting files, or `outputs/`.
It organizes how future novel, storyboard, prompt, video, and CapCut work should read those rules before writing.

Core rule:

```text
Existing contract / regulation / worldview / character rules win.
The story-production-kit files are supporting style and quality-control layers.
Reference video analysis can improve pacing, hooks, dialogue rhythm, and visual grammar,
but it must never overwrite canon, character voice, relationships, or reveal order.
```
