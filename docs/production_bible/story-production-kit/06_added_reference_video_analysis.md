# Added Reference Video Analysis

목적: 추가 영상 6개를 분석하여 기존 스타일 바이블에 보강할 기준을 정리한다.

사용자 취향 메모:

```text
계속 보게 만들고 다음화가 보고 싶어야 한다.
스토리에 매력이 있어야 한다.
다만 지금 샘플들은 조금 더 강하게 끌렸으면 한다.
판타지 부분은 잘하는 편이 좋다.
대사는 제대로 알아듣고 무슨 말인지 명확해야 한다.
인물은 적당한 타이밍에 들어와야 한다.
```

## 1. 추가 영상 전체 판단

이 정도면 기준 자료로 충분하다.

현재까지 본 영상들은 다음 기준을 잡기에 충분하다.

```text
초반 훅
현대 공간 + 판타지 침입
공사장/사무실/가정집/밤길 전환
명확한 대사 리듬
캐릭터 진입 타이밍
소품 중심 사건
다음화 유도 방식
```

부족한 부분도 명확하다.

```text
다음화 유도가 약한 샘플은 마지막에 새 질문이 부족하다.
중간에 장소 전환이 많으면 시청자가 핵심 사건을 놓친다.
판타지 액션은 좋지만, 주인공 감정 또는 대가가 없으면 끌림이 약해진다.
```

## 2. 영상별 분석

### D_00fb098

핵심 구조:

```text
밤 공사장
자전거/현대 인물
요괴 또는 동물형 존재
협상/조건 대사
전화 또는 주문형 요청
사무실 권력자 장면으로 전환
```

좋은 점:

```text
밤 공사장 분위기가 즉시 질문을 만든다.
대사가 비교적 명확하고 목적성이 있다.
현대 소품인 휴대폰이 판타지 요청과 연결된다.
인물이 단계적으로 들어와서 혼잡하지 않다.
```

아쉬운 점:

```text
공사장 사건과 사무실 장면의 감정 연결이 더 강해야 한다.
마지막 질문이 "그래서 다음에 무슨 일이 터지나"까지 밀어붙이지 못하면 끌림이 약해진다.
```

적용 규칙:

```text
외부 요청/전화/문자 같은 현대 장치를 판타지 사건의 도화선으로 쓴다.
하지만 마지막에는 반드시 구체적인 대가나 다음 행동을 남긴다.
```

### E_26b423 / G_198e922

두 영상은 프레임 흐름상 사실상 같은 내용으로 보인다.

핵심 구조:

```text
공사장 권력자 등장
노란 옷의 법사/스승 등장
차량, 수행자, 군중
의식 준비
액션/달리기/주술
법사 쓰러짐
```

좋은 점:

```text
인물 진입 타이밍이 명확하다.
공사장 공간과 판타지 인물이 시각적으로 잘 대비된다.
법사 의상과 의식 소품이 기억에 남는다.
대사 목적이 비교적 선명하다.
```

아쉬운 점:

```text
액션은 강하지만 "다음화를 꼭 봐야 하는 정보"가 마지막에 부족할 수 있다.
법사가 쓰러지는 것만으로는 질문이 약하다.
왜 쓰러졌는지, 누가 조작했는지, 다음 피해자가 누구인지가 더 필요하다.
```

적용 규칙:

```text
주술/판타지 액션 뒤에는 반드시 원인 미해결 질문을 둔다.
예: "누가 의식을 망쳤나?", "봉인 안의 존재가 누구인가?", "다음 타깃은 누구인가?"
```

```text
법사/의식 이후

주인공 계열 남자 등장
밤 장면

부상자 또는 죽음
숨겨진 정체/과거 암시
```

좋은 점:

```text
판타지 효과가 사건과 연결된다.
현대 공사장 사건에서 밤의 결계 장면으로 확장된다.
인물의 능력과 과거를 암시해 다음 이야기를 만든다.
```

아쉬운 점:

```text
중간 정보가 많아지면 초점이 흐릴 수 있다.
주인공이 무엇을 선택해야 하는지 더 빨리 잡아줘야 한다.
```

적용 규칙:

```text
능력 공개는 좋지만, 능력보다 선택을 먼저 세운다.
예: "구할 것인가 숨길 것인가", "자료를 열 것인가 감출 것인가".
```

핵심 구조:

```text
갑옷 입은 인물의 실내 등장
동물/봉인/초자연 눈빛
사무실 권력 대화

공사장 하청/노동자 사건
자전거/현장 이동

```

좋은 점:

```text
현대 일상과 판타지 설정의 섞임이 좋다.
인물들이 각각 기능을 갖고 들어온다.
대사로 목적을 빠르게 전달한다.
검은 연기 같은 시각 훅이 강하다.
```

아쉬운 점:

```text
여러 갈래가 동시에 열리면 중심 사건이 약해진다.
초반에 주인공 목표를 더 명확하게 잡아야 한다.
```

적용 규칙:

```text
한 회차에 사건 가지는 2개 이하로 제한한다.
주사건 1개 + 보조사건 1개까지만 둔다.
```

### I_ccb6dbe

핵심 구조:

```text
말하는 동물 클로즈업
감정 대사
봉인 지도/문서
대문 앞 만남
가정집 식사
가족 갈등
무기 또는 갑옷 소품
동물의 행동으로 끝
```

좋은 점:

```text
동물 캐릭터가 감정을 강하게 끌어온다.
대사가 이해하기 쉽고 표정이 잘 받쳐준다.
가정집 장면이 판타지 사건에 인간 감정을 붙인다.
마지막 동물 액션이 다음 장면을 궁금하게 만든다.
```

아쉬운 점:

```text
귀여운 동물 감성, 가족극, 판타지가 섞이기 때문에 톤이 흔들릴 수 있다.
너무 오래 가족 대화로 가면 판타지 사건의 긴장이 줄어든다.
```

적용 규칙:

```text
감정 장면은 좋지만, 반드시 핵심 판타지 소품 또는 위험과 연결한다.
가족/일상 대화는 1개의 폭로 또는 1개의 선택으로 끝낸다.
```

## 3. 이번 추가 분석으로 확정된 취향 규칙

### 대사 규칙

사용자는 대사가 분명히 들리고 이해되는 것을 선호한다.

규칙:

```text
한 대사에 정보 하나만 넣는다.
대사 목적을 명확히 한다.
대사 뒤에는 표정 또는 행동 반응을 둔다.
설정 설명보다 명령, 경고, 거래, 선택 압박을 우선한다.
```

좋은 대사 기능:

```text
이걸 하지 마라
지금 당장 가라
이 물건을 열지 마라
이름을 말하지 마라
네가 찾는 사람은 여기 없다
이 일은 돈으로 끝나지 않는다
```

### 인물 진입 규칙

사용자는 인물이 적당할 때 들어오는 구성을 선호한다.

규칙:

```text
첫 10초 안에 인물을 너무 많이 넣지 않는다.
핵심 인물은 사건 질문이 생긴 뒤 들어온다.
새 인물은 등장 즉시 기능을 가져야 한다.
인물 등장 이유가 소리, 소품, 전화, 문, 시선으로 연결되어야 한다.
```

### 판타지 처리 규칙

판타지는 강해도 되지만 사건과 연결되어야 한다.

규칙:

```text
판타지 효과는 하나의 사건 원인 또는 결과여야 한다.
효과만 멋있고 선택/대가가 없으면 약하다.
주술/결계/봉인은 반드시 누군가의 선택을 압박해야 한다.
```

### 다음화 유도 강화 규칙

현재 샘플보다 더 끌리게 하려면 마지막에 새 질문을 더 선명하게 남긴다.

약한 끝:

```text
인물이 쓰러짐
연기가 남음
풍경으로 끝남
```

강한 끝:

```text
인물이 쓰러졌는데 손바닥에 다른 사람 이름이 나타남
연기 속에서 익숙한 목소리가 들림
닫힌 가방 안에서 전화가 울림
봉인 지도에서 다음 장소가 스스로 타오름
누군가 "그 사람은 이미 죽었다"라고 말하고 컷아웃
```

## 4. 네 작품에 적용할 강화 방향

현재 네 작품에는 아래 방식이 잘 맞는다.

```text
C의 감성
+ D/H의 현대 장치 연결
+ E/G/F의 공사장식 사건 추진
+ I의 감정이 있는 비인간 캐릭터 활용
```

단, 톤이 섞이지 않게 아래를 지킨다.

```text
한 회차의 중심 감정은 하나
한 회차의 중심 소품은 하나
판타지 효과는 하나의 원인/결과만 담당
마지막 컷은 반드시 다음 행동 직전에 끊음
```

## 5. 소설 작성에 반영할 문장 규칙

소설 단계에서 이미 아래를 넣어야 분경이 강해진다.

```text
첫 문단: 이상한 행동 또는 장소
두 번째 문단: 인물의 짧은 목적
세 번째 문단: 핵심 소품 또는 금지사항
중반: 새 인물은 소리/전화/문/시선으로 들어옴
후반: 대가의 조짐
마지막 문장: 답이 아니라 질문
```

마지막 문장 예시:

```text
背包里，那只早已关机的手机忽然亮了。
地图上的第二个红点，落在了陆沉舟的名字上。
黑七忽然抬头，像是听见了一个死人在叫他。
沈泊川还没来得及开口，韩知玄的镇灵司令牌先裂了一道缝。
```
