# Novel Before Storyboard Template

목적: 분경을 만들기 전에 먼저 사용할 소설/장면 원문 템플릿이다. 이 단계에서는 아직 컷을 나누지 않는다. 먼저 이야기, 대사, 감정, 소품의 이유를 잡는다.

## 1. 장면 기본 정보

```text
작품명:
회차:
장면 번호:
장면 제목:
장면 길이 목표:
영상 비율:
장면 톤:
참고 스타일:
```

예시:

```text
작품명: 棺中檀色
회차: 20화
장면 번호: SCENE-01
장면 제목: 昆仑结界 이후의 정적
장면 길이 목표: 60~90초
영상 비율: 16:9
장면 톤: 죽은 공기, 침묵, 외부 사건의 침입
참고 스타일: C의 감성 + A의 미스터리
```

## 2. 이 장면의 한 문장 사건

```text
이 장면에서 실제로 일어나는 일:
```

예시:

```text
陆沉舟가 昆仑结界가 사라진 방향을 바라보며 침묵하고, 沈泊川의 오토바이 진입으로 외부 사건이 현장에 들어온다.
```

## 3. 장면의 질문

시청자가 이 장면에서 가져야 할 질문을 적는다.

```text
초반 질문:
중반 질문:
끝 질문:
```

예시:

```text
초반 질문: 왜 아무도 움직이지 않는가?
중반 질문: SHEN은 무엇을 알고 온 것인가?
끝 질문: 닫힌 가방 안에는 무엇이 있고, 왜 지금 꺼내지 않는가?
```

## 4. 등장인물 상태

```text
캐릭터:
현재 위치:
현재 감정:
말하고 싶은 것:
말하지 않는 것:
겉행동:
속마음:
절대 하면 안 되는 행동:
```

예시:

```text
캐릭터: LU
현재 위치: 왼쪽 전경, 앉아 있음
현재 감정: silent grief
말하고 싶은 것: 아무것도 없음
말하지 않는 것: 결계가 사라진 순간 본 것
겉행동: 먼 곳을 봄
속마음: 아직 현실을 받아들이지 못함
절대 하면 안 되는 행동: 즉시 크게 울거나 설명하지 않음
```

## 5. 핵심 소품

```text
소품명:
누가 가지고 있는가:
처음 보이는 시점:
보이면 안 되는 시점:
이 소품이 만드는 질문:
이 소품이 다음 장면으로 넘기는 기능:
```

예시:

```text
소품명: 닫힌 背包
누가 가지고 있는가: SHEN
처음 보이는 시점: SHEN이 오토바이를 멈춘 뒤
보이면 안 되는 시점: 01-1, 01-2 안의 资料
이 소품이 만드는 질문: 안에 무엇이 들어 있는가?
이 소품이 다음 장면으로 넘기는 기능: 자료 공개 직전의 긴장 유지
```

## 6. 원문 대사 보존 구역

중요 대사는 여기에 먼저 고정한다. 분경 단계에서 의미를 바꾸지 않는다.

```text
원문 대사 1:
발화자:
반드시 유지할 의미:
줄여도 되는 범위:
절대 바꾸면 안 되는 표현:
```

예시:

```text
원문 대사 1: “我不是来打扰你的。”
발화자: SHEN
반드시 유지할 의미: SHEN은 끼어들고 싶지 않지만 어쩔 수 없이 왔다.
줄여도 되는 범위: “我不是来打扰。”
절대 바꾸면 안 되는 표현: 책임을 회피하는 말로 바꾸면 안 됨
```

## 7. 소설 원문 작성 구역

아래는 실제 소설처럼 쓴다. 아직 컷을 나누지 않는다.

```text
[장면 원문 시작]


[장면 원문 끝]
```

작성 원칙:

```text
1. 화면에 보이는 행동을 중심으로 쓴다.
2. 대사는 기능에 맞게 둔다. 짧음보다 이해, 감정, 행동 변화가 우선이다.
3. 감정 설명보다 시선, 손, 침묵, 거리감으로 표현한다.
4. 핵심 소품은 너무 빨리 공개하지 않는다.
5. 장면 끝은 답을 공개하지 말고 질문을 남긴다.
```

## 8. 분경 전 검수

분경으로 넘기기 전에 체크한다.

```text
- 이 장면의 사건이 한 문장으로 말해지는가?
- 대사가 너무 길지 않은가?
- 인물 위치가 논리적으로 맞는가?
- 핵심 소품이 너무 빨리 공개되지 않는가?
- 다음 장면으로 넘어갈 질문이 남는가?
- 새 인물/장소/소품을 임의로 추가하지 않았는가?
```
