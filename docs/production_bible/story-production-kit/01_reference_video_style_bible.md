# Reference Video Style Bible

목적: 사용자가 좋아하는 AI 숏폼 드라마 샘플 3개의 공통 문법을 정리하여, 이후 소설 작성, 분경, 영상 프롬프트에 반복 적용한다.

이 문서는 원문보다 위에 있지 않다. 실제 제작 우선순위는 다음과 같다.

```text
1. 작품 원문 / 대사
2. 캐릭터 카드 / 장소 카드 / 소품 카드
3. 이 스타일 바이블
4. 영상 도구별 프롬프트 조건
```

## 1. 전체 취향 방향

좋은 샘플들의 공통점:

```text
현실적인 장소
+ 비현실적인 존재 또는 사건
+ 짧은 대사 충돌
+ 핵심 소품
+ 감정 정지
+ 해결 직전 종료
```

좋은 첫 장면은 설명하지 않는다. 바로 이상한 이미지나 행동을 보여준다.

```text
왜 이 장소에 저 인물이 있지?
왜 저 소품이 중요하지?
왜 죽은 사람/요괴/신적 존재가 지금 나오지?
```

시청자가 첫 3~5초 안에 질문을 갖게 해야 한다.

## 2. 세 샘플의 기능 분류

### A 샘플: 미스터리 거래 / 위협 구조

사용할 곳:

```text
수상한 거래
금지된 물건
돈/조건/협박
신적 존재 또는 봉인된 힘
밤에 시작되는 대가
```

분경 특징:

```text
낡은 실내 또는 좁은 장소
강한 인물과 약해 보이는 인물의 대화
조건 제시
큰 존재의 이미지 삽입
밤 장면으로 전환
괴이한 눈빛 또는 기척으로 종료
```

### B 샘플: 자극적 사건 추진 / 현대 현실 충돌

사용할 곳:

```text
현대 공간에서 괴이한 사건 발생
소품 또는 시체 발견
군중 반응
성급한 처리
뒤늦은 대가 암시
복수/저주/재앙 예고
```

분경 특징:

```text
말하는 동물/요괴 같은 강한 훅
공사장/거리 같은 현실 장소
핵심 소품을 화면 중앙에 둠
명령형 대사
불태우기/파괴/처리 같은 즉각 행동
보스 또는 권위자의 분노로 종료
```

### C 샘플: 감성 / 상실 / 죽은 사람과의 연결

사용할 곳:

```text
상실감
죽은 공기
가족/동료/과거 인연
말하지 못한 후회
영혼 또는 기억의 등장
조용한 눈물 대신 멈춤과 시선
```

분경 특징:

```text
빈 집 또는 조용한 공간
작은 준비 행동
밤의 묘지 또는 기억의 장소
죽은 사람에게 말을 거는 행동
영혼/기억/잔상이 등장
대사를 크게 울리지 않고 조용히 처리
큰 주제 문장 또는 미해결 감정으로 종료
```

## 3. 네 작품에 적용할 기본 조합

추천 조합:

```text
C의 정서
+ A의 미스터리
+ B의 사건 추진력
```

즉, 네 작품은 다음 방식이 잘 맞는다.

```text
죽은 공기 또는 침묵으로 시작
외부 사건 또는 소품이 침입
인물들이 크게 설명하지 않고 반응
핵심 정보는 전부 보여주지 않음
다음 컷/다음 화에서 대가가 시작될 것처럼 끝냄
```

## 4. 분경 기본 구조

각 장면은 아래 순서로 설계한다.

```text
HOOK: 이상하지만 이해 가능한 첫 이미지
SETUP: 누가 어디서 무엇을 하려는지
OBJECT: 핵심 소품/존재 등장
CONFLICT: 짧은 대사 충돌
REACTION: 얼굴, 시선, 손동작, 침묵
TURN: 소품/행동의 진짜 의미 드러남
CLIFF: 대가, 비밀, 침입, 각성 직전에 종료
```

## 5. 대사 방식

대사는 짧고 기능이 분명해야 한다.

좋은 대사 기능:

```text
명령한다
거절한다
값을 부른다
경고한다
비밀을 조금만 밝힌다
상대의 상처를 찌른다
뒤늦게 정체를 알아차린다
```

피할 것:

```text
설정 설명이 긴 대사
감정을 직접 설명하는 대사
한 줄 안에 정보가 너무 많은 대사
이미 화면에 보이는 것을 다시 말하는 대사
```

원칙:

```text
한 대사 = 한 기능
한 컷 = 한 감정 변화
긴 설명 = 소품/행동/시선으로 전환
```

## 6. 감성 방식

감정은 말보다 행동으로 보여준다.

좋은 감정 표현:

```text
혼자 장소를 준비함
상대가 없는 자리를 바라봄
대답하기 전에 멈춤
손에 든 물건을 놓지 못함
울기보다 숨을 삼킴
고개를 돌리지 못함
```

피할 것:

```text
처음부터 통곡
감정을 직접 설명하는 자막
모든 인물이 동시에 크게 반응
과한 판타지 이펙트로 감정을 덮음
```

## 7. 다음화 유도 방식

회차 끝은 문제를 해결하지 않는다. 더 큰 결과 직전에 멈춘다.

좋은 종료 방식:

```text
누군가 문 앞에 도착한다
닫힌 가방/상자/자료를 열기 직전 멈춘다
죽은 사람의 목소리가 들린다
봉인이 흔들리지만 완전히 깨지지 않는다
상대가 진짜 이름을 부르기 직전 끊는다
주인공이 반응하기 직전 컷아웃
```

나쁜 종료 방식:

```text
모든 정보를 설명하고 끝냄
갈등이 완전히 해결됨
다음 사건의 질문이 없음
마지막 컷이 단순 풍경만 남음
```

## 8. 프롬프트로 옮길 때의 핵심 문장

```text
Do not explain the emotion directly. Show it through stillness, gaze, hand movement, object placement, and delayed response.
```

```text
Keep the scene grounded in a realistic location, then introduce one abnormal element as the narrative hook.
```

```text
End before the answer is revealed. Preserve the consequence, secret, or intrusion for the next beat.
```

## 9. 추가 샘플 6개 반영 메모

추가 영상 분석에서 확인된 사용자 취향:

```text
대사는 분명해야 한다.
인물은 사건 질문이 생긴 뒤 적당한 타이밍에 들어와야 한다.
판타지는 강해도 되지만 사건의 원인/결과와 연결되어야 한다.
현재 샘플들은 계속 보게 하는 힘은 있으나, 다음화 유도는 더 강해질 필요가 있다.
```

강화 규칙:

```text
첫 10초 안에 인물을 너무 많이 넣지 않는다.
한 회차의 중심 소품은 하나로 제한한다.
주사건 1개 + 보조사건 1개까지만 둔다.
판타지 효과 뒤에는 반드시 선택, 대가, 비밀 중 하나를 남긴다.
마지막 컷은 단순 붕괴/연기/풍경이 아니라 새 질문으로 끝낸다.
```

좋은 마지막 질문:

```text
누가 의식을 망쳤나?
봉인 안의 존재는 누구인가?
다음 타깃은 누구인가?
왜 닫힌 물건에서 소리가 나는가?
왜 죽은 사람의 이름이 지금 나타나는가?
```

## 10. 전컷 연결과 대사 표현력 보강

중요 수정:

```text
"대사를 짧게"는 대사를 삭제하라는 뜻이 아니다.
영상에서 이해 가능한 호흡으로 나누되, 캐릭터 말투와 감정 표현력은 유지한다.
```

스토리 연결 원칙:

```text
전 컷 마지막 대사/행동/소리/소품은 다음 컷 첫 이미지나 첫 반응으로 반드시 이어진다.
이 연결이 없으면 숏폼이 아니라 장면 나열처럼 보인다.
```

대사 원칙:

```text
명령과 제지는 짧게.
감정과 관계 대사는 중간 길이를 허용.
세계관 정보와 의뢰 설명은 대사 + 자료 + 반응 컷으로 분할.
캐릭터 말투는 캐릭터 카드가 최우선.
```
