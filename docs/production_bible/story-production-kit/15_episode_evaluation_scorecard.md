# Episode Evaluation Scorecard

목적: 소설, 분경, 영상 결과를 감으로만 판단하지 않고 점수로 검수한다.

## 1. 기본 점수표

각 항목은 1~5점으로 평가한다.

```text
회차:
장면:
버전:

첫 3초 훅:
전컷 연결:
대사 명확성:
대사 재미:
캐릭터 말투 일관성:
감정 몰입:
스토리 정보 전달:
판타지/미스터리 매력:
조연 기능성:
소품 활용:
카메라/분경 명확성:
다음화 궁금증:
영상화 가능성:
총평:
```

## 2. 통과 기준

```text
첫 3초 훅: 4점 이상
캐릭터 말투 일관성: 4점 이상
다음화 궁금증: 4점 이상
영상화 가능성: 4점 이상
```

아래 항목 중 하나라도 2점 이하면 수정한다.

```text
캐릭터 말투 일관성
전컷 연결
스토리 정보 전달
다음화 궁금증
```

## 3. 수정 판단

```text
1~2점: 반드시 수정
3점: 보강 후보
4점: 사용 가능
5점: 재사용 규칙 후보
```

## 4. 점수 후 행동

```text
좋은 장면: success_patterns에 기록
실패 장면: failure_blacklist에 기록
애매한 장면: A/B 테스트 후보
```
