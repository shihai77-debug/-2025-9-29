# Novel AI Short Drama Gold Standard

목적: 소설 자체의 재미와 AI 숏폼 영상화 가능성을 동시에 만족시키는 최종 품질 기준을 고정한다.

핵심:

```text
구체적인 디테일로 감정 있는 이야기를 한다.
세계관은 사건 속에서 드러난다.
인물은 항상 C位다.
소설은 읽혀야 하고, 동시에 화면으로 옮길 수 있어야 한다.
```

## 1. 최하층 원칙

좋은 장면의 본질:

```text
인물의 감정
인물의 선택
인물의 반차
그 감정을 보여주는 구체적 행동
```

관객은 설정을 외우지 않는다.

```text
기억되는 것은 陆沉舟의 손끝이 멈추는 순간,
黑七이 입으로는 비꼬면서 옷자락을 잡아당기는 순간,
沈泊川이 겁을 내면서도 도망가지 않는 순간이다.
```

## 2. 인물: 표면 라벨 + 내면 모순

캐릭터는 형용사로 만들지 않는다.

나쁜 방식:

```text
黑七은 음험하고 말 많고 욕심이 많다.
沈泊川은 겁이 많다.
陆沉舟는 차갑다.
```

좋은 방식:

```text
黑七은 입으로 비꼬지만 위험할 때 가장 먼저 주인공 옷자락을 잡아당긴다.
沈泊川은 겁이 많지만 의뢰를 버리고 도망가지는 않는다.
陆沉舟는 말이 적지만 위험 앞에서는 가장 먼저 손을 뻗어 막는다.
```

원칙:

```text
인물이 하는 말과 행동은 항상 바닥 동기에서 나와야 한다.
멋있어 보이려고 과묵하면 안 된다.
웃기려고 겁쟁이로 만들면 안 된다.
독설하려고 檀缺을 세우면 안 된다.
```

## 3. 정보: 치약 짜듯 방출

세계관, 阵法, 族群, 无相会, 气, 局 같은 정보는 한 번에 설명하지 않는다.

방식:

```text
보이는 이상함
상식과 맞지 않는 점
인물의 감각 반응
주인공의 단계적 판단
작은 반전
다음 질문
```

예:

```text
凶宅 = 有鬼 = 害人
처럼 시작해도,
符痕对称 / 没有杀气 / 气往里吸
를 거쳐
凶宅 = 关东西 = 保护或封禁
으로 뒤집는다.
```

철칙:

```text
관객은 세계관 자체보다 주인공이 지금 만난 문제를 먼저 본다.
설정은 사건을 위해 존재한다.
사건이 설정을 설명하기 위해 멈추면 안 된다.
```

## 4. 리듬: 훅, 정보, 충돌, 폭발, 여백, 새 미스터리

표준 리듬:

```text
훅으로 시작
정보 전진
작은 충돌
클라이맥스
1~2초 감정 여백
새 미스터리로 끝
```

주의:

```text
전부 고에너지면 고에너지가 아니다.
전부 정적이면 이탈한다.
정적은 감정이나 위험이 있을 때만 길게 둔다.
```

21화 적용:

```text
문 안쪽 숨소리 훅
먼지/문틈/바람 없음으로 정보 전진
沈泊川의 현실 질문과 黑七의 감각 충돌
陆沉舟의 局 판단
문 안쪽 반응으로 클라이맥스
짧은 정적
안쪽의 불가능한 흔적/목소리/글자로 다음 화 유도
```

## 5. 대화: 단방향 설명 금지

대사는 정보 전달이 아니라 밀고 당기는 힘이어야 한다.

나쁜 방식:

```text
A가 설명한다.
B가 듣는다.
C가 놀란다.
```

좋은 방식:

```text
A가 오해한다.
B가 감각으로 반박한다.
C가 차갑게 관찰을 찌른다.
주인공이 진짜 구조를 말한다.
마지막 대사가 더 큰 질문을 남긴다.
```

추천 구조:

```text
追问 -> 反驳 -> 爆点 -> 升华
```

단, 모든 장면에 기계적으로 적용하지 않는다. 위기 장면에서는 짧게, 정보 장면에서는 반응 컷과 함께 나눈다.

## 6. 영상화: 심리보다 촬영 가능한 디테일

AI가 바로 그릴 수 없는 문장:

```text
그는 白清檀이 그리워 가슴이 아팠다.
```

AI가 화면으로 옮길 수 있는 문장:

```text
陆沉舟가 손끝의 종잇조각을 비비다 멈춘다.
손가락 관절이 아주 조금 희어진다.
시선이 문이 아니라 멀리 비어 있는 곳에 걸린다.
```

감정은 항상 매체를 가진다.

```text
그리움 = 종잇조각, 印鉴, 손끝 멈춤, 시선 흐림
긴장 = 문고리 앞에서 멈춘 손, 뒤로 반보 물러남
압박 = 낮아진 어깨, 고정된 시선, 줄어든 움직임
불안 = 호흡, 손목 긴장, 말끝 흐림
```

## 7. 대사: 짧은 교전 리듬, 글자 수 제한 아님

중요 보정:

```text
3~5字 규칙은 절대 고정 글자 수 제한이 아니다.
그 의미는 영상 리듬에서 대사를 한 번에 길게 뭉치지 말라는 것이다.
```

좋은 방식:

```text
행동
짧은 판단
반응
추가 단서
상대 질문
더 깊은 판단
```

예:

```text
陆沉舟抬手按住符痕，眉峰微蹙。
"气不对。"
他指尖发力，红光逆着纹路往回缩。
"是往里吸的。"
```

주의:

```text
시청자가 이해하지 못할 정도로 짧게 자르지 않는다.
중요한 세계관/사건 정보는 화면, 소품, 반응 컷과 나눠서 전달한다.
```

## 8. 시각 기호: 추상 설정을 물체로 만든다

추상 설정은 화면 기호로 변환한다.

```text
阵法 = 红光符痕, 灰烬光点, 对称刻痕
白清檀 = 印鉴, 指尖残影, 女王印의 흔적
无相会 = 单字刻痕, 비정상적으로 깨끗한 흔적
局 = 气의 흐름이 막힌 공간, 바람 없는 복도, 안쪽으로 빨려드는 먼지
```

장점:

```text
AI 생성이 쉽다.
관객이 기억한다.
반복 등장하면 장기 복선이 된다.
```

## 9. 爽点: 힘보다 규칙 이해

陆沉舟의 강함은 법력 과시가 아니라 구조 이해에서 나온다.

원칙:

```text
상위자의 압도감은 더 세게 때리는 데서 오지 않는다.
남들이 싸우는 문제를 규칙으로 읽고, 가장 안정적으로 푸는 데서 온다.
```

좋은 방향:

```text
阵을 폭파하지 않는다.
먼저 구조를 보고, 어디를 건드리면 위험한지 판단한다.
일반인이 보는 공포를 규칙의 문제로 낮춰 해결한다.
```

## 10. 자산 관리

소설 단계부터 제작 비용을 생각한다.

우선 사용:

```text
고정 주인공단
고정 복장
고정 장소
반복 가능한 소품
老宅, 골목, 문, 자료, 印鉴, 符痕
```

주의:

```text
일회성 새 인물 남발 금지
일회성 새 장소 남발 금지
한 컷만 쓰고 버릴 복잡한 소품 금지
```

## 11. 단체 장면 기능 분담

기본 기능:

```text
陆沉舟: 판단, 파국, 정조
黑七: 감지, 입씨름, 감정 완충
檀缺: 관찰, 黑七 견제, 냉정한 보강
沈泊川: 관객 시점, 현실 질문, 감정 대입
韩知玄: 镇灵司 책임, 제도적 압력, 공무식 절제
```

기능이 없으면 장면에서 빼거나, 반응 컷으로만 둔다.

## 12. 강약 연기 분급

상위자:

```text
움직임 적음
눈 깜빡임 적음
시선 고정
어깨와 손끝으로 압박
말은 늦게, 짧게, 정확하게
```

하위자/현실 대변자:

```text
몸이 작게 움츠러듦
발걸음이 흔들림
호흡 변화가 크다
표정 반응이 더 빠르다
말로 관객의 질문을 대신한다
```

주의:

```text
강한 인물도 사람이므로 감정 흔적은 있어야 한다.
약한 인물도 도구가 아니므로 선택과 기능이 있어야 한다.
```

## 13. 반전: 예측 위반 + 복선

반전은 갑자기 만들지 않는다.

조건:

```text
관객이 기본 가정을 한다.
앞에서 작은 이상 신호가 쌓인다.
주인공이 그 신호를 다른 방향으로 읽는다.
반전이 나오면 앞 장면이 다시 이해된다.
```

나쁜 방식:

```text
앞에 근거 없이 사실을 뒤집기
충격만 있고 논리가 없음
인물이 알아야 할 이유 없이 알아맞힘
```

## 14. 금지 5개

```text
开篇堆设定: 초반에 설정부터 쌓기
全知视角大段心理: 촬영 불가능한 심리 독백 남발
配角工具人: 조연이 "怎么办", "好厉害"만 반복
为了反转而反转: 복선 없는 반전
节奏全程平: 훅/충돌/여백/새 질문이 없는 평평한 진행
```

## 15. 21화 최종 검사

21화를 쓰기 전과 쓴 후 모두 확인한다.

```text
20화 마지막 老宅 문 앞 긴장에서 바로 이어지는가?
첫 3초에 문 안쪽 반응 또는 불가능한 소리가 있는가?
인물이 왜 여기 있는지 다시 설명하지 않아도 보이는가?
陆沉舟가 감정을 말하지 않고 행동과 판단으로 보이는가?
黑七은 단순 냄새 설명이 아니라 감각과 걱정이 같이 있는가?
檀缺은 독설보다 구조 관찰을 하는가?
沈泊川은 바보가 아니라 관객의 현실적 불안을 말하는가?
설정 설명이 사건을 멈추지 않는가?
마지막에 새 미스터리와 감정 여운이 동시에 남는가?
```

## 16. 한 문장 결론

```text
소설은 감정으로 붙잡고, 영상은 화면 훅으로 붙잡는다.
둘을 동시에 성공시키는 방법은 공허한 설명이 아니라,
인물의 행동, 대화, 선택으로 이야기와 감정을 보여주는 것이다.
```
