# Dialogue Freshness And Non Template Rules

목적: 대사가 매번 비슷한 구조와 비슷한 말로 반복되는 문제를 막는다. 캐릭터 말투는 유지하되, 같은 문장 공식으로 찍어내지 않는다.

핵심:

```text
캐릭터답다 = 매번 같은 말을 한다는 뜻이 아니다.
캐릭터답다 = 같은 사람의 사고방식으로, 다른 상황에 다르게 반응한다는 뜻이다.
```

## 1. 금지되는 대사 문제

아래가 보이면 수정한다.

```text
매화마다 "别动", "不是鬼", "不是人" 같은 판단어만 반복
沈泊川가 매번 "那怎么办" 식으로만 질문
黑七가 매번 같은 방식으로 비꼼
檀缺가 매번 짧은 독설만 함
陆沉舟가 매번 한 줄 판정만 하고 끝남
설명 대사 앞뒤에 캐릭터 감정이 없음
위험 장면마다 같은 호흡, 같은 단어, 같은 반응
```

문제:

```text
시청자는 캐릭터를 보는 게 아니라 작성 패턴을 보게 된다.
긴장 장면이 새롭지 않게 느껴진다.
다음 화를 봐도 비슷할 것 같다는 느낌이 든다.
```

## 2. 같은 기능, 다른 표현

대사의 기능은 같아도 표현은 매번 달라야 한다.

예: LU가 멈추라고 해야 하는 상황

반복 금지:

```text
别动。
别动。
别动。
```

대체 방식:

```text
手别碰门。
退半步。
别让它听见你。
你再往前，局就醒了。
站住。脚下有线。
```

예: LU가 귀신이 아니라고 판단해야 하는 상황

반복 금지:

```text
不是鬼。
不是人。
```

대체 방식:

```text
有鬼的地方，不会这么干净。
这屋子没闹鬼，是被人收拾得太干净了。
鬼会留下怨气。这里留下的是手法。
别按闹鬼查，方向错了。
它怕的不是阳光，是流动。
```

## 3. 캐릭터별 대사 변주 기준

### LU / 陆沉舟

고정 성격:

```text
판단 먼저
설명은 늦게
감정은 말보다 멈춤과 선택으로 보임
```

하지만 매번 짧은 판정문만 쓰면 안 된다.

변주 방식:

```text
명령형: 手别碰门。
관찰형: 灰是往外走的。
비교형: 有鬼的地方，不会这么安静。
위험 예고형: 门一开，里面压着的东西会找出口。
감정 억제형: 我现在没空怕。
```

### HEIQI / 黑七

고정 성격:

```text
먼저 비꼼
뒤에 걱정
감각이 빠름
위험하면 목소리가 낮아짐
```

반복 금지:

```text
또 "没鬼味"만 말하기
또 "晦气"만 말하기
```

변주 방식:

```text
这味儿不脏，反倒干净得瘆人。
有东西把气咬断了。
我不喜欢这门缝，像有人在里面闻我们。
别笑，我尾巴都不想靠近。
```

### TAN / 檀缺

고정 성격:

```text
짧고 날카로움
黑七을 자주 찌름
관찰이 차갑다
```

반복 금지:

```text
매번 黑七만 놀리고 끝내기
매번 한 글자 독설처럼 쓰기
```

변주 방식:

```text
你不是怕门，你是怕门后面没味道。
太干净了。干净得像有人擦过命。
他不是在问鬼，他是在问谁动过这里。
```

### SHEN / 沈泊川

고정 성격:

```text
시청자 대변인
현실적 불안
말이 생활감 있음
상대 분위기를 읽음
```

반복 금지:

```text
매번 "怎么办"
매번 "这也太怪了吧"
매번 보고서처럼 의뢰 설명
```

변주 방식:

```text
我接这活的时候，以为最多是凶宅传闻。
你们别都不说话，我会更慌。
这门我刚才差点就推了，是不是已经算捡回一条命？
我现在问一句会不会显得我很没见识：它在听吗？
```

## 4. 대사 작성 전 질문

대사를 쓰기 전에 반드시 묻는다.

```text
이 인물이 지금 무엇을 숨기고 있는가?
이 인물이 지금 무엇을 가장 먼저 알아챘는가?
이 인물이 말하지 않으면 관객이 무엇을 모르는가?
이 말은 지난 화에도 비슷하게 했는가?
같은 기능을 다른 방식으로 말할 수 있는가?
이 대사는 캐릭터 성격 + 현재 감정 + 스토리 정보를 같이 담는가?
```

## 5. 대사 중복 검사

소설/분경 완성 후 아래 검사를 한다.

```text
같은 단어가 같은 인물에게 2번 이상 반복되는가?
같은 구조의 질문이 2번 이상 반복되는가?
매번 "판단 -> 설명 -> 놀람" 순서만 반복되는가?
시청자 대변인 질문이 너무 직접적인가?
비꼼/농담이 상황을 가볍게 만들고 있지는 않은가?
캐릭터 말투가 아니라 작가 설명처럼 보이는가?
```

수정 기준:

```text
같은 의미라도 다른 물리적 행동, 다른 감각, 다른 비유, 다른 감정 압력으로 바꾼다.
```

## 6. 21화 적용

21화는 20화 老宅 문 앞에서 이어지므로, 대사는 문 앞 공포와 局 해체에 맞춰 새롭게 변주한다.

나쁜 방향:

```text
陆沉舟: 别动。
沈泊川: 里面有人？
陆沉舟: 不是人。
沈泊川: 那怎么办？
```

개선 방향:

```text
沈泊川手刚碰到门环。
陆沉舟: 手离门远点。它等的就是有人先碰。

沈泊川: 我刚才要是真推了呢？
陆沉舟: 你现在就不会站着问我。门里那口气，缺的就是一个出口。

黑七压低耳朵。
黑七: 这屋子没鬼味。干净得像刚死过一遍。

檀缺看向门缝。
檀缺: 不是没人住。是气不准出来。
```

주의:

```text
LU는 멋있는 말만 하지 않는다. 그는 먼저 사고하고, 필요한 만큼 말해 장면의 방향을 바꾼다.
SHEN은 바보처럼 놀라기만 하지 않는다. 그는 현실 감각으로 위험을 관객에게 번역한다.
HEIQI는 농담 기계가 아니다. 감각이 위험을 만나면 톤이 바뀐다.
TAN은 독설 기계가 아니다. 한마디로 장면의 구조를 찌른다.
```

## 7. 최종 원칙

```text
대사는 짧아서 좋은 것이 아니다.
대사는 캐릭터가 그 순간에만 할 수 있는 말일 때 좋다.
같은 캐릭터라도 같은 위협을 매번 같은 말로 설명하지 않는다.
```
