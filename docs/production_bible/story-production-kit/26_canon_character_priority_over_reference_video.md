# Canon Character Priority Over Reference Video

목적: 참고 영상 분석이 아무리 좋아도 `《棺中檀色》`의 세계관, 캐릭터, 관계, 말투를 덮어쓰지 못하게 한다.

핵심:

```text
참고 영상은 연출 문법이다.
세계관과 캐릭터는 작품의 뼈대다.
연출은 뼈대 위에 붙어야지, 뼈대를 바꾸면 안 된다.
```

## 1. 우선순위

작업 전 항상 아래 순서를 따른다.

```text
1. 계약/규제/금지 규칙
2. 세계관
3. 캐릭터 카드
4. 인물 관계와 기존 회차 사건
5. 장소/소품 설정
6. 대사/말투 규칙
7. 스토리 연속성
8. 참고 영상 분석
9. 분경/영상/CapCut 기술 규칙
```

참고 영상 분석은 8번이다.
참고 영상이 좋아 보여도 1~7번을 바꾸면 안 된다.

## 2. 참고 영상에서 가져와도 되는 것

```text
첫 3초 후킹 방식
컷 전환 리듬
인물 등장 타이밍
소품을 먼저 보여주는 방식
대사와 반응 컷을 나누는 방식
클리프행어 설계
현실 공간 속 이상 현상 처리
대사가 잘 들리게 하는 호흡
```

## 3. 참고 영상에서 가져오면 안 되는 것

```text
다른 작품의 캐릭터 성격
다른 작품의 인물 관계
다른 작품의 세계관 법칙
다른 작품의 말투
다른 작품의 사건 구조를 그대로 복사
다른 작품의 감정 결론
우리 캐릭터를 참고 영상 캐릭터처럼 행동하게 만들기
```

## 4. 캐릭터 고정 검사

소설/분경/영상 프롬프트를 쓰기 전에 각 인물을 확인한다.

```text
LU / 陆沉舟:
판단 먼저, 설명은 필요한 만큼. 감정은 과장하지 않되, 말/행동/침묵이 함께 장면을 끌고 간다.

HEIQI / 黑七:
검은 여우 본체. 감각이 빠르고, 비꼼 뒤에 걱정이 있다. 위험하면 톤이 낮아진다.

TAN / 檀缺:
짧고 날카로운 관찰. 黑七을 찌르되 장면 기능 없이 놀리기만 하면 안 된다.

SHEN / 沈泊川:
현실적 불안과 질문으로 시청자를 데려온다. 분위기를 읽고 선을 넘지 않는다.

HAN / 韩知玄:
镇灵司의 공무식 절제와 책임감. 사건 밖으로 빠질 때도 기능과 이유가 있어야 한다.
```

## 5. 세계관 고정 검사

장면마다 아래를 확인한다.

```text
이 현상은 작품 세계관 안에서 가능한가?
술법/结界/气/局의 규칙이 기존 설정과 충돌하지 않는가?
새로운 룰을 임의로 만든 것은 아닌가?
캐릭터가 몰라야 할 정보를 갑자기 아는가?
소품이나 자료가 공개될 타이밍을 앞당기지 않았는가?
```

## 6. 21화 적용

21화는 참고 영상의 공포 리듬을 빌릴 수 있다.

가져올 수 있는 것:

```text
문 안쪽 숨소리로 첫 3초 후킹
닫힌 문/문틈/먼지/소리 인서트 컷
대사 사이 반응 컷
마지막 5초에 안쪽 반응으로 끊기
```

가져오면 안 되는 것:

```text
LU를 다른 작품의 냉혈 탐정처럼 만들기
SHEN을 바보 리액션 담당으로 만들기
HEIQI를 단순 개그 담당으로 만들기
TAN을 독설만 하는 인물로 만들기
老宅 현상을 세계관과 상관없는 일반 공포물처럼 처리하기
```

