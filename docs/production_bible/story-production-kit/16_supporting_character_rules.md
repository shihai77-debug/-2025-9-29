# Supporting Character Rules

목적: 조연이 장면을 얇게 만들거나 주연을 잡아먹지 않도록, 기능 중심으로 관리한다.

## 1. 조연 기본 원칙

```text
조연은 주연보다 튀면 안 된다.
조연은 정보, 갈등, 감정, 떡밥 중 하나의 기능을 가져야 한다.
한 회차에 새 조연은 1~2명만 직접 등장한다.
나머지는 자료, 전화, 사진, 소문, 과거 기록으로 처리한다.
```

## 2. 조연 유형

```text
의뢰자: 사건을 가져온다.
피해자: 위험을 증명한다.
목격자: 단서를 제공하지만 전부 알지는 못한다.
방해자: 주인공의 판단을 막거나 지연시킨다.
정보 제공자: 다음 장소/소품/이름을 준다.
떡밥 조연: 지금은 작지만 다음화에서 의미가 커진다.
```

## 3. 조연 카드 양식

```text
이름:
회차:
역할:
처음 등장 위치:
주인공과의 관계:
가져오는 정보:
숨기는 정보:
만드는 갈등:
대표 대사:
퇴장 방식:
재등장 가능성:
```

## 4. 20화 적용

20화에는 조연을 많이 넣지 않는다.

```text
직접 등장 가능: 老宅 근처 이웃 1명 또는 관리인 1명
자료로만 등장: 保洁工人, 과거 집주인
직접 등장 금지: 사건 설명용 조연 여러 명
```

좋은 조연 대사:

```text
"那屋子白天也冷。不是阴冷，是闷得人喘不上气。"
"他们不是吓跑的，是捂着胸口冲出来的。"
```

## 5. 검수 질문

```text
이 조연이 없으면 장면 정보가 줄어드는가?
이 조연이 주연보다 튀지 않는가?
대사가 설명만 하고 있지 않은가?
재등장할 가치가 있는가?
```
