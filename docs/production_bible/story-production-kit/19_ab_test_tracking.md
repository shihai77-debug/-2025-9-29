# A/B Test Tracking

목적: 수익형 숏폼에서 도입부, 첫 대사, 마지막 컷을 여러 버전으로 테스트하고, 가장 강한 버전만 본편에 반영한다.

## 1. 테스트 대상

전체 회차를 여러 번 만들지 않는다. 아래만 테스트한다.

```text
첫 3초
첫 대사
핵심 소품 첫 노출
마지막 5초
다음화 질문
```

## 2. 기록 양식

```text
회차:
테스트 항목:

A안:
강점:
약점:
점수:

B안:
강점:
약점:
점수:

C안:
강점:
약점:
점수:

선택안:
선택 이유:
다음 적용 규칙:
```

## 3. 20화 도입부 테스트 예시

A안: 감정 정적 시작

```text
LU가 昆仑 방향을 보고 움직이지 않음.
아무도 말하지 않음.
멀리서 오토바이 소리.
```

B안: 소리 훅 시작

```text
검은 화면에 오토바이 엔진음.
HEIQI 귀가 움직임.
LU는 여전히 반응하지 않음.
```

C안: 소품 훅 시작

```text
SHEN의 닫힌 背包 클로즈업.
가방 안에서 무언가 진동하지만 열리지 않음.
```

추천:

```text
20화는 A안이 감정 연결에 가장 좋다.
B안은 속도감이 좋다.
C안은 20화 중후반 또는 마지막 컷 후보로 적합하다.
```

## 4. 선택 기준

```text
첫 3초에 질문이 생기는가?
전 화 감정을 받는가?
새 사건을 바로 예고하는가?
캐릭터가 흔들리지 않는가?
영상 제작 비용이 과하지 않은가?
```
