# Dialogue Stage Necessity Gate

목적: 대사를 "역할 분담표"처럼 쓰는 오류를 막는다. 대사는 캐릭터별 기능 순서가 아니라, 현장 위험과 말할 수밖에 없는 이유에서 나와야 한다.

핵심:

```text
스토리 주제 먼저 고정.
현장 위험 고정.
인물별 말할 이유 고정.
그다음 대사 작성.
```

좋은 대사 구조도 기계적으로 적용하면 실패한다.

```text
沈泊川 질문 -> 黑七 감각 -> 檀缺 찌르기 -> 陆沉舟 정리
```

이 순서는 고정 틀이 아니라, 장면이 실제로 그렇게 요구할 때만 쓴다.

## 1. 대사 작성 전 4단계 고정

대사를 한 줄이라도 쓰기 전에 아래 네 질문을 반드시 통과한다.

```text
1. 지금 장면에서 누가 가장 위험한가?
2. 누가 먼저 말할 수밖에 없는가?
3. 누가 말하지 않고 행동해야 하는가?
4. 이 대사가 없으면 장면이 안 움직이는가?
```

하나라도 답이 약하면 그 대사는 삭제하거나 행동/시선/소품으로 바꾼다.

## 2. 역할표 대사 금지

아래 방식은 실패다.

```text
沈泊川는 항상 질문한다.
黑七는 항상 냄새를 말한다.
檀缺는 항상 黑七을 찌른다.
陆沉舟는 항상 마지막에 정리한다.
```

이건 사람이 아니라 기능표다.

수정 원칙:

```text
沈泊川는 겁나거나 책임이 걸렸을 때 말한다.
黑七는 감각이 끊기거나 이상하게 비었을 때 말한다.
檀缺는 黑七의 허세나 현장 오판을 찌를 이유가 있을 때만 말한다.
陆沉舟는 설명하려고 말하는 것이 아니라, 사람을 막고 현장을 장악하려고 말한다.
```

## 3. 현장 감정 순서

대사 순서는 캐릭터 순서가 아니라 현장 압력 순서다.

```text
누가 문을 만지려 하는가?
누가 먼저 위험을 몸으로 느끼는가?
누가 침묵해야 압력이 커지는가?
누가 지금 말하면 위험이 더 커지는가?
누가 말해야 다음 행동이 생기는가?
```

이 순서가 정해진 뒤에만 대사를 배치한다.

## 4. 대사 삭제 기준

아래 대사는 삭제한다.

```text
장면을 움직이지 않는 대사
누가 말해도 같은 정보 대사
캐릭터 감정 없이 설정만 말하는 대사
앞 대사를 다른 말로 반복하는 대사
고정 역할 때문에 억지로 넣은 대사
```

대신 아래로 바꾼다.

```text
위험을 멈추는 말
상대 행동을 바꾸는 말
오해를 깨는 말
다음 행동을 여는 말
숨은 감정을 압축해 드러내는 말
```

## 5. 21화 老宅 적용

21화의 주제는 아래처럼 고정한다.

```text
法阵封口
半松不破
引出布阵人
```

하지만 주제가 맞아도 대사는 바로 쓰지 않는다.

먼저 현장 위험을 고정한다.

```text
문을 열면 안 된다.
문은 잠긴 것이 아니라, 누군가 열어주기를 기다리는 상태다.
집 안의 气는 죽은 것이 아니라 막혀 있다.
阵은 완전히 깨면 안 되고, 반쯤 느슨하게 해야 사람/布阵人이 드러난다.
```

그다음 말할 이유를 고정한다.

```text
沈泊川: 자기가 문을 만질 뻔했거나 의뢰 책임이 걸렸을 때만 말한다.
黑七: 냄새가 나서가 아니라, 냄새가 비정상적으로 비었을 때 말이 끊긴다.
檀缺: 黑七가 허세로 덮거나 현장을 잘못 단순화할 때만 찌른다.
陆沉舟: 모두를 멈추고, 왜 완전히 깨면 안 되는지 행동 지시를 내리기 위해 말한다.
```

## 6. 통과 예시

나쁜 방식:

```text
沈泊川: 里面有人？
黑七: 没有鬼味。
檀缺: 你鼻子不灵了？
陆沉舟: 这是局。
```

좋은 방향:

```text
沈泊川的手已经碰到门环，指节发白。
陆沉舟: 手离门远点。它等的不是钥匙，是第一个替它开口的人。

黑七原本想笑，鼻尖动了两下，声音低下去。
黑七: 不对。不是脏，是太干净了。干净得像味儿被人刮没了。

檀缺看他一眼。
檀缺: 你刚才还说小事。

陆沉舟没有看他们，视线落在门槛灰线上。
陆沉舟: 小事不会把风也关在门外。这个局不能破，只能松。松给里面的人一口气，也松给布阵的人一个破绽。
```

## 7. 输出 명령

```text
Dialogue Stage Necessity Gate를 실행해.
대사 작성 전, 장면 위험/먼저 말할 사람/말하지 말아야 할 사람/대사 없으면 장면이 안 움직이는지 검사해.
"沈 질문 -> 黑 감각 -> 檀 찌름 -> 陆 정리" 같은 역할표 반복을 제거하고,
현장 감정 순서와 말할 필연성으로 대사를 다시 배치해.
```
