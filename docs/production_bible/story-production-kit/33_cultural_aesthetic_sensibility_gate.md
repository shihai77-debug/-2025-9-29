# Cultural Aesthetic Sensibility Gate

목적: 예술적 감각을 단순한 문장 장식이나 카메라 멋으로 착각하지 않게 한다. 이 작품의 미감은 문화감각, 장르감각, 인물 위계, 말투의 결, 장소의 생활감에서 나온다.

핵심:

```text
예술감각 = 문화감각 + 장르감각 + 인물감각 + 절제감각.
멋있는 문장보다, 이 세계의 사람이 이 순간에 이렇게 말하고 움직이는가가 먼저다.
```

## 1. 작업 전 문화 질문

소설/분경/대사/카메라 전에 반드시 묻는다.

```text
이 장면은 《棺中檀色》의 문화권과 장르 안에서 자연스러운가?
인물의 말투가 중국어 세계의 호칭/위계/정서와 맞는가?
현대 도시와 玄学/志怪 요소가 서로 따로 놀지 않는가?
침묵, 예의, 직무, 관계 거리감이 캐릭터답게 살아 있는가?
공포가 서양식 점프스케어가 아니라 이 작품의 气/局/结界 논리에서 나오는가?
```

하나라도 약하면 장면을 다시 설계한다.

## 2. 이 작품의 문화/장르 좌표

기본 좌표:

```text
현대 도시 현실
+ 오래된 술법/局/结界/气의 논리
+ 천 년 단위의 기억과 관계
+ 공무/조직/민간 의뢰의 현실성
+ 감정을 직접 터뜨리지 않는 절제
+ 작은 물증으로 큰 세계관을 드러내는 방식
```

피해야 할 좌표:

```text
일반 서양식 귀신의 집
탐정물이 된 주인공
개그 리액션만 하는 조연
설정 강의형 판타지
감정 과잉 멜로드라마
무국적 AI 공포물
```

## 3. 말투 문화감각

대사는 문법적으로 맞는 것만으로 부족하다. 위계, 거리감, 직무, 성격이 들어가야 한다.

검사:

```text
이 호칭이 관계에 맞는가?
이 인물이 이 정도까지 직접 말할 사람인가?
이 장면에서 농담이 허용되는 온도인가?
공무식 인물은 사적 감정을 너무 빨리 드러내지 않는가?
현대 인물의 말투가 너무 고풍스럽거나, 고대 인물의 말투가 너무 가볍지 않은가?
```

캐릭터별 주의:

```text
陆沉舟: 말없는 쿨함 금지. 필요할 때는 말로 사건을 붙잡고, 판단과 책임의 무게로 장면을 끌어야 한다.
黑七: 입이 가벼워도 감각과 걱정은 진짜다. 단순 개그 금지.
檀缺: 차갑고 짧지만 장면의 구조를 찌른다. 독설 장식 금지.
沈泊川: 현대 현실 감각. 겁은 있어도 바보가 아니다.
韩知玄: 镇灵司의 직무와 절제. 공무식 위계와 보고 책임을 잊지 않는다.
```

## 4. 장소 문화감각

장소는 배경이 아니라 문화적 압력이다.

老宅라면:

```text
낡은 철문
습기 먹은 벽
빛은 들어오지만 바람이 없는 복도
문틈의 먼지 방향
낡은 전선과 낮은 골목 소리
주변 이웃의 회피
```

昆仑라면:

```text
공간의 규칙이 인간보다 위에 있다.
감정으로 통과할 수 없다.
결계는 판타지 효과가 아니라 질서다.
```

镇灵司라면:

```text
조직, 보고, 절차, 책임이 먼저다.
인물이 감정적으로 남고 싶어도 직무 때문에 움직인다.
```

## 5. 미감: 절제와 물증

좋은 미감:

```text
큰 감정을 작은 행동에 담는다.
큰 세계관을 작은 물증에 담는다.
무서움은 설명보다 규칙 위반에서 나온다.
인물의 품격은 큰 소리보다 선택에서 나온다.
```

나쁜 미감:

```text
멋있는 한자어만 늘어놓기
판타지 빛 효과로 감정 덮기
무섭게 보이려고 소리만 키우기
인물이 자기 감정을 전부 설명하기
세계관을 강의처럼 말하기
```

## 6. 관객 문화감각

관객은 이런 질문으로 붙는다.

```text
이건 귀신인가, 사람이 만든 局인가?
이 사람들은 왜 감정을 참고 있는가?
이 조직/규칙은 왜 개인 감정보다 강한가?
천 년 전 관계가 지금 도시 사건에 어떻게 묻어 나오는가?
작은 물증이 나중에 어떤 큰 비밀로 돌아오는가?
```

관객이 싫어하는 것:

```text
어느 나라/어느 장르인지 모를 무국적 장면
현대 인물이 갑자기 고풍 문어체로만 말함
고대/영적 인물이 현대 농담만 함
공포, 수사, 감정이 각각 따로 노는 장면
```

## 7. 21화 적용

21화 老宅는 일반 공포물이 아니다.

문화/장르 방향:

```text
凶宅처럼 보이지만 핵심은 귀신이 아니라 사람이 만든 局.
문을 여는 행동은 공포 클리셰가 아니라 규칙을 건드리는 행위.
먼지, 바람 없음, 숨소리, 符痕이 모두 물증이다.
陆沉舟는 영웅처럼 과시하지 않고, 규칙을 읽는 사람처럼 움직인다.
沈泊川는 현대 의뢰인의 현실 감각으로 관객을 데려온다.
黑七와 檀缺는 단순 리액션이 아니라 서로 다른 감각과 관찰을 제공한다.
```

21화 금지:

```text
서양식 haunted house처럼 처리
문 안에서 갑자기 괴물 튀어나오기만 함
陆沉舟가 긴 술법 강의
沈泊川가 무식한 질문만 반복
黑七가 냄새만 말함
檀缺가 비꼼만 함
```

## 8. 출력 전 문화감각 체크

최종 출력 전 아래를 체크한다.

```text
[ ] 이 장면은 무국적 AI 장면이 아니라 《棺中檀色》 장면인가?
[ ] 문화권/장르/세계관이 대사와 행동에 배어 있는가?
[ ] 인물 간 위계와 거리감이 말투에 반영됐는가?
[ ] 공포와 미스터리가 气/局/结界의 논리에서 나오는가?
[ ] 큰 감정이 작은 행동으로 보이는가?
[ ] 소품과 공간이 장식이 아니라 물증으로 기능하는가?
[ ] 참고 영상 스타일이 캐릭터와 세계관을 덮지 않았는가?
```

## 9. Final Command

```text
Cultural Aesthetic Sensibility Gate를 실행해.
장면의 문화권, 장르 좌표, 말투 위계, 장소 물증, 공포 논리가 《棺中檀色》에 맞는지 검사한 뒤,
무국적/서양식/설정강의/개그리액션으로 흐른 부분을 수정해.
```
