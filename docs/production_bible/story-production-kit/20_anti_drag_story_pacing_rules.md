# Anti Drag Story Pacing Rules

목적: 감정, 설명, 조사, 이동 장면이 길어져서 스토리가 질질 끄는 문제를 막는다.

핵심:

```text
장면은 감정을 오래 붙잡아도 되지만, 사건은 멈추면 안 된다.
한 컷이 끝날 때마다 정보, 감정, 위험, 선택 중 하나는 반드시 앞으로 가야 한다.
```

## 1. 질질 끄는 신호

아래가 보이면 수정한다.

```text
같은 감정을 여러 컷 반복한다.
이미 관객이 아는 정보를 다시 설명한다.
한 장소에서 같은 질문만 반복한다.
대사가 길지만 새 정보가 없다.
인물이 말은 많이 하는데 선택하지 않는다.
조사 장면에서 단서가 단계적으로 나오지 않는다.
이동 장면이 분위기만 있고 사건 기능이 없다.
클리프행어 전에 이미 힘이 빠진다.
```

## 2. 장면별 허용 시간 감각

숏폼 기준 권장:

```text
감정 정적: 3~6초
새 인물 등장: 3~5초 안에 기능 제시
의뢰 설명: 10~15초 안에 핵심 위험 제시
현장 조사: 20~35초 안에 반전 또는 새 단서
티키타카: 2~4턴 이상 반복 금지
클리프행어 준비: 마지막 5~8초
```

예외:

```text
감정 클라이맥스는 길어도 되지만, 시선/손/소품/소리 변화가 있어야 한다.
```

## 3. 한 컷당 전진 조건

각 CUT은 아래 중 최소 하나를 해야 한다.

```text
새 정보
새 감정 변화
새 위험
새 선택 압박
새 위치 변화
새 소품 반응
다음 컷 질문
```

아무것도 하지 않으면 삭제하거나 다른 컷과 합친다.

## 4. 대사 질질 끌림 방지

나쁜 대사:

```text
이미 보이는 것을 다시 설명한다.
한 사람이 설정을 길게 말한다.
상대가 반응하지 않아 대화가 멈춘다.
```

좋은 방식:

```text
정보 대사 뒤에는 즉시 반박/질문/소품 반응을 둔다.
긴 정보는 대사 + 자료 컷 + 반응 컷으로 나눈다.
티키타카는 새 정보나 새 감정이 있을 때만 유지한다.
```

검수:

```text
이 대사를 빼도 장면이 이해되는가?
빼도 되면 삭제하거나 행동으로 바꾼다.
```

## 5. 조사 장면 리듬

조사 장면은 단서를 계단식으로 줘야 한다.

```text
1단계: 보이는 이상함
2단계: 상식과 맞지 않는 점
3단계: 귀신/일반 원인 배제
4단계: 사람이 만든 局 가능성
5단계: 열면 안 되는 이유
6단계: 다음화 질문
```

20화 老宅 예시:

```text
보이는 이상함: 먼지가 문 밖으로 밀려 있음
상식과 불일치: 빛은 있는데 바람이 없음
원인 배제: 귀신 냄새가 없음
판단: 누군가 기운을 막음
위험: 문을 열면 압력이 터짐
질문: 안에 산 사람이 없는데 숨소리가 들림
```

## 6. 감정 정적 질질 끌림 방지

감정 정적은 반복하면 힘이 빠진다.

좋은 감정 정적:

```text
정지
작은 소리
한 인물의 미세 반응
외부 사건 침입
선택 강제
```

나쁜 감정 정적:

```text
여러 인물이 계속 같은 방향만 봄
대사 없이 오래 멈춤
새 소리/소품/반응이 없음
```

20화 적용:

```text
19화의 기다림을 3~4초 보존한다.
그다음 오토바이 소리로 사건을 밀어 넣는다.
정적을 더 길게 늘리지 않는다.
```

## 7. 이동 장면 질질 끌림 방지

이동 장면은 분위기용만으로 쓰지 않는다.

이동 장면이 해야 할 일:

```text
장소 전환
소품 확인
인물 관계 변화
짧은 대사 충돌
다음 장소 위험 예고
```

예:

```text
오토바이 이동 중 SHEN이 老宅 자료를 말한다.
LU는 창밖을 보며 아직 昆仑에서 빠져나오지 못한다.
HEIQI가 도시 냄새 속에서 이상한 빈 냄새를 먼저 감지한다.
```

## 8. 삭제/합치기 기준

삭제:

```text
새 정보 없음
감정 반복
캐릭터 말투만 있고 사건 전진 없음
이미 다음 컷에서 더 잘 표현됨
```

합치기:

```text
같은 장소, 같은 감정, 같은 정보인 컷 2개 이상
긴 설명 대사 + 같은 의미의 반응 컷
이동 컷 + 단순 풍경 컷
```

유지:

```text
전컷 연결에 필요
캐릭터 선택을 보여줌
다음화 질문을 만듦
중요 소품을 처음 보여줌
```

## 9. 최종 검수 질문

```text
이 장면은 10초마다 새 질문이나 새 정보가 있는가?
감정 컷이 사건을 멈추게 하는가?
대사가 같은 정보를 반복하는가?
인물이 선택하지 않고 말만 하는가?
이 컷을 삭제해도 이야기가 그대로인가?
마지막 5초까지 힘이 남아 있는가?
```

## 10. 적용 원칙

```text
느린 장면은 가능하다.
멈춘 장면은 위험하다.
감정은 길어도 되지만, 장면 기능은 계속 전진해야 한다.
```
