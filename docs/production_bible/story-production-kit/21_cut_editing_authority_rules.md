# CUT Editing Authority Rules

목적: CapCut 또는 기타 편집 단계에서 CUT 길이와 순서를 조정할 수 있는 범위와, 절대 건드리면 안 되는 서사/캐릭터 기준을 고정한다.

핵심:

```text
CUT 편집은 가능하다.
하지만 편집은 "이야기를 더 잘 보이게 하는 작업"이지,
세계관/캐릭터/사건 의미를 바꾸는 작업이 아니다.
```

## 1. 편집이 바꿔도 되는 것

아래 항목은 CapCut 단계에서 조정 가능하다.

```text
CUT 길이
CUT 분할 수
반응 컷 추가
손/소품/문/발걸음 같은 인서트 컷 추가
대사 사이 호흡 길이
자막 등장 타이밍
효과음 시작 위치
배경음악 볼륨과 진입 타이밍
장면 전환 속도
느린 컷의 앞뒤 일부 삭제
같은 장면 안에서 시선 반응 컷 순서 조정
이미지 기반 줌/패닝으로 미세한 카메라 움직임 만들기
저비용 전환 카드 추가
```

허용 이유:

```text
시청자가 더 빨리 이해하게 만들기 위해서.
감정 호흡을 더 선명하게 만들기 위해서.
AI 영상 생성 실패를 재생성 없이 덮기 위해서.
다음 컷을 보고 싶게 만드는 리듬을 만들기 위해서.
```

## 2. 편집이 바꾸면 안 되는 것

아래 항목은 편집 단계에서 자동 변경 금지다.

```text
세계관 규칙
캐릭터 나이/외형/말투/성격
인물 관계의 의미
이전 화 마지막 대사와 다음 화 첫 장면의 연결
핵심 사건 순서
중요 소품 공개 타이밍
자료/문서/증거 공개 타이밍
누가 먼저 알고 누가 나중에 아는지
대사의 핵심 의미
클리프행어의 사실
금지된 시각 요소
```

예:

```text
CUT-01에서 SHEN 오토바이 진입 시간을 1초 늦추는 것 = 가능.
CUT-01에서 SHEN이 처음부터 서 있게 만드는 것 = 금지.

자료 화면을 CUT-03에서 CUT-02로 앞당기는 것 = 보통 금지.
자료를 보여주지 않고 SHEN의 닫힌 가방 반응으로 긴장만 만드는 것 = 가능.

LU의 침묵을 4초에서 3초로 줄이는 것 = 가능.
LU가 "괜찮다"처럼 감정을 직접 말하게 바꾸는 것 = 금지.
```

## 3. 편집 전 필수 확인

편집하기 전에 아래를 먼저 확인한다.

```text
1. 이전 컷/이전 화 마지막 대사
2. 이번 컷의 첫 이미지
3. 이번 컷에서 반드시 전달해야 하는 정보
4. 이번 컷에서 변해야 하는 감정
5. 이번 컷에서 공개하면 안 되는 정보
6. 다음 컷을 보게 만드는 마지막 질문
7. 캐릭터 위치와 시선의 연속성
8. 대사 원문과 자막 타이밍
```

체크 질문:

```text
이 CUT을 2초 줄여도 사건 의미가 그대로인가?
반응 컷을 넣으면 감정이 더 잘 보이는가?
대사 없는 침묵이 감정을 만들고 있는가, 아니면 그냥 비어 있는가?
이 컷을 삭제해도 이야기가 그대로면 삭제하거나 합친다.
이 컷이 없어지면 캐릭터 감정 변화가 사라지면 유지한다.
```

## 4. 편집 수정 우선순위

문제가 생겼을 때 바로 재생성하지 않는다. 아래 순서로 해결한다.

```text
1. 자막 타이밍 조정
2. 효과음/음악으로 전환 보강
3. 컷 앞뒤 트리밍
4. 인서트 컷 추가
5. 반응 컷 추가
6. 같은 장면 안에서 컷 순서 미세 조정
7. 정지 이미지 줌/패닝 처리
8. 짧은 전환 카드 추가
9. 그래도 의미가 깨지면 해당 CUT만 재생성
```

재생성은 마지막 수단이다.

```text
얼굴이 완전히 변함
인물 정체가 헷갈림
핵심 소품이 잘못 나옴
공개 금지 정보가 보임
사건 순서가 바뀜
대사가 이해되지 않음
세계관 규칙을 어김
```

이 경우에는 편집으로 덮지 말고 다시 만든다.

## 5. CUT 분할 판단

한 CUT이 길어질 때는 아래 기준으로 나눈다.

```text
감정이 바뀌는 순간
새 인물이 들어오는 순간
시선 방향이 바뀌는 순간
소품이 처음 보이는 순간
위험이 드러나는 순간
대사의 주제가 바뀌는 순간
카메라 목적이 바뀌는 순간
```

나누면 안 되는 경우:

```text
한 호흡으로 가야 하는 고백/충격/침묵
인물이 한 문장을 끝까지 밀고 가야 하는 대사
카메라가 일부러 멈춰서 감정을 견디는 장면
```

## 6. 대사 편집 기준

대사는 무조건 짧게 줄이지 않는다.

```text
대사가 캐릭터 성격 + 현재 감정 + 스토리 정보를 동시에 담고 있으면 유지한다.
정보만 반복하는 부분은 줄인다.
감정 없는 설명은 소품 컷/반응 컷/행동으로 나눠 보여준다.
중요 대사는 자막이 읽히는 시간을 확보한다.
한 인물의 긴 대사는 중간에 상대 반응 컷을 넣어 숨을 만든다.
```

대사 삭제 금지:

```text
캐릭터 말투가 드러나는 말
다음 사건의 단서
인물 관계를 바꾸는 말
감정 전환이 걸린 말
이전 화 마지막 대사와 연결되는 말
```

## 7. 3초 후킹 편집

첫 3초는 설명이 아니라 질문을 만든다.

가능한 방식:

```text
소리 먼저 들어오기
닫힌 문/가방/휴대폰 먼저 보이기
인물이 움직이지 않는 정적 먼저 보이기
이상한 공간 규칙 먼저 보이기
이전 화 마지막 대사의 여운을 먼저 보이기
```

금지:

```text
첫 3초에 배경 설명 자막 길게 넣기
처음부터 모든 정보를 말하기
캐릭터가 왜 여기 있는지 없이 갑자기 등장하기
전 화 감정과 상관없는 새 장면으로 튀기
```

## 8. 10초 리듬 체크

편집본은 대략 10초마다 아래 중 하나가 있어야 한다.

```text
새 정보
감정 변화
위험 증가
선택 압박
소품 반응
인물 진입/퇴장
다음 질문
```

10초 동안 아무것도 변하지 않으면:

```text
트리밍한다.
반응 컷을 넣는다.
효과음으로 위험을 앞당긴다.
대사 순서를 조정한다.
다음 단서를 조금 더 빨리 보여준다.
```

## 9. CapCut 편집자의 권한 문장

앞으로 편집 지시를 줄 때 아래 문장을 기준으로 한다.

```text
CapCut 편집은 CUT 길이, 반응 컷, 인서트 컷, 자막 타이밍, 효과음, 음악, 전환을 조정할 수 있다.
단, 세계관/캐릭터/사건 의미/대사 핵심/소품 공개 순서/클리프행어 사실은 바꾸지 않는다.
```
