# Shen 오토바이 진입 컷 실패 분석 및 프롬프트 교정 규칙

이 문서는 Shen이 오토바이를 타고 등장하는 컷에서 발생한 생성 실패를 기록한다. 목적은 다음 프롬프트에서 **黑七 종 오류, 장르 이탈, 인물/단체 추가, 타이밍 반복, Shen 소품 노출 오류**를 막는 것이다.

## 참고素材 명칭 교정

Shen 오토바이 진입 컷에서 白清檀 참고자료를 쓸 때 `@白清檀三面图`라고 쓰지 않는다. 현재 기준에서는 白清檀은 삼면도 명칭이 아니라 사용자가 제공한 캐릭터 기준 이미지 / 女主角参考图 계열로 부른다.

```text

권장 표기: @白清檀角色参考图 / @白清檀参考图 / @女主角参考图
```

프롬프트에는 아래처럼 쓴다.


## 실패 영상 분석에서 확인된 문제

생성 분석 자료에서 다음 오류가 확인되었다.

```text
1. 黑七가 小型黑色狐狸가 아니라 黑狼로 분석됨.
2. 배경이 “末世 / futuristic city / post-apocalyptic” 장르로 튐.
3. 白清檀이 “深色开衫女子”처럼 檀缺과 섞일 위험이 생김.
4. 韩知玄의 소속 표기가 “辑灵司”로 틀어짐. 원래는 镇灵司.
5. Shen이 “handsome man / sharp features / intense gaze” 기본 AI 미남으로 변할 위험.
6. Shen의 背包에서 문서가 보이거나 소품이 노출될 위험.
7. 0-1.5 / 1.5-4 / 4-6초 구조가 두 번 반복되어 12초 컷이 6초 반복처럼 깨짐.
8. 영어 키워드와 8K/hyper-detailed가 화질보다 캐릭터 일반화를 유도함.
```

## 时间结构防重复

该 컷是 12초 3단 구조이며, 생성 분석에서 6초 구조가 반복되는 문제가 발생했다. 다음 고정문을 사용한다.

```text
本CUT总时长12秒，只分三段，不重复任何段落。
0-4秒：固定镜头，原有四个角色位置锁定，无台词。
4-8秒：镜头平稳跟随Shen骑摩托车从右侧真实进入。
8-12秒：Shen停下摩托车，取下或手持头盔，镜头落在Shen，其他角色位置不变。
不要重复0-4秒，不要循环同一动作。
```
