# 연기 분할도 형식 고정 규칙｜Acting Split Reference Format Rules

## 목적

사용자가 말하는 **연기 분할도 / 감독 연기자료 / 분할 참고도**는 공식 분진의 CUT 수나 공식 格 수를 그대로 보여주는 단순 스틸이 아니다.

공식 분진은 영상 구조 기준이고, 연기 분할도는 생성 안정화를 위한 배우 연기·시선·손동작·몸 긴장·카메라 구도 참고 자료다.

```text
공식 분진 = 최종 영상 컷 구조.
연기 분할도 = 배우가 그 컷을 어떻게 연기해야 하는지 보여주는 제작 참고도.
두 개는 목적이 다르다.
```

---

## 1. 공식 분진과 연기 분할도 구분

공식 분진에 2格이라고 쓰여 있어도, 사용자가 “연기 분할도”를 요구하면 필요한 경우 내부 연기를 더 세분화해 보여줄 수 있다.

단, 이것은 공식 컷 수와 대사 순서를 바꾸는 것이 아니다.

```text
공식 CUT 초수는 유지한다.
공식 대사는 유지한다.
공식 사건 순서는 유지한다.
다만 연기 참고도에서는 표정, 시선, 손, 몸, 카메라를 더 잘 보이게 작은 연기 단계로 나눈다.
```

---

## 2. 연기 분할도가 반드시 보여줘야 하는 것

연기 분할도는 아래 요소를 이미지 안에서 명확히 보여줘야 한다.

```text
배우 위치.
몸의 방향.
시선 방향.
고개 각도.
손 위치.
어깨와 몸의 긴장.
표정 변화.
상대와의 거리.
카메라 구도.
다음 동작으로 이어지는 마지막 포즈.
```

배경만 좋거나 인물이 그냥 서 있는 장면은 연기 분할도가 아니다.

---

## 3. 정적 컷의 연기 분할 규칙

정적인 컷도 연기 분할도에서는 2~3格로 나눌 수 있다.

예:

```text
格1：전체 자리와 관계.
格2：주요 인물의 표정, 손, 몸 긴장.
格3：보조 인물과 저위角色의 조용한 반응.
```

단, 모든 格에서 같은 포즈를 반복하면 안 된다.
각 格은 하나의 명확한 연기 정보만 보여준다.

---

## 4. CUT-01 적용 기준

第二十集 CUT-01 공식 구조는 다음과 같다.

```text
0-4秒：固定镜头，四人静止全景。
4-12秒：沈泊川骑摩托车入场，停稳，摘头盔，扫视四人神色。
```

공식 구조는 2段이지만, 사용자가 “연기 분할도”를 요청하면 다음처럼 제작한다.

```text
CUT-01-A｜0-4秒 静止死寂：3格 연기 분할도
  格1：四人一狐 전체 자리.
  格2：陆沉舟의 공허한 얼굴, 손, 몸 긴장.
  格3：韩知玄、檀缺、黑七의 낮은 반응.

CUT-01-B｜4-12秒 沈泊川入场：3格 연기 분할도
  格1：摩托车远处入画，众人 아직 정지.
  格2：沈泊川停稳，一脚撑地，摘头盔.
  格3：沈泊川扫视众人，众人只微动视线.
```

이렇게 나누어도 최종 영상은 여전히 CUT-01 하나, 총 12초다.

---

## 5. 금지

```text
연기 분할도 요청에 단일 장면 스틸만 출력 금지.
배경 설정도처럼 인물만 세워두기 금지.
공식 분진을 이유로 연기 세부 분할을 생략 금지.
단, 연기 분할을 이유로 공식 스토리, 대사, 초수 변경 금지.
```

---

## 6. 출력 전 확인

연기 분할도 프롬프트를 출력하기 전 아래를 확인한다.

```text
이것이 배경 설정도인가? 아니면 연기 분할도인가?
사용자가 원하는 것은 배우 연기 자료인가?
각 格이 서로 다른 연기 정보를 주는가?
공식 분진의 사건 순서와 대사를 바꾸지 않았는가?
인물 위치와 다음 컷 연결이 보이는가?
```
