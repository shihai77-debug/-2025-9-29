# 연기 분할도 형식 고정 규칙｜Acting Split Reference Format Rules

## 목적

이 문서는 사용자가 말하는 **연기 분할도 / 감독 연기자료 / 분할 참고도**의 의미를 정확히 고정한다.

중요:

```text
연기 분할도는 단순 배경 설정도가 아니다.
연기 분할도는 단순 한 장 스틸도 아니다.
연기 분할도는 공식 분진의 사건·초수·대사·연결을 바꾸는 새 분진도 아니다.
```

연기 분할도는 최종 영상 제작 전에 배우가 어떻게 서고, 어디를 보고, 손을 어떻게 두고, 몸의 긴장을 어떻게 잡는지 보여주는 **감독용 연기 참고 페이지**다.

---

## 1. 공식 분진과 연기 분할도의 관계

공식 분진은 최종 영상의 시간 구조다.
연기 분할도는 그 시간 구조를 바꾸지 않고, 그 안의 연기 정보를 더 정확히 보이게 하는 제작 참고도다.

```text
공식 분진 = 영상 컷 구조 / 사건 순서 / 초수 / 대사.
연기 분할도 = 배우 위치 / 시선 / 손 / 몸 긴장 / 표정 / 카메라 구도 참고.
```

절대 규칙:

```text
공식 초수 변경 금지.
공식 대사 변경 금지.
공식 사건 순서 변경 금지.
공식 CUT 연결 변경 금지.
```

---

## 2. 연기 분할도에서 보여줘야 하는 것

연기 분할도에는 반드시 아래 정보가 보여야 한다.

```text
인물 자리.
몸의 방향.
시선 방향.
고개 각도.
손 위치.
어깨와 몸의 긴장.
표정 변화.
상대와의 거리.
카메라 구도.
다음 동작으로 이어지는 마지막 자세.
```

배경만 좋고 인물이 그냥 서 있으면 연기 분할도가 아니다.

---

## 3. 연기 분할도 형식

연기 분할도는 무조건 3格, 5格, 9格로 고정하지 않는다.

사용자가 요구한 컷의 성격에 따라 다음 중 하나를 선택한다.

```text
1. 큰 메인 구도 1장 + 작은 연기 디테일 인셋 2~3개.
2. 공식 시간 흐름에 맞춘 2格 또는 3格 분할.
3. 움직임이 많은 컷은 4~5格 분할.
4. 5초 이하 정적 감정 컷은 과분할 금지, 2~3개 디테일만.
```

중요:

```text
格 수를 늘리는 것이 목적이 아니다.
배우 연기 정보가 보이는 것이 목적이다.
```

---

## 4. 정적 컷 처리법

정적인 컷은 한 장 스틸처럼 만들면 안 된다.
하지만 공식 분진을 무시해 억지로 시간 사건을 늘려서도 안 된다.

정적 컷의 올바른 방식:

```text
메인 패널：전체 자리와 카메라 구도.
디테일 인셋 1：주요 인물 얼굴, 시선, 어깨, 손.
디테일 인셋 2：보조 인물 반응 또는 저위角色 위치.
필요 시 디테일 인셋 3：다음 컷으로 이어지는 마지막 자세.
```

이 인셋들은 새 사건이 아니라 **연기 정보 확대**다.

## 5. CUT-01 적용 기준

第二十集 CUT-01 공식 구조:

```text
0-4秒：固定镜头，四人静止全景。
4-12秒：沈泊川骑摩托车入场，停稳，摘头盔，扫视四人神色。
```

Codex 기준을 유지하면서 연기 분할도를 만들 때는 다음처럼 이해한다.

```text
CUT-01 공식 시간 구조는 2段이다.
최종 영상은 하나의 12초 CUT이다.
연기 참고자료는 생성 안정성을 위해 A/B로 나눌 수 있다.
```

CUT-01-A｜0-4秒 静止死寂:

```text
필수 메인：四人一狐 전체 자리, 현대 도시 외곽 도로 배경.
필수 디테일：陆沉舟의 공허한 시선, 손, 어깨 긴장.
보조 디테일：韩知玄、檀缺、黑七의 낮은 반응.
沈泊川와摩托车는 아직 중심에 나오지 않는다.
```

CUT-01-B｜4-12秒 沈泊川入场:

```text
필수 메인：沈泊川와摩托车 입장 후 정지.
필수 디테일：摘头盔 손동작, 背包, 头盔.
보조 디테일：陆沉舟、韩知玄、檀缺、黑七는 위치 유지, 시선만 미세 반응.
```

주의:

```text
CUT-01-A를 단순 풍경 스틸로 만들면 FAIL.
CUT-01-A를 공식 분진과 무관한 새 3개 사건으로 쪼개도 FAIL.
올바른 방식은 메인 자리 + 연기 디테일 인셋이다.
```

## 6. 이미지 안 텍스트 처리

이미지 생성 안정성을 위해, 원칙적으로 화면 안에 글자, 번호, 화살표, 시간코드는 넣지 않는다.

다만 프롬프트 문장에는 패널/인셋 역할을 설명한다.

```text
画面里不要文字。
提示词里可以说明：主画面 / 细节插图 / 表演细节。
```

---

## 7. 금지

```text
연기 분할도 요청에 배경 설정도만 출력 금지.
연기 분할도 요청에 단순 합성 스틸만 출력 금지.
공식 분진을 이유로 배우 연기 디테일 생략 금지.
연기 분할을 이유로 공식 스토리, 대사, 초수 변경 금지.
무조건 3格으로 고정 금지.
무조건 9格으로 고정 금지.
```

---

## 8. 출력 전 확인

연기 분할도 프롬프트를 출력하기 전 아래를 확인한다.

```text
이것이 배경 설정도인가, 연기 분할도인가?
메인 자리와 연기 디테일이 모두 보이는가?
공식 분진 초수와 사건 순서를 바꾸지 않았는가?
사용자가 원하는 배우 연기 자료인가?
각 패널/인셋이 서로 다른 연기 정보를 주는가?
인물 위치와 다음 컷 연결이 보이는가?
```
