# 第十九集｜연기 분할 참고도 시간 분절 규칙

이 문서는 《棺中檀色》第19集 분镜/导演演技参考图 제작 중 누적된 경험값을 저장한다. 목적은 即梦/Seedance 영상 생성에서 같은 장면 반복, 대사 중복, 감정 단계 누락을 줄이기 위해 **연기 분할 참고도 자체를 행동·감정·대사 변화 지점 기준으로 나누는 것**이다.

---

## 핵심 원칙

연기 분할 참고도는 초수를 기계적으로 균등 분할하지 않는다. 반드시 다음 기준으로 나눈다.

```text
动作改变点 ＞ 眼神/情绪改变点 ＞ 台词开始点 ＞ 镜头功能改变点
```

즉, 8초 컷이라고 해서 무조건 0-4 / 4-8로 나누지 않는다. 정적인 감정 컷은 0-3 / 3-5 / 5-8처럼 **침묵 → 눈빛 정리 → 대사** 구조로 분할하는 것이 더 안정적이다.

---

## 分割参考图의 역할

분할 참고도는 완성 영상의 예쁜 한 장면이 아니라, 영상 생성 모델에게 **시간별 동작·시선·감정 변화·입모양 변화**를 알려주는 설계도다.

```text
分割参考图不是成片画面。
它是动作、视线、站位、情绪变化的时间设计图。
最终视频只参考动作和节奏，不参考图中文字、箭头、边框、黑白画风。
```

사용 시 반드시 다음을 프롬프트에 붙인다.

```text
导演参考图只参考动作、站位、视线、镜头节奏和每格时间分割。
不要参考黑白画风、文字、箭头、编号、边框和临时人物外形。
```

---

## 컷별 권장 분할 개수

분할 수는 컷 길이가 아니라 **행동 변화량**에 따라 정한다.

```text
5秒 이하 미세 감정컷：2-3分割 우선.
7-8秒 정적 감정컷：3分割 우선.
8秒 액션컷：2分割 또는 3分割.
10秒 동선+대사컷：4-5分割.
12秒 이상 복합 감정/동선컷：5-6分割 가능.
```

정지 감정 컷에서 5분할 이상으로 쪼개면 오히려 표정이 흔들리거나 같은 장면이 반복될 수 있다. 액션 컷은 실제 이동과 착지 상태가 보이면 2분할도 충분하다.

---

## 대사 시작 프레임 규칙

대사가 있는 컷은 입모양 변화 지점을 반드시 분리한다. 대사 전 구간과 대사 구간을 같은 그림으로 만들면, 모델이 입을 먼저 움직이거나 대사를 반복할 수 있다.

```text
대사 전 구간：입 닫힘, 숨만 있음, 무대사.
대사 시작 구간：입이 처음 열림, 해당 대사만 1회.
대사 후 구간：입이 자연스럽게 닫히거나 정지.
```

최종 영상 투입용에는 아래 문장을 사용한다.

```text
本CUT只有一句对白，只朗读一次。
不要重复朗读。
不要拆成多次对白。
对白只出现一次：……
```

## CUT-10 경험값｜“我等你” 감정 정지 컷

CUT-10은 움직임이 거의 없는 감정 수락 컷이다. 영상 실행에서 0-4초 같은 표정이 반복되면 대사 타이밍이 사라지고, 인물이 “悲伤幸存者 / 末世感”으로 흐를 위험이 있다. 따라서 3분할이 가장 안정적이다.

### 0-3秒｜沉默看她

```text
陆沉舟坐在地上，抬眼看白清檀。
嘴巴闭着。
呼吸轻微起伏。
眼神疲惫，还没有完全定下来。
白清檀前景肩背虚化，仍蹲在他身边。
无台词。
```

### 3-5秒｜眼神定下来

```text
陆沉舟仍坐在地上，视线固定在白清檀脸上。
眼神从疼痛、疲惫，慢慢变成安静接受。
嘴巴仍然闭着。
白清檀不动，不离开。
无台词。
```

### 5-8秒｜只说一次“我等你”

```text
陆沉舟轻轻开口。
只说一次：“我等你。”
说完后嘴巴自然合上。
白清檀仍在前景肩背位置，不说话。
```

## 정적 감정 컷 전용 연기 규칙

정적 감정 컷은 몸동작보다 눈빛 변화가 중요하다. 같은 구도 안에서 아래 3단계를 보여주는 것이 안정적이다.

```text
1단계：바라봄 / 입 닫힘 / 무대사.
2단계：눈빛 변화 / 감정 정리 / 무대사.
3단계：대사 1회 / 입 열림 / 대사 후 정지.
```

정적 감정 컷에서는 카메라와 배경을 되도록 유지한다. 움직임을 늘리기 위해 고개를 크게 돌리거나, 몸을 일으키거나, 카메라를 바꾸면 인물 얼굴과 감정이 흔들릴 수 있다.

---

## 대사 중복 방지 규칙

실제 지멍/영상 생성 투입용 프롬프트에서는 대사를 여러 섹션에 반복하지 않는다. 긴 감독판 설명에는 대사 설명이 들어가도 되지만, 최종 투입용 문장에는 아래처럼 한 번만 남긴다.

```text
本CUT只有一句对白，只朗读一次。
不要重复朗读。
不要拆成多次对白。
对白只出现一次：【陆沉舟】我等你。
```

대사 반복 오류가 난 경우, 다음을 확인한다.

```text
1. 本CUT重点 / 分镜执行 / 对白 / 最终压缩版에 같은 대사가 여러 번 들어갔는지 확인한다.
2. 영상 분석 자료가 같은 시간 블록을 두 번 반복하고 있는지 확인한다.
3. 최종 투입용에는 대사를 한 번만 남긴다.
```

## CUT-09 경험값｜折返动作은 단계로 보여야 함

CUT-09처럼 “다시 돌아오는 동선”이 중요한 컷은 바로 얼굴 클로즈업으로 튀면 안 된다. 분할 참고도는 반드시 0-4초에 折返 동선을 보여준다.

```text
0-2秒：白清檀看到陆沉舟脱力，开始从昆仑边界旁折返。
2-4秒：白清檀真实快步走回陆沉舟身边，蹲下。
4-6秒：右手轻扶陆沉舟脸颊，第一句台词。
6-8秒：维持单手扶脸，继续解释规则。
8-10秒：仍维持单手扶脸，说完“只认女王，不认外人”。
```

### CUT-09 손 오류 방지

```text
白清檀全程只允许出现一只可见的右手。
另一只手全程不出现。
不要双手捧脸。
不要多余手指。
不要手链、道具、盒子、印章进入画面。
```

## 손·도구·道具 가시성 규칙

손이 중요한 컷은 먼저 보이는 손의 개수를 정한다. 손 개수를 정하지 않으면 모델이 다중 손, 무주 손, 손가락 과다, 도구 손상 오류를 만들 수 있다.

```text
손이 보이는 컷：可见手数量先锁定。
손이 중요하지 않은 컷：不要出现手部特写。
한 손만 필요한 컷：只允许出现一只可见右手 / 左手。
나머지 손：全程不出现，或被身体遮挡。
```

女王印이 보이는 컷과 보이지 않는 컷도 명확히 나눈다.

```text
女王印가 필요한 컷：必须参考@女王印参考图，保持印章信物形态。
女王印가 필요 없는 컷：本CUT不显示女王印，不显示盒子，不显示任何手持道具。
```

특히 CUT-09, CUT-10처럼 얼굴·손·대사가 가까운 컷에서는 도구를 빼는 편이 안정적이다.

## 액션 컷 분할 기준

액션 컷은 감정 컷보다 단계가 적어도 된다. 다만 “실제 전진”과 “충돌 후 상태 변화”가 분명해야 한다.

```text
CUT-05 第一次尝试进入：0-2秒真实前移，2-4秒被无形阻力推回并准备起身。
CUT-07 第二次尝试进入：0-2秒再次快速前移，2-4秒更重地被推回，白清檀开口制止。
CUT-08 第三次尝试进入：0-4秒第三次拼尽全力前冲，4-8秒被推回后坐地脱力。
```

액션 컷에서 반드시 추가해야 하는 실제 이동 문장:

```text
陆沉舟必须真实前移，不是原地跑。
地面碎石和远山必须产生真实位移变化，证明他离开原位置。
```

## 액션컷 실제 이동 강화 규칙

액션컷 분할 참고도에는 시작 위치와 도착 위치가 달라야 한다. 같은 위치에서 자세만 바꾸면 영상에서 원地跑가 발생한다.

```text
起点：陆沉舟在画面后侧或近景低处。
过程：真实向前冲，脚步踩过碎石地面。
终点：身体位置明显靠近昆仑边界，或被推回后落到新的地面位置。
背景：碎石地面和远山产生视差变化。
```

프롬프트에는 아래 고정문을 사용한다.

```text
陆沉舟必须真实向前移动，不是在原地摆动手脚。
他的身体必须离开起始位置，地面和远山必须产生真实位移变化。
```

## 정적 감정 컷 분할 기준

정적 감정 컷은 5분할보다 2-3분할이 안정적이다. 움직임이 적은 컷에서 5분할 이상으로 쪼개면 표정이 흔들리거나, 같은 장면이 반복될 수 있다.

```text
5秒 이하 미세 감정 컷：2-3분할 우선.
7-8秒 정적 감정 컷：3분할 우선.
10秒 이상 동선+대사 컷：동작과 대사 시작점 기준으로 4-5분할 가능.
```

## 품질 안정 문장

영상 분석 자료에서 “末世 / 战斗后的幸存者 / 悲伤MV / post-apocalyptic / handsome young man / hyper-realistic 8k” 같은 단어가 나오면 다음 생성에서 제거한다. 대신 아래 문장을 쓴다.

```text
高质量彩色真人电影画面，真实演员质感，人物脸部清晰不变形，肤色自然，服装材质清楚，动作自然，画面稳定。
情绪是疲惫但稳定，不是哀求，不是崩溃，不是悲伤MV。
```

## 실패 영상 분석 자료 반영 규칙

영상 실행 후 분석 자료에 실패 단어가 나오면, 다음 프롬프트에 반드시 교정문을 넣는다. 실패 단어는 그대로 반복하지 않고, 정확한 대체 표현으로 바꾼다.

```text
portal / energy portal / 光圈 / 能量门户 → 昆仑边界不是门户，只是空气阻力或冷灰半透明空气幕。
lightning / 电流 / 蓝白光 / 落雷 → 结界不是雷电，只表现为轻微空气波纹。
antique box / 宝盒 / 盒子 → 女王印是古老印章信物，不是盒子。
running in place / 原地跑 → 必须真实前移，地面和远山有视差变化。
repeated dialogue / duplicated line → 对白只出现一次，只朗读一次。
post-apocalyptic / 末世 / 悲伤MV → 情绪克制稳定，真实短剧电影感。
```

앞으로 새 프롬프트 작성 순서:

```text
1. 当前CUT分镜图 확인.
2. 上一次生成视频分析 자료 확인.
3. 실패 단어와 잘못된 장르를 추출.
4. 다음 프롬 앞부분에 교정문 추가.
5. 최종 투입용은 대사와 핵심 동작을 중복 없이 한 번만 정리.
```

## 다른 모델/렌더러로 옮길 때 우리에게 맞는 적용 기준

다른 모델로 분할 참고도나 영상을 이어 만들 때는 아래 기준만 사용한다. 사용자 제공 자료 중 **시간 고정, 카메라 방식, 인물 연기 반복 명시, 스타일/배경 통일**은 채택하고, **최종 영상에서 흑백画风을 따라가게 하는 문장, 경계 초마다 무조건 하드컷하는 문장, 대사 반복, 정면 포즈 연결**은 제외한다.

### 1. 시간 구간은 고정하되, “하드컷”이 아니라 변화 지점으로 본다

분할 참고도에서 정한 시간 범위는 바꾸지 않는다. 그러나 하나의 연속 영상으로 만들 때는 매 구간을 칼같이 자르는 의미가 아니라, **동작·시선·입모양·대사 시작 지점**을 해당 시간에 맞추는 의미다.

```text
시간 범위는 유지한다.
단, 하나의 연속 영상에서는 하드컷이 아니라 내부 연기 변화 지점으로 처리한다.
```

분할 컷을 각각 따로 렌더링하는 경우에만 경계 초에서 편집 컷을 맞춘다.

### 2. 분할 참고도용 스타일과 최종 영상용 스타일을 분리한다

분할 참고도를 만들 때는 흑백 연필 스토리보드가 가능하다. 하지만 최종 영상에서는 흑백画风, 글자, 화살표, 패널, 번호를 따라가면 안 된다.

```text
导演参考图只参考动作、站位、视线、镜头节奏和每格时间分割。
不要参考黑白画风、文字、箭头、编号、边框和临时人物外形。
```

최종 영상용에는 아래처럼 실제 영상 스타일만 남긴다.

```text
高质量彩色真人电影画面，真实演员质感，人物脸部清晰不变形，肤色自然，服装材质清楚，动作自然，画面稳定。
```

### 3. 카메라 방식은 매 구간마다 직접 적는다

모델마다 기본 카메라 움직임이 다르므로, 각 구간에 촬영 방식을 반복해서 고정한다.

```text
移动段：中景稳定跟随拍摄，镜头轻跟人物移动，不摇晃，不绕圈。
固定段：固定近景，镜头不动，不推，不拉，不摇，不绕圈。
```

CUT-12 같은 구조는 다음처럼 쓴다.

```text
0-5秒：中景稳定跟随，表现白清檀真实离开陆沉舟身侧，移动到昆仑边界。
5-11秒：固定近景，白清檀站在昆仑边界边缘，身体偏向结界，头部回望陆沉舟。
```

### 4. 인물 연기 규칙은 매 구간에 반복한다

캐릭터 표정과 행동이 모델마다 다르게 해석되므로, 중요한 상태는 매 구간에 반복한다.

```text
白清檀：清冷、郑重、温柔克制，不哭，不崩溃，不回头跑向陆沉舟。
陆沉舟：始终坐在地面，只看着她，不站起来，不追，不拉住她，不说话。
```

### 5. 대사는 최종 투입용에서 한 번만 남긴다

분할 참고도에는 구간별로 대사를 표시할 수 있지만, 최종 영상 프롬프트에서는 대사를 한 번만 남긴다.

```text
本CUT只有一句对白，只朗读一次。
不要重复朗读。
不要拆成多次对白。
对白只出现一次：【白清檀】等我回来。三个月。昆仑。
```

### 6. 다음 컷 연결 포즈는 “정면 포즈”로 쓰지 않는다

연결 포즈는 다음 컷의 실제 행동에 맞춰야 한다. CUT-12의 마지막은 정면 포즈가 아니라, 다음 CUT-13의 “转身踏入结界”로 이어질 수 있는 회망 자세다.

```text
12-5最后一帧：白清檀身体仍靠近昆仑边界，身体方向偏向结界，头部回望陆沉舟。
不要正面对镜头。
下一CUT可以自然衔接为她转身踏入结界。
```

## CUT-12 적용값｜三个月昆仑

CUT-12는 “起身 → 이동 → 回头承诺” 구조다. 분할 참고도는 5분할이 안정적이다.

```text
12-1｜0-2秒：白清檀从陆沉舟身侧蹲姿起身。陆沉舟仍坐在地面。无台词。
12-2｜2-5秒：白清檀真实移动到昆仑边界。陆沉舟原地坐着，不追。无台词。
12-3｜5-7秒：固定近景，白清檀回头看陆沉舟，说“等我回来。”
12-4｜7-9秒：固定近景，白清檀保持回头，说“三个月。”
12-5｜9-11秒：固定近景，白清檀保持身体偏向结界、头部回望陆沉舟，说“昆仑。”说完停住。
```

영상용 핵심 문장:

```text
0-5秒中景稳定跟随，白清檀真实离开陆沉舟身侧，移动到昆仑边界，陆沉舟始终坐在地面不追。
5-11秒固定近景，白清檀站在昆仑边界边缘，身体偏向结界，头部回望陆沉舟。
本CUT只有一句对白，只朗读一次：【白清檀】等我回来。三个月。昆仑。
```
