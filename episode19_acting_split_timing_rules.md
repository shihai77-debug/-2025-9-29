# 第十九集｜연기 분할 참고도 시간 분절 규칙

이 문서는 《棺中檀色》第19集 분镜/导演演技参考图 제작 중 누적된 경험값을 저장한다. 목적은 即梦/Seedance 영상 생성에서 같은 장면 반복, 대사 중복, 감정 단계 누락을 줄이기 위해 **연기 분할 참고도 자체를 행동·감정·대사 변화 지점 기준으로 나누는 것**이다.

---

## 핵심 원칙

연기 분할 참고도는 초수를 기계적으로 균등 분할하지 않는다. 반드시 다음 기준으로 나눈다.

```text
动作改变点 ＞ 眼神/情绪改变点 ＞ 台词开始点 ＞ 镜头功能改变点
```

즉, 8초 컷이라고 해서 무조건 0-4 / 4-8로 나누지 않는다. 정적인 감정 컷은 0-3 / 3-5 / 5-8처럼 **침묵 → 눈빛 정리 → 대사** 구조로 분할하는 것이 더 안정적이다.

---

## CUT-10 경험값｜“我等你” 감정 정지 컷

CUT-10은 움직임이 거의 없는 감정 수락 컷이다. 영상 실행에서 0-4초 같은 표정이 반복되면 대사 타이밍이 사라지고, 인물이 “悲伤幸存者 / 末世感”으로 흐를 위험이 있다. 따라서 3분할이 가장 안정적이다.

### 0-3秒｜沉默看她

```text
陆沉舟坐在地上，抬眼看白清檀。
嘴巴闭着。
呼吸轻微起伏。
眼神疲惫，还没有完全定下来。
白清檀前景肩背虚化，仍蹲在他身边。
无台词。
```

### 3-5秒｜眼神定下来

```text
陆沉舟仍坐在地上，视线固定在白清檀脸上。
眼神从疼痛、疲惫，慢慢变成安静接受。
嘴巴仍然闭着。
白清檀不动，不离开。
无台词。
```

### 5-8秒｜只说一次“我等你”

```text
陆沉舟轻轻开口。
只说一次：“我等你。”
说完后嘴巴自然合上。
白清檀仍在前景肩背位置，不说话。
```

## 대사 중복 방지 규칙

실제 지멍/영상 생성 투입용 프롬프트에서는 대사를 여러 섹션에 반복하지 않는다. 긴 감독판 설명에는 대사 설명이 들어가도 되지만, 최종 투입용 문장에는 아래처럼 한 번만 남긴다.

```text
本CUT只有一句对白，只朗读一次。
不要重复朗读。
不要拆成多次对白。
对白只出现一次：【陆沉舟】我等你。
```

대사 반복 오류가 난 경우, 다음을 확인한다.

```text
1. 本CUT重点 / 分镜执行 / 对白 / 最终压缩版에 같은 대사가 여러 번 들어갔는지 확인한다.
2. 영상 분석 자료가 같은 시간 블록을 두 번 반복하고 있는지 확인한다.
3. 최종 투입용에는 대사를 한 번만 남긴다.
```

## CUT-09 경험값｜折返动作은 단계로 보여야 함

CUT-09처럼 “다시 돌아오는 동선”이 중요한 컷은 바로 얼굴 클로즈업으로 튀면 안 된다. 분할 참고도는 반드시 0-4초에 折返 동선을 보여준다.

```text
0-2秒：白清檀看到陆沉舟脱力，开始从昆仑边界旁折返。
2-4秒：白清檀真实快步走回陆沉舟身边，蹲下。
4-6秒：右手轻扶陆沉舟脸颊，第一句台词。
6-8秒：维持单手扶脸，继续解释规则。
8-10秒：仍维持单手扶脸，说完“只认女王，不认外人”。
```

### CUT-09 손 오류 방지

```text
白清檀全程只允许出现一只可见的右手。
另一只手全程不出现。
不要双手捧脸。
不要多余手指。
不要手链、道具、盒子、印章进入画面。
```

## 액션 컷 분할 기준

액션 컷은 감정 컷보다 단계가 적어도 된다. 다만 “실제 전진”과 “충돌 후 상태 변화”가 분명해야 한다.

```text
CUT-05 第一次尝试进入：0-2秒真实前移，2-4秒被无形阻力推回并准备起身。
CUT-07 第二次尝试进入：0-2秒再次快速前移，2-4秒更重地被推回，白清檀开口制止。
CUT-08 第三次尝试进入：0-4秒第三次拼尽全力前冲，4-8秒被推回后坐地脱力。
```

액션 컷에서 반드시 추가해야 하는 실제 이동 문장:

```text
陆沉舟必须真实前移，不是原地跑。
地面碎石和远山必须产生真实位移变化，证明他离开原位置。
```

## 정적 감정 컷 분할 기준

정적 감정 컷은 5분할보다 2-3분할이 안정적이다. 움직임이 적은 컷에서 5분할 이상으로 쪼개면 표정이 흔들리거나, 같은 장면이 반복될 수 있다.

```text
5秒 이하 미세 감정 컷：2-3분할 우선.
7-8秒 정적 감정 컷：3분할 우선.
10秒 이상 동선+대사 컷：동작과 대사 시작점 기준으로 4-5분할 가능.
```
