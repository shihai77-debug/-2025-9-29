# 연기 분할도와 분진의 작업 구분｜Acting Split vs Shot Script Workflow

## 핵심 정의

연기 분할도는 분진과 다른 작업물이다.

```text
분진 = 영상 생성 AI에게 줄 컷별 지시문.
연기 분할도 = 감독/배우/연출이 참고하는 정지 이미지형 연기 자료.
```

분진은 CUT 번호, 초수, 대사, 카메라워크, BGM/효과음, 장면 연결을 포함한다.

연기 분할도는 영상 길이나 실제 시간 흐름을 갖는 자료가 아니다. 한 장면에서 인물들이 어떻게 서고, 어떤 표정과 시선, 손동작, 몸 긴장을 가져야 하는지 보여주는 감독 참고 자료다.

---

## 1. 세트 제작 원칙

한 CUT은 아래 두 자료를 세트로 가질 수 있다.

```text
1. 분진 텍스트：최종 영상 생성용.
2. 연기 분할도 이미지 프롬프트：감독/배우 참고 정지 이미지 제작용.
```

예시:

```text
CUT-01 분진 텍스트
+
CUT-01 연기 분할도 이미지 프롬프트
```

이 둘은 같이 가지만 목적이 다르다.

---

## 2. 분진의 역할

분진은 실제 영상 생성 기준이다.

포함 요소:

```text
CUT 번호.
초수.
대사.
카메라워크.
BGM/효과음.
장면 연결.
인물 행동 순서.
```

분진은 영상 길이를 가진다.

---

## 3. 연기 분할도의 역할

연기 분할도는 정지 이미지 참고 자료다.

포함 요소:

```text
메인 구도.
인물 자리.
표정.
시선.
손 위치.
몸의 긴장.
상대와의 거리.
카메라 구도.
필요한 디테일 인셋.
```

연기 분할도는 영상 길이가 없다.
시간 흐름을 새로 만드는 자료가 아니다.

---

## 4. 이미지 생성 프롬프트 작성자의 역할

Claude / ChatGPT / Codex는 직접 이미지를 그리는 것이 아니라, 이미지 생성 모델에게 넘길 **텍스트 프롬프트**를 작성한다.

```text
작성자 역할：연기 분할도 이미지 생성용 프롬프트 작성.
이미지 생성 모델 역할：그 프롬프트를 바탕으로 정지 참고 이미지 생성.
```

따라서 프롬프트에는 이미지 생성기가 이해할 수 있도록 다음을 명확히 적어야 한다.

```text
메인 구도.
디테일 인셋.
인물 위치.
표정과 시선.
손과 몸의 긴장.
캐릭터 기준.
배경 기준.
금지사항.
```

---

## 5. 절대 혼동 금지

```text
연기 분할도를 분진처럼 쓰지 않는다.
분진을 연기 분할도처럼 쓰지 않는다.
연기 분할도에 영상 길이를 강제하지 않는다.
연기 분할도를 이유로 공식 대사와 사건 순서를 바꾸지 않는다.
분진을 이유로 연기 디테일 참고도를 생략하지 않는다.
```

---

## 6. EP20 적용 방식

第二十集부터는 각 CUT 작업 시 아래 순서로 진행한다.

```text
1. Codex 공식 분진 확인.
2. 해당 CUT의 공식 초수, 대사, 사건 순서 확인.
3. 필요한 경우 연기 분할도 이미지 프롬프트 작성.
4. 연기 분할도는 메인 구도 + 디테일 인셋 방식으로 제작.
5. 최종 영상 프롬프트는 다시 분진 기준으로 압축 작성.
```

---

## 7. CUT-01 적용 예

```text
CUT-01 분진：12초 영상 구조.
0-4秒：死寂静止。
4-12秒：沈泊川骑摩托车入场。

CUT-01 연기 분할도：정지 이미지 참고 자료.
목적：人物站位、表情、视线、手部动作、身体紧张、镜头构图을 맞추는 것.
형식：메인 구도 + 디테일 인셋.
```

연기 분할도는 CUT-01의 시간을 새로 쪼개는 것이 아니라, CUT-01을 촬영하기 전 배우와 연출이 볼 수 있는 정지 참고 이미지다.

---

## 최종 고정문

```text
분진은 영상 생성 지시문이다.
연기 분할도는 감독/배우 참고용 정지 이미지 자료다.
둘은 같은 CUT에 함께 붙을 수 있지만, 서로 같은 작업물이 아니다.
```
