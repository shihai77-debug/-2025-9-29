```text
檀缺 복장 오류 위험：중간~높음。

陆沉舟 시선 변화 누락 위험：높음。
原因：本集核心情绪线是“望昆仑消散方向 → 资料/责任唤回 → CUT-07第一次转眼看韩知玄 → CUT-15彻底聚焦老宅”。必须做连续性锚点。
```

## CUT-01｜沈泊川入场｜12秒

### 图像关键帧Prompt

```text
一张16:9黑白导演演技分割参考图，只做左右两个大主画面，不要底部插图，不要文字。
左侧格1：纯昆仑旷野，暮色暗沉，四人一狐静止全景。陆沉舟坐在左侧，黑色皮夹克，视线望昆仑消散方向；黑七是小型黑狐，四足低位依偎陆沉舟身侧；檀缺深灰褐针织开衫+黑色中长百褶裙站立；韩知玄长款镇灵司黑色制服外套站立。没有城市、路灯、楼房、柏油路、老宅。
右侧格2：沈泊川骑摩托车从右侧入场，镜头横向跟随。沈泊川所在右侧区域出现路灯、老旧楼房、柏油路面、道路边界，城市背景从右向左覆盖昆仑旷野；昆仑仍残留在左侧。沈泊川短黑发、深色工装短夹克、背包，停稳后摘头盔。四人一狐位置延续左侧，不重新站位，不互相对视，不围过去，只轻微视线反应。黑七仍是狐狸低位。
```

### 视频执行Prompt

```text
12秒，16:9，写实电影感。
0-4秒固定全景：完整昆仑旷野，无城市元素。陆沉舟坐姿低沉望向昆仑消散方向；黑七小黑狐低位依偎身侧；檀缺、韩知玄站立不动，沉默。
4-12秒镜头横向跟随沈泊川骑摩托车从右侧驶入。随着沈泊川进入，右侧现实城市轮廓逐步显现：路灯、老旧楼房、柏油路面，道路边界从右向左压入昆仑旷野。沈泊川停稳，一脚撑地，摘下头盔，扫视四人神色。四人一狐不重新站位，不围过去，只极轻微视线反应。无台词。
```

### 대사时间

```text
全CUT无台词。
0-4秒：风声，远处摩托车引擎若有若无。
4-10秒：摩托车引擎由远及近，刹车。
10-12秒：头盔扣具声，沈泊川摘头盔。
```

### 连续性锚点

```text
入：接第十九集CUT-14离别后静滞余韵，陆沉舟仍望昆仑消散方向。
出：沈泊川已停稳，摘头盔，姿势保持，下一CUT直接开口。
道具：摩托车、头盔、背包首次出现。
```

### 失败风险

```text
不要把格1做成现代城市。
不要让韩知玄和檀缺面对面。
不要把沈泊川画成陆沉舟或韩知玄。
不要把黑七画成人形。
不要把城市预先铺满画面；城市应随沈泊川进入。
```

## CUT-02｜棘手委托｜11秒

### 图像关键帧Prompt

```text
16:9中近景，沈泊川为主。沈泊川短黑发，深色实用短夹克/工装夹克，右肩背包，手从背包取出资料。背景保持CUT-01后现实城市压入后的过渡环境，昆仑余韵已淡，现实环境占主。陆沉舟、檀缺、韩知玄、黑七可在边缘虚化保持原位。沈泊川不是韩知玄，不穿长款镇灵司制服；不是陆沉舟，不穿皮夹克主角造型。
```

### 视频执行Prompt

```text
11秒，固定中近景，沈泊川为主。
0-5秒沈泊川弯腰或侧身从背包取出资料，动作爽利，带点熟人闯入后的轻松。
5-11秒沈泊川直立，资料微微前递，陈述委托。镜头不切，不乱动。其他人边缘维持沉默，不抢戏。黑七如出现必须是小型黑狐低位。
```

### 대사时间

```text
0-1秒：背包拉链声。
1-5秒 沈泊川：看你们几个这脸色，跟天塌了半截似的。不过我今天不是来探你们私事的，有个棘手的委托。
5-11秒 沈泊川：城南老城区一栋老旧民宅，接连几个保洁工人进去打扫，全都莫名窒息心慌，拼了命从屋里冲出来，没人敢再踏进一步。
```

### 连续性锚点

```text
入：接CUT-01沈泊川摘头盔后的直立姿势。
出：沈泊川手中资料已经取出，下一CUT准备递给陆沉舟。
道具：背包打开，资料出现。
```

### 失败风险

```text
沈泊川不能穿韩知玄长款制服。
沈泊川不能像陆沉舟皮夹克男主。
台词很长，需口型稳定，镜头固定。
```

## CUT-03｜强塞资料｜9秒

### 图像关键帧Prompt

```text
16:9双人中景，沈泊川与陆沉舟分处两侧。沈泊川深色工装短夹克+背包，手持资料前递；陆沉舟黑皮夹克坐姿未变，视线仍望昆仑消散方向，不看沈泊川。第二关键帧：沈泊川上前一步，把资料塞进陆沉舟手中，陆沉舟手部被动接触资料，但视线仍未移动。黑七如入画必须是小黑狐低位。
```

### 视频执行Prompt

```text
9秒固定双人中景。
0-4秒沈泊川保持递资料姿势，说出原因；陆沉舟坐姿不变，视线仍空望昆仑方向。
4-9秒沈泊川上前一步，强行把资料塞进陆沉舟手里。陆沉舟手部被动接受，身体和视线不回应。镜头稳定，动作清楚。
```

### 대사时间

```text
0-4秒 沈泊川：寻常阴邪作祟不会是这个反应，我查不出端倪，这已经超出我能处理的范围了，只能来找你。
4-9秒：无台词，脚步声，纸张摩擦声。
```

### 连续性锚点

```text
入：接CUT-02资料前递。
出：资料已经在陆沉舟手中，下一CUT旁观反应接上。
陆沉舟视线仍未转移。
```

### 失败风险

```text
不要让陆沉舟主动接资料或主动看沈泊川。
不要让沈泊川像韩知玄。
资料必须进入陆沉舟手里。
```
