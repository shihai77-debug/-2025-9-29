# Stable Character Video Prompt Template｜稳定人物视频提示词模板

本文件记录用户实测后人物相对稳定的提示词结构。后续 EP22 或新CUT 生成视频提示词时，应优先采用此结构，再结合 source index、三面图、批准 storyboard board 和当前CUT内容进行改写。

---

## 1. 为什么这个结构更稳定

稳定原因：

```text
1. 明确写出本CUT使用哪些角色参考图。
2. 明确写出本CUT不使用哪些角色参考图。
3. 明确写出哪些角色不出镜、不站位、不在背景等待。
4. 明确锁定本CUT只允许出现的真人角色数量。
5. 把道具、假人、动物、路人和真人角色分开定义。
6. 每个参考素材只说明用途，避免模型把导演分镜里的临时脸当成最终脸。
7. 每个时间段都写清楚动作、表情、视线、站位和镜头。
8. 最后用关键限制重复禁止项，防止额外人物、重复人物和角色混淆。
```

---

## 2. 必须保留的稳定结构

每个视频提示词尽量按以下顺序输出：

```text
【参考素材】
【重要说明】
【画面风格】
【参考素材使用规则】
【角色外形锁定】
【本CUT人物数量规则】
【道具 / 动物 / 假人 / 路人规则】
【场景】
【画面主体】
【导演参考图使用规则】
【摄影机】
【分段时间轴】
【对白 / 台词】
【表演要求】
【声音设计】
【BGM】
【连接逻辑】
【关键限制】
【建议时长】
```

---

## 3. 参考素材写法

必须分开写：

```text
角色参考：@角色A三面图 / @角色A参考图
角色参考：@角色B三面图 / @角色B参考图
导演参考：@CUT-XX批准storyboard reference board
```

如果某角色不出镜，必须明确写：

```text
本CUT不使用【角色名】角色参考。
【角色名】本CUT不出镜，不需要生成，不需要站位，不需要背景等待。
```

禁止只写“参考所有角色图”。这样容易把不该出镜的人物生成出来。

---

## 4. 人物数量锁定写法

必须写得像清点演员一样明确：

```text
本CUT只允许出现 X 名真人角色：
【角色A】、【角色B】。

不要出现【角色C】。
不要出现【角色D】。
不要出现额外白衣女子。
不要出现额外黑衣男人。
不要出现第二个【角色A】。
不要出现第二个【角色B】。
背景中不要出现长得像主角的人。
```

如果有道具人物，比如橱窗假人，必须写：

```text
橱窗假人不是人。
橱窗假人是道具。
橱窗假人不计入人物数量。
```

如果有黑七，必须写：

```text
黑七是小型黑色狐狸本体，不是真人，不是狗，不是狼，不是第二个角色人形。
黑七只出现一只。
```

---

## 5. 角色外形锁定写法

每个角色必须单独写，不要用泛称。

推荐写法：

```text
【角色名】：
严格参考@角色名三面图。
年龄、脸型、发型、服装、身材比例、气质保持一致。
本CUT的心理状态：……
本CUT的行为方式：……
不得变成……
不得出现第二个【角色名】。
```

禁止写法：

```text
handsome young man
cool young man
beautiful woman
mysterious man
刚才坐着的男子此时骑摩托
两名同伴
一名女子
一名男子
```

这些泛称会导致模型把角色混成默认脸或把两个男性合并。

---

## 6. 动作主体防混淆

如果有新角色入场，必须写：

```text
【新角色名】作为独立角色进入画面。
他/她不是【已有角色A】，不是【已有角色B】。
【已有角色A】仍在原位，不移动，不替换成新角色。
```

错误写法示例：

```text
刚才坐着的男子此时骑摩托车进入画面。
```

正确写法示例：

```text
沈泊川作为第五个独立角色，从画面右侧骑黑色摩托车进入。
他不是陆沉舟，不是韩知玄。
陆沉舟仍坐在左侧岩石上，不起身，不骑摩托。
```

---

## 7. 道具 / 假人 / 路人规则

对容易被模型生成人的元素，必须写清楚。

```text
假人是商店展示道具，不是真人，不计入人物数量。
假人不能移动，不能说话，不能眨眼，不能有真实人脸，不能和主角相似。
路人只能是远处模糊背景，不能清晰露脸，不能成为同伴，不能停下围观。
道具不能变成额外人物。
动物不能变成人。
```

---

## 8. 时间轴写法

每个时间段必须写：

```text
【0-1.5秒｜动作标题】
镜头：
人物站位：
动作：
表情：
视线：
道具状态：
禁止项：
```

不要只写“画面切换”或“表现动态过程”。模型需要具体动作和位置。

---

## 9. 表演和喜剧/情绪写法

要写角色为什么这样做，而不是只写动作。

例如：

```text
檀缺不是故意打断韩知玄，也不是耍脾气。
她只是认为事情已经结束，所以自然转身离开。

韩知玄不是生气，不是训斥。
他只是解释失败后的无奈，话说到一半自然停住。
```

这种写法能让动作更像演员表演，而不是机械执行。

---

## 10. 声音和留白

声音必须服务表演节奏。

```text
对白必须写 @角色名。
没说完的台词必须写清楚“被自然截断”。
笑点、尴尬、停顿可以用半秒留白。
BGM必须按source index；如果无音乐，不得新增。
环境音保持真实连续。
不要恐怖音效，除非source明确要求。
```

---

## 11. 输出前自检

每次输出最终视频提示词前检查：

```text
[ ] 本CUT出镜角色是否清楚？
[ ] 本CUT不出镜角色是否明确禁止？
[ ] 是否没有使用 handsome / cool / 一名男子 / 一名女子 等泛称？
[ ] 是否没有把一个角色写成另一个角色的动作主体？
[ ] 道具/假人/动物是否明确不计入真人角色？
[ ] 路人是否只作为模糊背景？
[ ] 导演分镜是否只用于站位、动作、镜头，不用于脸和服装？
[ ] 每个时间段是否有动作、表情、视线、道具状态？
[ ] 是否有关键限制重复禁止额外人物、重复人物、混脸？
```
