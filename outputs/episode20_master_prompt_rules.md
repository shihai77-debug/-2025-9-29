# EP20 Master Prompt Rules｜第20集视频提示词总规则

本文件用于 EP20 所有 CUT 的视频生成提示词。  
若与 `outputs/episode20_final_cut_source_index.md` 冲突，以 source index 为准。

---

## 1. 原文优先级

每个 CUT 的对白 / BGM / SFX 必须来自：

```text
outputs/episode20_final_cut_source_index.md
```

执行规则：

```text
对白必须逐字使用。
BGM必须逐字使用。
SFX必须逐字使用。
不得创作、改写、增删、重排。
如果BGM或SFX为“无”，提示词里必须明确写“无”，不得补写风声、纸张声、摩托车声、环境声。
```

---

## 2. 批准 storyboard board 使用规则

批准 board 只用于：

```text
构图
人物站位
动线
镜头节奏
时间分割
道具位置
空间关系
```

禁止用于：

```text
最终画风
角色临时外形
文字
箭头
编号
边框
英文标签
临时注释
对白创作
BGM创作
SFX创作
```

最终画面必须是高质量彩色真人电影画面，不是黑白分镜图，不是插画，不是草图。

## 3. 角色锁定

### 陆沉舟

```text
黑色短款皮夹克，黑色T恤，深色长裤，黑色军靴，年轻男性，黑色短发，疲惫但稳定。
禁止换成长风衣、镇灵司制服、古装、白衣、甲胄。
```

### 黑七

```text
必须是小型黑色狐狸。
黑色皮毛，橙黄或金色眼睛，尖耳，细长狐狸嘴，蓬松长尾。
不是黑狼，不是狗，不是猫，不是人，不是怪物。
```

### 檀缺

```text
深色现代日常装，深灰开衫或深色外套，深色上衣，黑色裙装，平静、认真、直接。
她不是白清檀，不是女王，不穿白衣，不穿黑色古风长裙。
```

### 韩知玄

```text
黑色短发，黑色长款镇灵司制服外套，胸前“镇灵司”标识。
不要写成“辑灵司”。
不要穿成陆沉舟。
不要穿普通长风衣。
```

### Shen

```text
黑色皮夹克，黑色裤子，黑色靴子，黑色背包，黑色摩托车头盔或手持头盔。
不是夸张英雄登场，不是AI默认帅哥海报。
```

## 4. EP20 CUT-02 固定规则

CUT-02 是 Shen 取出文件并说明情况的镜头。

```text
中央女性：檀缺，不是白清檀。
背景：BG A，同一片昆仑余波后的冷灰旷野。
远处：只有很淡的现代城市天际线剪影。
镜头：CAM HOLD，全程固定。
时间：11秒，0-5秒 + 5-11秒。
文件：只在 Shen 手中或背包附近。
限制：NO HANDOFF。
```

必须避免：

```text
文件交接
递文件
接文件
文件飞散
纸张满天飞
资料散落
背包内容全露
摩托车移动
陆沉舟起身
檀缺移动
韩知玄移动
```

## 5. 背景与风格

```text
昆仑余波后的冷灰旷野。
碎石地面，低矮荒坡，灰暗天空。
远处只有很淡的现代城市天际线剪影。
背景安静、低饱和、真实短剧电影感。
```

禁止：

```text
未来都市
赛博城市
末世片
科幻片
英雄登场海报
宫殿
山门
云海
传送门
强光屏障
```

---

## 6. 对白规则

```text
对白只能出现一次。
不得在多个段落重复同一句对白。
不得把一句对白拆成多次发言。
不得显示字幕。
如果source index里有对白，必须放进最终提示词内。
如果source index里写“无对白”，不得添加任何台词或旁白。
```

---

## 7. 音频规则

```text
BGM和SFX只看source index。
source index写“无”时，提示词必须写：无背景音乐，无音效，无环境声。
不得私自添加风声、纸张声、摩托车声、衣料声、音乐、合成器音。
```

---

## 8. 负面提示精简规则

负面提示只放本 CUT 最容易出错的内容。  
不要无限堆叠无关负面词。

EP20 CUT-02 常用负面：

```text
字幕，文字，箭头，UI，分镜边框，黑白分镜风格，白清檀，额外人物，黑狼，狗，黑七变成人，未来都市，赛博城市，末世感，科幻感，辑灵司，文件交接，递文件，接文件，文件飞散，资料散落，摩托车移动，重复对白，新增对白，新增BGM，新增SFX。
```
