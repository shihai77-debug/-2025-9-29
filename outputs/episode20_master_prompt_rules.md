# EP20 Master Prompt Rules｜第20集视频提示词总规则

本文件用于 EP20 所有 CUT 的视频生成提示词。若与 `outputs/episode20_final_cut_source_index.md` 冲突，以 source index 为准。

---

## 1. 原文优先级

每个 CUT 的对白 / BGM / SFX / 情绪 / 声音 / 时长必须来自：

```text
outputs/episode20_final_cut_source_index.md
```

执行规则：

```text
对白必须逐字使用。
BGM必须逐字使用。
SFX必须逐字使用。
不得创作、改写、增删、重排。
不得因为storyboard board没写声音就删除对白/BGM/SFX/环境音。
```

---

## 2. 批准 storyboard board 使用规则

批准 board 只用于：

```text
构图
人物站位
动线
镜头节奏
时间分割
道具位置
空间关系
```

禁止用于：

```text
最终画风
角色脸
角色发型细节
角色服装细节
角色体型
角色气质
文字
箭头
编号
边框
英文标签
临时注释
对白创作
BGM创作
SFX创作
```

必须写入视频提示词：

```text
批准 storyboard reference board 只用于站位、构图、动线、镜头、道具状态和连续空间关系；不要参考该board里的人脸、五官、发型细节、服装细节或人物气质。所有角色外貌必须严格以各自三面图/角色参考图为准。
```

最终画面必须是高质量彩色真人电影画面，不是黑白分镜图，不是插画，不是草图。

不能多出第二个陆沉舟、第二个沈泊川、第二个韩知玄、第二个檀缺，不能出现第二只黑七。

## 5. Clean keyframe 规则

批准 storyboard board 不是最终图生视频源图。

如果要稳定人物脸和身份，必须使用 clean keyframe：

```text
16:9彩色真人电影画面。
无文字、无箭头、无标签、无分镜边框。
人物脸严格来自三面图，不来自storyboard board。
说话者/动作主体脸部足够大，五官清晰稳定。
无重复人物，无额外路人，角色数量正确。
```

## 6. CUT-02 固定规则

CUT-02 是 Shen 取出文件并说明委托的镜头。

```text
中央女性：檀缺，不是白清檀。
背景：BG A，同一片昆仑余波后的冷灰旷野。
远处：只有很淡的现代城市天际线剪影。
镜头：CAM HOLD，全程固定。
原始分镜：11秒，0-5秒 + 5-11秒。
语音稳定版：15秒，0-6秒 + 6-15秒。
文件：只在 Shen 手中或背包附近。
限制：NO HANDOFF。
```

CUT-02 必须使用 source index 中两段沈泊川原文对白。禁止使用：

```text
“委托人只说，让我把这份情况带给你们。文件不外借。”
```

若视频模型同时生成口型和声音，CUT-02 必须使用15秒安全版，不要为了11秒强行压缩语速。

必须避免：

```text
文件交接
递文件
接文件
文件飞散
纸张满天飞
资料散落
背包内容全露
摩托车移动
陆沉舟起身
檀缺移动
韩知玄移动
```

## 8. 对白与音频规则

```text
对白只能来自source index。
对白只出现一次。
不得改写、删减、重排、创作。
不得显示字幕。
BGM和SFX只看source index。
如果source index写无，才可以写无。
如果source index有环境音/SFX，不得删除。
音量优先级：人声 > 核心音效 > 环境音 > BGM。
同一角色跨CUT声线、音量基准、语速习惯保持一致。
```
