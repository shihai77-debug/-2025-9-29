# EP20 Master Prompt Rules｜第20集视频提示词总规则

本文件用于 EP20 所有 CUT 的视频生成提示词。若与 `outputs/episode20_final_cut_source_index.md` 冲突，以 source index 为准。

---

## 1. 原文优先级

每个 CUT 的对白 / BGM / SFX / 情绪 / 声音 / 时长必须来自：

```text
outputs/episode20_final_cut_source_index.md
```

执行规则：

```text
对白必须逐字使用。
BGM必须逐字使用。
SFX必须逐字使用。
不得创作、改写、增删、重排。
不得因为storyboard board没写声音就删除对白/BGM/SFX/环境音。
```

## 2. 批准 storyboard board 使用规则

批准 board 只用于：

```text
构图
人物站位
动线
镜头节奏
时间分割
道具位置
空间关系
```

禁止用于：

```text
最终画风
角色脸
角色发型细节
角色服装细节
角色体型
角色气质
文字
箭头
编号
边框
英文标签
临时注释
对白创作
BGM创作
SFX创作
```

必须写入视频提示词：

```text
批准 storyboard reference board 只用于站位、构图、动线、镜头、道具状态和连续空间关系；不要参考该board里的人脸、五官、发型细节、服装细节或人物气质。所有角色外貌必须严格以各自三面图/角色参考图为准。
```

最终画面必须是高质量彩色真人电影画面，不是黑白分镜图，不是插画，不是草图。

## 10. 台词时长与演员式读法

每个 CUT 输出前必须先检查台词长度和原时长是否匹配。不能把长台词压缩成机械快读。

时长判断规则：

```text
本CUT无台词，不需要台词时长调整。
原时长可承载台词，不需要延长。
台词偏密，建议延长到 X 秒。
台词过密，建议拆成 CUT-XX-A / CUT-XX-B 两个视频节点。
```

台词密度参考：

```text
短句：1.5-3秒，保留吸气和句尾停顿。
中等句：3-6秒，保留自然断句。
长句：6-10秒，必须写停顿点、重音词、换气。
信息量大的长台词：8-12秒；仍然拥挤时必须延长或拆节点。
```

每句对白必须带演员式读法，不允许只写“正常语气”：

```text
声线基准：
音量连续性：
语速：
气息：
外层情绪：
内层情绪：
重音词：
停顿点：
尾音处理：
口型与视线：
身体配合：
```

表演要求：

```text
角色像真实演员在现场说话，不像朗读稿子。
情绪用呼吸、停顿、语速、重音、眼神和手部细节表达。
禁止突然拔高音量。
禁止全程一个调子念完。
禁止长台词无停顿。
禁止为了原秒数强行加速。
禁止跨CUT同一角色音量忽高忽低。
```

输出前新增自检：

```text
[ ] 是否检查台词长度和CUT时长？
[ ] 长台词是否已延长或拆节点？
[ ] 每句对白是否有演员式读法？
[ ] 是否写明重音、停顿、气息、视线和身体配合？
[ ] 同一角色声音是否承接上一CUT？
```

## 11. 演员脸部稳定与防抖

演员脸部稳定是独立检查项，不等同于“三面图优先”。即使写了三面图，如果视频源图是远景小脸、黑白分镜、多人同框、长台词加大幅镜头运动，仍然会出现脸部漂移和身份混乱。

必须写入最终视频提示词：

```text
演员脸部稳定锁：所有人物脸部严格以三面图/角色参考图为准，不参考storyboard board中的临时脸。说话者脸部清晰、五官稳定、身份不变；同一角色在整个CUT中保持同一张脸、同一发型、同一年龄感、同一气质。禁止脸部漂移、五官闪烁、换脸、混脸、脸部融化、背景复制主角脸。
```

脸部风险判断：

```text
说话者脸太小：必须生成clean keyframe或改成中近景。
远景多人同框：只用于站位，不适合承载长台词口型。
长台词 + 快速跟拍/横移：必须拆成动作节点和说话节点。
道具动作 + 长台词 + 小脸：必须延长或拆节点。
直接用带字分镜board做I2V：必须提示脸部漂移风险，建议先做clean keyframe。
```

镜头规则：

```text
长台词优先固定镜头、慢推或轻微呼吸感镜头。
说话时避免大幅横移、快速跟拍、快速变焦、频繁切角度。
动作复杂时先完成动作，再进入稳定说话构图。
脸部不清楚时，不要让模型同时承担口型、表情、走位和镜头运动。
```

新增自检：

```text
[ ] 是否使用clean keyframe而不是带字storyboard board直接I2V？
[ ] 说话者脸是否足够大、清晰、稳定？
[ ] 是否避免长台词同时叠加大幅镜头运动？
[ ] 必要时是否拆成动作节点/说话节点？
[ ] 是否写入演员脸部稳定锁？
[ ] 是否禁止脸部漂移、五官闪烁、换脸、混脸、背景复制主角脸？
```

## 12. 音频混音与BGM/SFX平衡

音频必须像成片混音一样分层，不能只列出声音元素。每个CUT都要判断人声、核心音效、环境音、BGM的相对关系，保证听感不闷、不糊、不空、不忽大忽小。

源文件边界：

```text
BGM/SFX/环境音必须来自source index。
如果source index写无音乐，不得新增BGM。
如果source index写无SFX，不得自行创作SFX。
如果source index有环境音或核心音效，不得删除。
```

每个CUT必须输出音频混音判断：

```text
音频混音判断：本CUT有对白，人声必须最前，BGM/SFX全部下压。
音频混音判断：本CUT无对白，环境音和核心音效负责空间感，BGM只做低层情绪铺底。
音频混音判断：source index写无音乐，本CUT只保留对白/音效/环境音，不新增BGM。
音频混音判断：上一CUT声音需要淡出/承接，本CUT必须写明渐入、渐弱或留白。
```

相对音量参考：

```text
人声：100，始终清楚居前。
核心音效：50-70，关键瞬间可短暂到75，但不能盖住人声。
环境音：18-30，持续铺底，不能抢戏。
BGM对白段：8-18，必须ducking下压。
BGM无对白段：18-35，按情绪轻推，不能突然煽情。
静默/余韵段：保留空气感，不要把所有声音填满。
```

执行规则：

```text
对白出现时，BGM自动下压，人声结束后BGM再轻微回升。
核心音效必须贴合动作时间点，不提前、不拖后。
同一场景环境音要连续，不能每个CUT换一种底噪。
转场处必须写明BGM/SFX/环境音是延续、淡出、渐入还是留白。
禁止声音忽大忽小，禁止闷、糊、远、空，禁止BGM压人声。
禁止把所有音效做得一样响，禁止无对白段完全死静除非source明确要求。
```

新增自检：

```text
[ ] 是否读取source index里的BGM/SFX/环境音？
[ ] 是否没有新增source index不存在的音乐或音效？
[ ] 对白段人声是否始终最清楚？
[ ] BGM是否在对白段下压？
[ ] 核心音效是否贴合动作时间点？
[ ] 环境音是否与上一CUT连续？
[ ] 转场处是否写明淡入、淡出、延续或留白？
[ ] 是否避免声音忽大忽小、发闷、发糊、BGM盖台词？
```

## 13. Storyboard 标签语言规则

Storyboard reference board 可以使用短标签、箭头、编号，但必须使用中文角色全名或中文短标签，禁止使用英文简称。

必须使用：

```text
陆沉舟
黑七
檀缺
韩知玄
沈泊川
陆沉舟位置锁定
黑七位置锁定
檀缺位置锁定
韩知玄位置锁定
沈泊川入场
沈泊川停下
无沈泊川
头盔戴着
头盔摘下
背包关闭
无资料
背景连续
固定全景
摩托车入场
```

禁止使用：

```text
LU
TAN
HAN
SHEN
HEIQI
LU POS LOCK
TAN POS LOCK
HAN POS LOCK
SHEN ENTER
HEIQI POS LOCK
```

原因：英文简称会削弱角色身份绑定，容易造成檀缺/白清檀、沈泊川/韩知玄、陆沉舟/沈泊川混淆。

Clean keyframe 与最终视频画面禁止出现任何标签、文字、箭头、编号或分镜边框。
