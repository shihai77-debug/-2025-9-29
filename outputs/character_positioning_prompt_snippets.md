# 角色定位图 Prompt Snippets

## 完整规则块

```text
【角色定位图使用规则】
本次上传的角色定位图不是最终画面素材，也不是复制粘贴素材。
它们只用于锁定人物身份、发型、服装、体型、姿态、轮廓和方向。

生成分镜分割图时，必须严格参考上传的角色定位图：
- 人物发型必须一致
- 服装款式必须一致
- 身高体型比例必须一致
- 正面/侧面/背面方向必须按定位图理解
- 角色气质必须一致
- 不得重新设计人物
- 不得把一个角色画成另一个角色

如果同一画面超过2个清晰人脸，不要画清楚五官。
可以保留脸部外轮廓、头部方向和发型轮廓。
人物身份通过发型、服装、身形、姿态、道具和画面位置识别。

角色定位图只用于分镜参考，不用于最终视频人脸。
最终视频人脸仍以三面图/角色参考图为准。
```
