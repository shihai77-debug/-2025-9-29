# 角色定位图使用规则｜分镜草图 / 分镜分割图专用

## 用途

本次上传的角色定位图不是最终画面素材，也不是复制粘贴素材。
它们只用于锁定人物身份、发型、服装、体型、姿态、轮廓和方向。

生成分镜分割图时，必须严格参考上传的角色定位图：

- 人物发型必须一致。
- 服装款式必须一致。
- 身高体型比例必须一致。
- 正面 / 侧面 / 背面方向必须按定位图理解。
- 角色气质必须一致。
- 不得重新设计人物。
- 不得把一个角色画成另一个角色。

## 多人画面脸部规则

如果同一画面超过 2 个清晰人脸，不要画清楚五官。
可以保留脸部外轮廓、头部方向和发型轮廓。
人物身份通过发型、服装、身形、姿态、道具和画面位置识别。

角色定位图只用于分镜参考，不用于最终视频人脸。
最终视频人脸仍以三面图 / 角色参考图为准。

## 角色定位锁定重点

### 陆沉舟无五官定位图

```text
用途：分镜草图 / storyboard reference board 中锁定陆沉舟外形。
锁定：短发轮廓、黑色皮夹克、黑T、黑裤、黑靴、年轻男性体型、沉默疲惫气质。
禁止：背包、摩托车、镇灵司长外套、韩知玄制服、沈泊川清洁服务员工身份。
最终视频脸：仍以陆沉舟三面图 / 角色参考图为准。
```

### 檀缺无五官定位图

```text
用途：分镜草图 / storyboard reference board 中锁定檀缺外形。
锁定：半扎长发、深灰开衫 / 外套、深色上衣、黑色百褶中长裙、黑色平底鞋、现代日常感、安静克制气质。
禁止：白衣、旗袍、古装、白清檀女王气质、黑色古风长裙。
最终视频脸：仍以檀缺三面图 / 角色参考图为准。
```

### 韩知玄无五官定位图

```text
用途：分镜草图 / storyboard reference board 中锁定韩知玄外形。
锁定：黑色短发、黑色长款镇灵司制服外套、黑衬衫、黑裤、黑靴、胸前“镇灵司”标识、公务式站姿、肩背挺直、克制警觉气质。
禁止：沈泊川清洁服务员工造型、陆沉舟皮夹克便装、背包、摩托车身份、把“镇灵司”写成“辑灵司”。
最终视频脸：仍以韩知玄三面图 / 角色参考图为准。
```

### 沈泊川角色参考图

```text
用途：分镜草图 / storyboard reference board / 角色设定参考中锁定沈泊川最新外形。
当前最新官方参考：@沈泊川角色参考图。旧的黑色皮夹克骑手参考作废，不再作为沈泊川外形标准。
角色身份：20多岁清洁服务公司外勤员工 / 青年委托人，不是学生，不是男主，不是镇灵司人员。
锁定：20代年轻男性脸型与发型、清洁服务/外勤工作者气质、工作衬衫外套、灰或深灰内搭、棕灰/卡其系工装裤、耐磨工作鞋、实用双肩包或工具包。
允许：背包、文件、委托人身份、工作包、清洁服务员工的现实职场感。
禁止：黑色皮夹克、黑T、黑裤、黑靴、全黑男主同款、摩托骑手重装、头盔作为固定身份、镇灵司长外套、韩知玄制服、陆沉舟低头坐姿、商务西装大叔感、学生感、白色休闲度假风。
最终视频脸：以最新沈泊川角色参考图为准。
```

## 禁止

```text
不要凭空设计新脸。
不要把角色定位图当成最终照片。
不要复制角色定位图的白底背景。
不要把定位图直接贴进画面。
不要画清楚多人小脸五官。
不要让檀缺变成白清檀。
不要让沈泊川变成韩知玄。
不要让沈泊川变成陆沉舟。
不要让沈泊川穿回黑色皮夹克、黑T、黑裤、黑靴。
不要让沈泊川变成学生、商务大叔、镇灵司人员或摩托骑手重装。
不要让韩知玄变成沈泊川。
不要让陆沉舟变成沈泊川或韩知玄。
不要让黑七变成狗、狼、人。
```

## 核心区分

```text
分镜草图 / 分镜分割图 / storyboard reference board = 位置、外形、轮廓、动作、镜头参考。
最终视频提示词 / clean keyframe = 三面图 / 角色参考图为脸部最高优先级。
```
