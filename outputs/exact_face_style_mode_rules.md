# Exact Face And Style Mode Rules｜脸部与风格精确模式

本文件用于用户要求“脸和风格准确”“不要模糊处理”“按三面图精确生成”时使用。

普通远景稳定策略可以用侧脸、远景、柔焦降低脸部风险；但精确模式不能靠模糊逃避，必须先建立可复用的角色脸部和风格基准。

---

## 1. 精确模式适用场景

使用精确模式：

```text
角色第一次正式露脸。
角色说重要台词。
角色有情绪特写。
用户明确要求脸和风格准确。
需要和三面图高度一致。
需要生成后续多CUT复用的clean keyframe。
```

不优先使用精确模式：

```text
远景站位确认。
摩托车入场、走位、背影、侧影等动作连接。
无台词的气氛镜头。
多人远景只看站位。
```

这些可以使用“电影感柔焦 / 中远景 / 侧脸 / 背影”策略。

---

## 2. 精确模式固定流程

必须按顺序执行：

```text
1. 三面图 / 角色参考图
2. face lock image（角色脸部锁定图）
3. style lock image（本集画面风格锁定图，可选但推荐）
4. clean keyframe（无文字、无箭头、无分镜框的16:9彩色电影画面）
5. Jimeng / Seedance 图生视频提示词
```

禁止直接用 storyboard reference board 做最终I2V源图来追求脸部准确。

---

## 3. Face Lock Image 规则

每个主要角色至少做一张 approved face lock image。

Face lock image 必须：

```text
单人画面。
正面或3/4侧面中近景。
脸部占画面足够大。
服装、发型、年龄感、气质严格来自三面图。
背景简单，不抢戏。
无文字、无标签、无箭头。
无其他人物。
```

Face lock image 提示词必须写：

```text
严格参考@角色名三面图，保持同一张脸、同一发型、同一年龄感、同一体型比例、同一气质。
不要AI默认帅哥/美女脸。
不要改变五官比例。
不要改变发型。
不要改变服装轮廓。
```

---

## 4. Style Lock Image 规则

如果一集里多CUT要统一画风，建议做一张 style lock image。

Style lock image 用于：

```text
色温。
对比度。
光线质感。
镜头焦段感。
城市 / 老宅 / 巷口 / 昆仑余波等场景统一气氛。
```

Style lock 不用于改变人物脸。

必须写：

```text
风格参考只用于光线、色调、质感和电影感，不改变角色脸、发型、服装、体型。
```

---

## 5. Clean Keyframe 精确规则

精确模式下的 clean keyframe 是真正给 Jimeng 的源图。

Clean keyframe 必须：

```text
16:9 横版彩色真人电影画面。
无文字、无箭头、无标签、无分镜边框。
角色脸来自 face lock image + 三面图。
构图和站位来自 storyboard board。
画面风格来自 style lock / 本集风格规则。
每个角色只出现一次。
背景没有长得像主角的额外人物。
```

提示词必须写：

```text
storyboard reference board 只参考站位、构图、镜头、动作节奏和道具关系；不要参考该board里的人脸、临时服装、黑白画风、文字、箭头、编号、边框。
```

---

## 6. 说话镜头精确规则

如果角色说台词并要求脸准确：

```text
说话者必须中近景或近景。
脸部必须清晰。
镜头优先固定、慢推或极轻呼吸感。
不要大幅横移、快速跟拍、快速变焦。
不要让角色一边大动作一边说长台词。
长台词必须延长或拆成多个节点。
```

如果同一CUT既有动作又有长台词，拆成：

```text
动作节点：完成走位、拿道具、转身、入场。
说话节点：稳定构图，中近景，说台词。
```

---

## 7. 精确模式提示词固定段落

每个精确模式视频提示词必须包含：

```text
【脸部与风格精确模式】
本CUT要求角色脸部和画面风格准确，不使用模糊处理来掩盖脸部问题。
所有角色脸部、五官、发型、年龄感、体型、服装轮廓、气质必须严格参考各自三面图和已批准face lock image。
画面光线、色调、质感参考本集style lock和当前场景规则。
storyboard reference board只参考站位、构图、镜头、动作节奏和道具关系，不参考人物脸、临时服装、黑白画风、文字、箭头、编号、边框。
禁止AI默认帅哥/美女脸，禁止换脸，禁止混脸，禁止五官漂移，禁止脸部融化，禁止背景人物长出主角脸。
```

---

## 8. 精确模式负面提示

必须加入：

```text
AI默认帅哥脸，AI默认美女脸，换脸，混脸，五官漂移，脸部闪烁，脸部融化，发型改变，服装改变，年龄改变，体型改变，背景复制主角脸，额外人物，重复人物，分镜文字，箭头，标签，边框，黑白故事板风格。
```

---

## 9. 精确模式与柔焦模式的选择

每个CUT输出前必须判断：

```text
脸部策略判断：本CUT需要精确露脸，使用Exact Face And Style Mode。
脸部策略判断：本CUT主要是远景/走位/气氛，使用电影感柔焦和轮廓识别，不强求正脸细节。
脸部策略判断：本CUT有长台词，必须使用clean keyframe + 中近景稳定说话节点。
脸部策略判断：本CUT含动作和台词，建议拆成动作节点与说话节点。
```
