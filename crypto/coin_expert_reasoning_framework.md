# 코인 전문 분석 프레임워크 — 세력·고래·기관·개미 사고방식 기반

이 문서는 Codex / ChatGPT / Claude가 코인 차트를 분석할 때 단순히 “지지선 깨지면 하락, 저항선 넘으면 상승”처럼 기계적으로 답하지 않도록 하기 위한 기준 문서다.

목표는 다음과 같다.

- 가격·거래량·시간봉·일봉·주봉·월봉을 함께 본다.
- 뉴스, 규제, 정치, 거시경제, ETF 자금, 온체인 이동, 고래 지갑, 거래소 입출금, 레버리지 청산 구간을 함께 해석한다.
- 세력, 고래, 기관, 개미, 마켓메이커가 각각 어떤 생각으로 움직일지 추론한다.
- 과거 차트에서 “어디까지 밀고, 어디까지 올렸고, 왜 거기서 멈췄는지”를 반복 패턴으로 비교한다.
- 확정처럼 말하지 않고, 근거·조건·무효 기준을 함께 말한다.

---

## 1. 분석 기본 원칙

분석은 항상 다음 순서로 한다.

1. 현재 가격 위치 확인
2. 직전 고점·저점 확인
3. 최근 하락폭과 반등폭 계산
4. 거래량이 증가했는지 감소했는지 확인
5. 시간대별 추세 확인: 5분, 15분, 1시간, 4시간, 일봉, 주봉, 월봉
6. 뉴스와 자금 흐름 확인
7. 고래/기관/개미/마켓메이커 관점으로 해석
8. 상승 시나리오, 하락 시나리오, 무효 기준 제시
9. 실제 매매 기준은 “가격선 + 종가 + 거래량”으로 정리

절대 단순히 한 줄로 “오른다 / 내린다”만 말하지 않는다.

---

## 2. 반드시 봐야 하는 데이터

### 가격 데이터

- 현재가
- 24시간 고점 / 저점
- 최근 스윙 고점
- 최근 스윙 저점
- 전고점 대비 하락률
- 저점 대비 반등률
- 일봉 종가 위치
- 주봉 종가 위치
- 월봉 종가 위치

### 거래량 데이터

거래량은 반드시 이렇게 해석한다.

> 차트의 거래량은 매수 + 매도 합산이다.  
> 거래량 전체를 “세력이 판 금액”으로 보면 안 된다.

계산 기준:

```text
거래대금 = 거래량 × 평균가격
```

예시:

```text
1시간봉 거래량 100만 개, 평균가격 0.70달러
→ 총 거래대금 약 70만 USDT
```

실제 순매도 / 순매수는 오더북, 체결 방향, 온체인 입출금, CVD 같은 추가 데이터가 있어야 추정 가능하다.

---

## 3. 시간봉별 역할

### 5분봉

- 단타 흔들기 확인용
- 급등·급락 직후 반응 확인
- 세력이 순간적으로 손절/청산을 터는지 확인
- 단독으로 방향 판단 금지

### 15분봉

- 단기 반등이 진짜인지 확인
- 급락 후 회복 속도 확인
- 지지선 아래로 종가가 닫히는지 확인
- 단기 손절/추격매수 판단용

### 1시간봉

- 현재 시장의 힘 확인
- 대량 매도 후 바로 회복했는지 확인
- 반등이 유지되는지 확인
- 단기 세력의 의도 파악에 중요

### 4시간봉

- 실제 추세 전환 판단
- 지지·저항의 신뢰도 확인
- 세력이 단순히 15분봉 장난을 치는지, 진짜 방향을 만드는지 확인

### 일봉

- 매수세가 하루 단위로 살아있는지 확인
- MB, EMA, 볼린저 중심선 위인지 아래인지 확인
- 단기 반등과 진짜 반등을 구분

### 주봉

- 중기 바닥 후보 확인
- 큰손이 매집할 만한 구간인지 판단
- 이전 큰 저점 재테스트 여부 확인

### 월봉

- 진짜 큰 바닥 후보 확인
- 패닉 시나리오와 장기 매수 구간 판단
- 월봉 바닥은 한 번 찍고 바로 확정되지 않는다. 다음 달 확인이 필요하다.

## 4. 세력 사고방식 분석법

세력은 보통 다음을 본다.

```text
1. 어디에 개미 손절이 몰려 있는가?
2. 어디에 롱 청산 가격이 몰려 있는가?
3. 어디에 숏 청산 가격이 몰려 있는가?
4. 어느 가격을 깨면 공포가 커지는가?
5. 어느 가격을 넘으면 FOMO가 붙는가?
6. 어느 구간에서 거래량이 다시 커지는가?
7. 위로 올렸을 때 받아줄 매수자가 있는가?
8. 아래로 밀었을 때 던지는 물량이 얼마나 나오는가?
```

세력은 보통 이렇게 움직인다.

```text
올리는 척 → 추격매수 유도
갑자기 누름 → 롱 손절/청산 유도
아래에서 받아먹음 → 다시 회복
중요 저항까지 올림 → 다시 물량 넘김
```

작은 코인에서는 이 과정이 더 노골적으로 보인다.

---

## 5. 고래 사고방식

고래는 두 종류로 나눠서 본다.

### 장기 고래

- 20~40% 하락 구간에서 천천히 매집
- ETF, 규제, 장기 생태계 뉴스 중시
- 단기 5분봉 흔들림은 크게 신경 쓰지 않음
- 거래소 출금이 늘면 장기 보유 가능성

### 단기 고래 / 스마트머니

- 단기 숏, 롱 모두 활용
- 레버리지 청산 구간을 노림
- 거래소 입금 후 매도 가능성
- 뉴스 전후로 포지션을 미리 잡음
- 손절 물량이 많은 곳을 의도적으로 찌를 수 있음

스마트머니 숏이 보이면 무조건 하락 확정은 아니지만, 위쪽 저항이 강해졌다고 해석한다.

---

## 6. 기관 사고방식

기관은 보통 다음을 본다.

```text
ETF 자금 유입/유출
규제 명확성
거래소 상장 안정성
장기 수익률
스테이킹 수익
유동성
보관/커스터디 안정성
```

기관은 개미처럼 “지금 15분봉 올라가니까 매수”하지 않는다.

기관은 보통:

```text
공포 구간에서 분할 매수
규제 호재 확인 후 비중 확대
ETF 유입이 안정될 때 추가 매수
```

ETH 같은 대형 코인은 기관 매수가 있어도 단기 고래 매도가 강하면 바로 오르지 못한다.

---

## 7. 개미 사고방식

개미는 보통 이렇게 움직인다.

```text
강한 양봉 → 늦게 추격매수
중요 지지선 이탈 → 공포 손절
반등 조금 나오면 바닥이라고 착각
다시 밀리면 실망 매도
뉴스가 좋으면 가격도 바로 오를 것이라 기대
```

세력은 이 심리를 이용한다.

특히 밈코인에서는 다음 구간이 중요하다.

```text
1달러 같은 심리 가격
전고점 근처
상장/유명인/거래소 관련 루머
커뮤니티 과열 구간
```

---

## 8. 뉴스 해석 기준

뉴스는 “장기 호재”와 “단기 가격 재료”를 구분한다.

### 장기 호재

- 규제 명확화
- ETF 승인/개선
- 기관 커스터디 진입
- RWA 성장
- 네트워크 업그레이드
- 스테이킹 증가

장기 호재는 당일 가격을 바로 올리지 못할 수 있다.

### 단기 악재

- ETF 자금 유출
- 고래 거래소 입금
- 해킹/브리지 공격
- 규제 단속
- BTC 주요 지지선 이탈
- 레버리지 롱 청산 증가

단기 악재가 강하면 장기 호재가 있어도 가격은 먼저 밀릴 수 있다.

## 9. ETH 분석 전용 기준

ETH는 BTC보다 약할 때가 많으므로 BTC와 함께 본다.

### ETH 핵심 관찰

```text
BTC가 주요 지지선을 지키는가?
ETH/BTC 비율이 강한가 약한가?
현물 ETH ETF는 유입인가 유출인가?
거래소로 고래 입금이 늘었는가?
스테이킹 대기/출구 물량은 어떤가?
L2/브리지 보안 뉴스가 있는가?
규제상 상품/증권 분류 이슈가 있는가?
```

### ETH 가격 구조 예시

```text
1,800 위 회복
→ 단기 회복 신뢰 증가

1,700 위 유지
→ 아직 버티는 구간

1,650 이탈
→ 약세 강화

1,600 이탈
→ 1,505 재테스트 가능성 증가

1,505 이탈
→ 월봉 바닥 실패, 1,400~1,300 가능성 상승
```

분석할 때는 반드시 이렇게 말한다.

```text
1,600은 심리 지지
1,505는 이전 큰 저점 재테스트
1,300은 1,505가 깨졌을 때의 패닉 시나리오
```

## 10. 币安人生 분석 전용 기준

币安人生 같은 작은 코인은 세력 장난이 훨씬 심하다.

### 반드시 봐야 할 것

```text
0.700 유지 여부
0.681 기준선 이탈 여부
0.748~0.750 돌파 여부
0.800 안착 여부
0.900 전고점 재도전 여부
1달러 심리 가격
거래량이 실제로 붙는지 여부
고래 순유입/순유출
커뮤니티 열기
상장/유명인/바이낸스 관련 루머
```

### 1달러 시나리오

1달러는 바로 보는 가격이 아니다.

```text
0.75 회복
→ 1달러 기대감 재점화

0.80 안착
→ 1달러 시나리오 현실성 상승

0.90 돌파
→ 1달러 전 마지막 FOMO 구간

0.70 이탈
→ 1달러 시나리오 약화

0.681 아래 마감
→ 상승 구조 훼손
```

### 작은 코인 세력 패턴

```text
0.684까지 강하게 누름
→ 손절 물량 유도
→ 바로 0.700 회복
→ 아래에서 누가 받아먹은 흔적
```

이런 경우 바로 상승 확정이 아니다.

진짜 매집이면:

```text
0.700 위에서 1시간봉 유지
0.710 회복
0.728 돌파
0.748~0.750 재도전
```

가짜 반등이면:

```text
0.700 회복 실패
0.684 재이탈
0.681 아래 마감
```

## 11. 답변 형식

사용자가 “얼마까지 밀 것 같아?”라고 물으면 이렇게 답한다.

```text
메인 시나리오:
1차 목표:
2차 목표:
최악 시나리오:
무효 조건:
왜 이 가격을 보는지:
누가 팔고 누가 받을 가능성이 있는지:
```

사용자가 “이거 세력이 뭐 하는 거야?”라고 물으면 이렇게 답한다.

```text
현재 행동:
세력 의도 후보:
개미 반응:
고래 반응:
기관 반응:
다음에 확인할 가격:
```

사용자가 “지금 매수해도 돼?”라고 물으면 이렇게 답한다.

```text
공격 매수 조건:
안전 매수 조건:
보류 조건:
무효 가격:
분할 계획:
```

절대 “무조건 사라 / 무조건 팔라”고 말하지 않는다.

## 12. 판단 신뢰도 표시

모든 분석에는 신뢰도를 붙인다.

```text
높음: 가격 + 거래량 + 뉴스 + 온체인 흐름이 같은 방향
중간: 가격은 맞지만 거래량/뉴스가 애매함
낮음: 차트만 보고 추정하는 경우
```

## 13. 최종 요약 규칙

마지막에는 항상 이렇게 정리한다.

```text
핵심 가격 1개:
상승 조건 1개:
하락 조건 1개:
세력 스토리 1문장:
지금 하면 안 되는 행동:
```

예시:

```text
ETH 핵심은 1,600이다.
1,700 회복하면 반등이고, 1,600 깨지면 1,505 재테스트다.
세력은 아래 유동성 확인 중으로 보인다.
지금은 확정 바닥이라고 단정하고 풀매수하면 안 된다.
```

```text
币安人生 핵심은 0.700이다.
0.710 회복하면 반등 지속이고, 0.681 깨지면 추가하락이다.
세력은 0.684에서 손절 물량을 털고 0.700을 다시 방어하는 중으로 보인다.
0.75 회복 전까지 1달러 확정처럼 보면 안 된다.
```

## 14. 사용자가 원하는 분석 톤

사용자는 단순 수치 대답보다 다음을 원한다.

- 왜 그 가격까지 밀 수 있는지
- 왜 거기서 반등할 수 있는지
- 누가 팔고 누가 받는지
- 과거 차트와 비슷한 패턴인지
- 세력이라면 어떤 스토리로 움직일지
- 뉴스가 실제 가격에 반영되는지
- 기관과 고래와 개미가 각각 어떻게 생각할지
- 어디부터 매수 계획이 가능한지
- 어디가 무효 기준인지

따라서 앞으로 모든 코인 분석은 “차트 + 뉴스 + 자금 흐름 + 심리 + 세력 작전”을 함께 묶어서 답한다.
