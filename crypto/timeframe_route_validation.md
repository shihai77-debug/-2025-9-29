# 시간대별 예측 경로 검증 시스템 — 짧은 시간·중간 시간·긴 시간 분리

이 문서는 `crypto/self_improving_crypto_agent.md`를 보충한다.

핵심 목적은 예측이 맞았는지 틀렸는지를 하나의 시간 기준으로만 판단하지 않고, 짧은 시간·중간 시간·긴 시간으로 나누어 검증하는 것이다.

코인 시장에서는 짧은 시간에는 맞고 긴 시간에는 틀릴 수 있으며, 반대로 짧은 시간에는 틀린 것처럼 보여도 긴 시간에는 맞을 수 있다.

따라서 모든 분석은 반드시 시간대별 경로를 분리한다.

---

## 1. 왜 시간대를 나눠야 하는가

예시:

```text
분석: ETH 1,620을 지키면 1,650 테스트 가능
```

짧은 시간에는:

```text
1시간 안에 1,650 터치
→ 단기 예측 성공
```

중간 시간에는:

```text
4시간 뒤 1,650 위 안착 실패 후 다시 1,600
→ 중기 예측 실패 또는 약한 반등
```

긴 시간에는:

```text
다음날 1,552 재테스트
→ 큰 구조는 여전히 약세
```

이처럼 한 예측도 시간대별로 결과가 다를 수 있다.

---

## 2. 예측 시간대 분류

분석할 때 반드시 아래 4개 시간대를 구분한다.

### 초단기 경로

```text
검증 시간: 15분 ~ 1시간
사용 차트: 5분봉, 15분봉, 30분봉
목적: 급락 후 반등, 손절 털기, 단타 반응 확인
```

주요 질문:

```text
지지선 아래로 찌르고 바로 회복했는가?
5분/15분 MB 위로 회복했는가?
거래량이 터진 뒤 바로 말아 올렸는가?
```

---

### 단기 경로

```text
검증 시간: 1시간 ~ 4시간
사용 차트: 1시간봉, 4시간봉 일부
목적: 단기 반등 지속 여부 확인
```

주요 질문:

```text
1시간봉 중심선 위에서 마감했는가?
첫 저항까지 도달했는가?
반등 후 다시 지지선 위에서 버티는가?
```

---

### 중기 경로

```text
검증 시간: 4시간 ~ 24시간
사용 차트: 4시간봉, 일봉
목적: 진짜 추세 회복인지 확인
```

주요 질문:

```text
4시간봉 MB를 회복했는가?
일봉 아래꼬리 방어가 성공했는가?
BTC 4시간 구조가 같이 회복했는가?
```

---

### 장기 경로

```text
검증 시간: 1일 ~ 1주 이상
사용 차트: 일봉, 주봉, 월봉
목적: 바닥 후보가 진짜 바닥인지 확인
```

주요 질문:

```text
일봉 저점이 높아졌는가?
주봉 지지선이 유지됐는가?
월봉 큰 바닥 후보가 깨지지 않았는가?
ETF/기관 자금 흐름이 개선됐는가?
```

## 3. 시간대별 예측 기록 템플릿

분석을 기록할 때는 아래 형식을 사용한다.

```text
날짜/시간:
코인:

초단기 경로:
- 예상:
- 성공 조건:
- 실패 조건:
- 검증 시간:

단기 경로:
- 예상:
- 성공 조건:
- 실패 조건:
- 검증 시간:

중기 경로:
- 예상:
- 성공 조건:
- 실패 조건:
- 검증 시간:

장기 경로:
- 예상:
- 성공 조건:
- 실패 조건:
- 검증 시간:
```

## 4. ETH 예시

```text
날짜/시간: 2026-06-25
코인: ETH
현재가: 1,629

초단기 경로:
- 예상: 1,620 위 유지 시 1,650 테스트
- 성공 조건: 15분/1시간봉 1,620 위 유지 후 1,650 터치
- 실패 조건: 1,620 아래 빠른 재이탈
- 검증 시간: 1시간

단기 경로:
- 예상: 1,650 돌파 시 1,680 테스트
- 성공 조건: 1시간봉 1,650 위 마감
- 실패 조건: 1,650에서 윗꼬리 후 1,620 아래 재하락
- 검증 시간: 4시간

중기 경로:
- 예상: 1,680 회복 전까지 4시간 구조는 약세
- 성공 조건: 4시간봉 1,680 위 마감
- 실패 조건: 1,600 아래 4시간봉 마감
- 검증 시간: 24시간

장기 경로:
- 예상: 1,505는 큰 바닥 후보, 확정 바닥 아님
- 성공 조건: 1,505 위에서 주봉 반등 유지
- 실패 조건: 1,505 아래 주봉/월봉 마감
- 검증 시간: 1주~1개월
```

## 5. 币安人生 예시

```text
코인: 币安人生
현재가: 0.700 근처

초단기 경로:
- 예상: 0.700 위 유지 시 0.710 테스트
- 성공 조건: 15분/1시간봉 0.700 위 마감
- 실패 조건: 0.700 아래 빠른 재이탈
- 검증 시간: 1시간

단기 경로:
- 예상: 0.710 회복 시 0.728 테스트
- 성공 조건: 1시간봉 0.710 위 마감
- 실패 조건: 0.710 돌파 실패 후 0.684 재이탈
- 검증 시간: 4시간

중기 경로:
- 예상: 0.728 돌파 전까지는 방어 구간
- 성공 조건: 4시간봉 0.728 위 마감
- 실패 조건: 0.681 아래 마감
- 검증 시간: 24시간

장기 경로:
- 예상: 0.80 안착 전까지 1달러 시나리오는 초기 단계
- 성공 조건: 0.80 위 일봉 안착
- 실패 조건: 0.6368 이탈
- 검증 시간: 1주 이상
```

## 6. 시간대별 성공/실패를 따로 기록하기

예측 결과는 반드시 이렇게 나눠 기록한다.

```text
초단기: 성공 / 실패 / 보류
단기: 성공 / 실패 / 보류
중기: 성공 / 실패 / 보류
장기: 성공 / 실패 / 보류
```

예시:

```text
ETH 1,620 유지 → 1,650 테스트 예측

초단기 결과: 성공. 1시간 안에 1,650 터치.
단기 결과: 실패. 1,650 위 마감 실패.
중기 결과: 보류. 4시간봉 1,680 회복 아직 없음.
장기 결과: 보류. 1,505 지지 여부 아직 확인 전.
```

## 7. 시간대별 오류 분석

틀렸을 때는 어느 시간대에서 틀렸는지 분리한다.

### 초단기 오류

```text
5분/15분봉 반응을 과대평가함
순간 거래량을 진짜 매집으로 착각함
급락 후 숏 익절 반등을 강한 매수로 오해함
```

### 단기 오류

```text
1시간봉 마감 확인 전에 판단함
첫 저항에서 막힐 가능성을 낮게 봄
BTC 단기 방향을 덜 반영함
```

### 중기 오류

```text
4시간봉 MB 회복 전인데 추세 회복으로 착각함
일봉 중심선 아래 약세를 덜 반영함
ETF 유출/고래 매도를 과소평가함
```

### 장기 오류

```text
바닥 후보를 바닥 확정으로 말함
주봉/월봉 마감 확인 없이 장기 반등을 확정함
거시 자금 흐름을 무시함
```

## 8. 시간대별 자가업데이트 규칙

틀린 시간대에 따라 규칙을 다르게 수정한다.

### 초단기 규칙 수정

```text
5분/15분봉 회복만으로 강한 반등이라고 말하지 않는다.
초단기 반등은 1시간봉 마감 전까지 임시 반등으로 본다.
```

### 단기 규칙 수정

```text
1시간봉 중심선 위 마감 전까지 단기 회복으로 인정하지 않는다.
첫 저항 돌파 실패 시 반등 확률을 낮춘다.
```

### 중기 규칙 수정

```text
4시간봉 MB 회복 전까지 추세 회복이라고 말하지 않는다.
BTC가 4시간봉 MB 아래면 알트/ETH 반등 확률을 낮춘다.
```

### 장기 규칙 수정

```text
일봉/주봉/월봉 마감 확인 전까지 바닥 확정이라고 말하지 않는다.
바닥 후보와 바닥 확정을 반드시 구분한다.
```

## 9. 최종 답변에 시간대를 반드시 넣기

코인 분석 최종 답변은 아래처럼 시간대별로 말한다.

```text
초단기:
1,620 지키면 1,650 테스트 가능

단기:
1,650 위 1시간봉 마감하면 1,680 가능

중기:
1,680 위 4시간봉 마감 전까지 추세 회복 아님

장기:
1,505는 바닥 후보일 뿐, 주봉/월봉 확인 전까지 바닥 확정 아님
```

## 10. 핵심 원칙

```text
한 예측은 하나의 시간으로 맞고 틀리는 것이 아니다.
짧은 시간에는 맞고 긴 시간에는 틀릴 수 있다.
긴 시간에는 맞지만 짧은 시간에는 흔들릴 수 있다.
따라서 모든 코인 분석은 초단기·단기·중기·장기로 분리해 검증한다.
```

이 구조를 따르면 자가업데이트가 더 정확해진다.

```text
어느 시간대에서 판단이 맞았는지
어느 시간대에서 판단이 틀렸는지
왜 그 시간대에서 오류가 났는지
다음에는 어떤 시간대 규칙을 고칠지
```

이렇게 구분해야 전문가식 분석이 된다.
