# 시장 데이터 수집 설계

목적: 코인 분석에서 필요한 공개 데이터를 정리하고, 사용자가 차트를 매번 보내지 않아도 검토할 수 있는 구조를 설계한다.

## 1차 수집 대상

```text
ETHUSDT 가격과 캔들
BTCUSDT 가격과 캔들
거래량
볼린저 중심선
RSI
MACD
펀딩비
미결제약정
```

## 시간대

```text
5분봉
15분봉
1시간봉
4시간봉
일봉
```

## 기록 목적

```text
초단기 경로 검증
단기 경로 검증
중기 경로 검증
장기 경로 검증
맞은 이유 기록
틀린 이유 기록
반복 오류 확인
```

## 운영 원칙

```text
공개 데이터만 사용한다.
분석 기록만 남긴다.
실제 행동은 사용자가 직접 결정한다.
규칙 변경은 사용자 승인 후 반영한다.
```
