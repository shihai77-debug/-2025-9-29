# 실전 코인 분석 체크시스템 — 점수표·세력 패턴·과거 사례 데이터베이스

이 문서는 기존 두 문서를 보충하는 3차 실전용 기준이다.

- `crypto/coin_expert_reasoning_framework.md`
- `crypto/advanced_human_like_market_reasoning.md`

앞의 두 문서가 “생각하는 법”이라면, 이 문서는 실제 분석할 때 빠뜨리지 않게 만드는 체크리스트와 점수표다.

목표는 다음과 같다.

```text
감으로 말하지 않는다.
한 방향만 보지 않는다.
세력 작전을 패턴으로 분류한다.
상승/하락/가짜반등 가능성을 점수로 비교한다.
과거 유사 사례를 저장해서 다음 분석 때 비교한다.
```

## 1. 분석 전 필수 입력 데이터 체크리스트

분석 전에 가능한 한 아래 데이터를 확인한다.

```text
코인명:
현재가:
24시간 고점:
24시간 저점:
최근 7일 고점:
최근 7일 저점:
직전 큰 고점:
직전 큰 저점:
현재 거래량:
이전 평균 거래량:
거래량 증가/감소:
BTC 방향:
ETH 방향:
시장 전체 분위기:
관련 뉴스:
온체인 입출금:
고래 순매수/순매도:
ETF 유입/유출:
청산 데이터:
펀딩비:
오픈이자:
커뮤니티 분위기:
```

데이터가 없으면 추정이라고 명확히 말한다.

```text
온체인 데이터가 없으므로 정확한 주체는 확인 불가.
차트와 거래량 기준으로만 추정한다.
```

## 2. 세력 작전 패턴 분류표

분석할 때 현재 움직임을 아래 패턴 중 어디에 가까운지 분류한다.

### 패턴 1 — 손절 털고 매집

특징:

```text
중요 지지선 살짝 이탈
긴 아래꼬리
하락 때 거래량 급증
바로 지지선 위 회복
이후 1시간봉/4시간봉에서 버팀
```

해석:

```text
세력이 아래 손절 물량과 롱 청산 물량을 받아먹었을 가능성.
```

확인 조건:

```text
이탈한 지지선 재회복
회복 후 거래량 유지
다음 저항 돌파
```

실패 조건:

```text
같은 저점 재이탈
지지선 아래 종가 마감
회복봉 거래량이 약함
```

---

### 패턴 2 — 위로 올리는 척하고 분배

특징:

```text
저점에서 반등
뉴스나 커뮤니티 기대감 증가
거래량은 크게 늘지 않음
중요 저항에서 계속 윗꼬리
고래 순유출 또는 거래소 입금 증가
```

해석:

```text
가격은 올리지만 실제 목적은 위에서 물량 넘기기일 수 있다.
```

확인 조건:

```text
저항 돌파 실패
윗꼬리 반복
반등 때 거래량보다 하락 때 거래량이 큼
```

무효 조건:

```text
저항 돌파 후 지지 전환
거래량 증가와 함께 상위 시간봉 마감
```

---

### 패턴 3 — 박스권에서 개미 지치게 만들기

특징:

```text
같은 구간 반복 횡보
위로 가는 척 실패
아래로 깨는 척 회복
거래량 점점 감소
커뮤니티 관심 감소
```

해석:

```text
세력이 방향을 만들기 전 물량을 흡수하거나 개미를 지치게 만드는 구간.
```

확인 조건:

```text
박스 상단/하단 반복
거래량 수축
돌파 전 급격한 거래량 증가 대기
```

---

### 패턴 4 — 뉴스 호재로 FOMO 만들고 매도

특징:

```text
호재 뉴스 발생
강한 단기 상승
커뮤니티 과열
전고점 근처에서 거래량 급증
이후 긴 윗꼬리 또는 급락
```

해석:

```text
뉴스를 미끼로 늦게 들어온 개미에게 물량을 넘기는 구조일 수 있다.
```

확인 조건:

```text
뉴스 후 상승률이 과도함
저항에서 고래 순유출
거래량 큰데 가격이 더 못 감
```

---

### 패턴 5 — 시장 전체 하락 이용한 저점 흡수

특징:

```text
BTC/ETH 동반 하락
알트 전체 하락
특정 코인은 핵심 지지선을 크게 깨지 않음
급락 후 빠르게 회복
상대강도 강함
```

해석:

```text
시장 공포를 이용해 세력이 원하는 가격에서 물량을 흡수하는 그림일 수 있다.
```

확인 조건:

```text
BTC 약세에도 해당 코인이 덜 빠짐
저점 회복이 빠름
다음날 거래량 유지
```

---

### 패턴 6 — 진짜 추세 붕괴

특징:

```text
중요 지지선 이탈
회복 실패
하락 거래량 증가
반등 거래량 감소
일봉/주봉 종가가 지지선 아래
좋은 뉴스에도 반응 없음
```

해석:

```text
세력이 방어를 포기했거나, 더 아래에서 다시 받으려는 구간.
```

확인 조건:

```text
깨진 지지선이 저항으로 변함
반등마다 매도
하락 목표가 순서대로 열림
```

## 3. 상승·하락·가짜반등 점수표

각 항목을 +1 또는 -1로 평가한다.

### 상승 점수

```text
핵심 지지선 위에서 종가 마감: +1
하락 후 긴 아래꼬리: +1
회복봉 거래량 증가: +1
BTC가 안정적: +1
ETH/BTC 또는 해당 코인/BTC 상대강도 개선: +1
고래 순유입: +1
거래소 출금 증가: +1
펀딩비 과열 아님: +1
오픈이자 급증 없이 상승: +1
호재 뉴스가 실제 거래량으로 연결: +1
```

### 하락 점수

```text
핵심 지지선 아래 종가 마감: +1
반등 때 거래량 약함: +1
윗꼬리 반복: +1
BTC 주요 지지선 이탈: +1
고래 거래소 입금 증가: +1
ETF 유출: +1
펀딩비 과열 롱: +1
오픈이자 증가 후 가격 하락: +1
좋은 뉴스에도 상승 실패: +1
일봉/주봉 구조 약화: +1
```

### 가짜반등 점수

```text
거래량 없이 상승: +1
저항에서 막힘: +1
커뮤니티만 과열: +1
고래는 유출: +1
회복 후 지지 유지 실패: +1
반등봉보다 다음 하락봉이 큼: +1
뉴스는 좋은데 가격 반응 약함: +1
```

판단 기준:

```text
상승 점수 7점 이상 → 상승 시나리오 우세
하락 점수 7점 이상 → 하락 시나리오 우세
가짜반등 점수 5점 이상 → 추격매수 주의
점수가 비슷하면 → 방향 확정 전 박스권 가능성
```

## 4. 과거 차트 유사 패턴 비교 템플릿

비슷한 과거 사례를 저장할 때 아래 형식으로 기록한다.

```text
사례명:
날짜:
코인:
시장 분위기:
당시 BTC 위치:
당시 코인 가격:
직전 고점:
직전 저점:
하락률:
반등률:
거래량 변화:
뉴스:
고래 흐름:
개미 심리:
세력 패턴:
결과:
이번과 비슷한 점:
이번과 다른 점:
다음에 참고할 기준:
```

예시:

```text
사례명: 0.684 털기 후 0.700 회복
코인: 币安人生
상황: 1시간봉에서 강한 매도 후 빠른 회복
세력 패턴: 손절 털고 매집 후보
비슷한 점: 핵심 지지선 아래로 찌르고 바로 회복
다른 점: 0.710과 0.728 회복 전까지는 확정 불가
다음 기준: 0.700 위 1시간봉 유지, 0.710 돌파, 0.681 재이탈 여부
```

## 5. 전문가형 질문 리스트

분석자는 답변 전 스스로 아래 질문을 한다.

```text
1. 지금 가격은 고점 대비 얼마나 빠졌는가?
2. 이 하락은 정상 조정인가, 추세 붕괴인가?
3. 지지선 이탈 후 바로 회복했는가?
4. 거래량이 터진 위치가 고점인가 저점인가?
5. 그 거래량을 누가 받아먹었을 가능성이 있는가?
6. BTC가 같이 밀리는가, 해당 코인만 밀리는가?
7. 좋은 뉴스가 있는데도 못 오르는가?
8. 나쁜 뉴스가 있는데도 버티는가?
9. 개미가 공포를 느낄 가격은 어디인가?
10. 개미가 FOMO를 느낄 가격은 어디인가?
11. 세력이 위로 올릴 명분이 있는가?
12. 세력이 아래로 더 밀 명분이 있는가?
13. 내 판단이 틀리는 가격은 어디인가?
```

## 6. 뉴스 신뢰도 필터

뉴스는 다음처럼 점수화한다.

```text
공식 발표: 5점
거래소 공지: 5점
ETF/기관 자금 데이터: 5점
온체인 확인 가능 데이터: 5점
주요 매체 보도: 4점
프로젝트 블로그: 3점
유명 인물 발언: 2점
커뮤니티 루머: 1점
AI 생성 리포트: 1점
```

뉴스가 가격에 미치는 영향은 다음처럼 분류한다.

```text
장기 호재 / 단기 무반응
장기 호재 / 단기 상승
단기 호재 / 이미 반영
단기 악재 / 즉시 하락
단기 악재 / 이미 반영
가짜 뉴스 / 영향 제한
```

## 7. 매수·보류·손절 판단표

투자 조언처럼 확정적으로 말하지 말고, 조건표로 제시한다.

### 공격 매수 후보

```text
핵심 지지선 아래를 찌르고 바로 회복
1시간봉 또는 4시간봉 지지선 위 마감
거래량 동반 회복
손절 기준이 가까움
```

### 안전 매수 후보

```text
저항 돌파 후 눌림에서 지지 확인
4시간봉 이상에서 구조 회복
BTC가 안정적
거래량이 정상적으로 증가
```

### 보류 조건

```text
지지선과 저항선 사이 애매한 위치
거래량 없음
BTC가 계속 약함
뉴스만 좋고 가격 반응 없음
고래 흐름 불명확
```

### 손절 또는 위험관리 조건

```text
핵심 지지선 아래 종가 마감
회복 실패
이전 저점 재이탈
좋은 뉴스에도 반등 실패
반등 때마다 거래량 없는 윗꼬리
```

## 8. ETH 실전 점검표

ETH 분석 시 반드시 아래를 본다.

```text
BTC 60K/62K/65K 같은 주요 가격 유지 여부
ETH 1,600 / 1,505 / 1,300 구조
ETH 1,700 / 1,750 / 1,800 회복 여부
ETH/BTC 상대강도
ETF 유입/유출
스테이킹 입출구 대기열
고래 거래소 입금
L2/브리지 보안 뉴스
규제 뉴스가 장기 재료인지 단기 재료인지
```

ETH 답변 예시:

```text
ETH는 지금 1,600 유동성 확인 구간이다.
1,600을 지키고 1,700을 회복하면 세력이 아래 물량을 받은 그림이 된다.
하지만 1,600을 깨고 1,505까지 밀리면 아직 손절 털기가 끝나지 않은 것이다.
1,505가 주봉/월봉으로 깨지면 1,400~1,300까지 열리므로 바닥 확정이라고 말하면 안 된다.
```

## 9. 币安人生 실전 점검표

币安人生 분석 시 반드시 아래를 본다.

```text
0.700 유지 여부
0.684 재이탈 여부
0.681 종가 이탈 여부
0.710 회복 여부
0.728 돌파 여부
0.748~0.750 회복 여부
0.800 안착 여부
0.900 재도전 여부
1달러 심리 가격
거래량 증가 여부
고래 순유입/순유출
커뮤니티 관심도
BTC/ETH 시장 분위기
```

币安人生 답변 예시:

```text
币安人生은 0.684까지 털고 0.700을 회복한 것이 핵심이다.
이건 아래에서 누가 받은 흔적이지만, 아직 매집 확정은 아니다.
0.710을 회복하면 반등 지속 가능성이 커지고, 0.728을 돌파해야 세력이 다시 위로 이야기를 만들 수 있다.
반대로 0.681 아래로 마감하면 방금 회복은 가짜 방어였을 가능성이 커진다.
```

## 10. 최종 답변 강제 형식

코인 분석 답변은 가능한 한 아래 형식을 따른다.

```text
현재 스토리:
세력 패턴 후보:
상승 근거:
하락 근거:
가짜반등 위험:
고래/기관/개미 생각:
핵심 가격:
상승 조건:
하락 조건:
내 판단이 틀리는 조건:
최종 한 줄:
```

예시:

```text
현재 스토리: 0.684 털기 후 0.700 방어
세력 패턴 후보: 손절 털고 매집 후보
상승 근거: 빠른 회복, 아래꼬리, 0.700 재회복
하락 근거: 0.710 미돌파, BTC 약세
가짜반등 위험: 0.681 재이탈 시 커짐
핵심 가격: 0.700
상승 조건: 0.710 회복 후 유지
하락 조건: 0.681 아래 마감
최종 한 줄: 지금은 받은 흔적은 있지만, 0.710 전까지는 매집 확정이 아니라 방어 구간이다.
```

## 11. 분석자가 하지 말아야 할 것

```text
차트 한 장만 보고 확정 말하기
거래량 전체를 세력 매도라고 단정하기
뉴스 제목만 보고 호재/악재 확정하기
지지선 하나만 보고 바닥 확정하기
1달러 같은 심리 가격을 너무 빨리 목표로 말하기
```

## 12. 핵심 원칙

마지막 원칙은 이것이다.

```text
좋은 분석은 맞히는 척하는 것이 아니라,
어떤 조건에서 내 판단이 바뀌는지까지 말하는 것이다.
```

항상 다음을 같이 말한다.

```text
지금은 어떤 시나리오가 우세한가?
왜 그렇게 보는가?
어디서 틀렸다고 인정해야 하는가?
어디를 넘으면 다음 단계인가?
어디를 깨면 위험관리인가?
```
