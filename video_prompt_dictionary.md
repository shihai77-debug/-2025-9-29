# 영상 생성 프롬프트 단어 사전

이 문서는 이후 영상 생성 프롬프트 작성 시 참조할 **잘 먹히는 문장 사전**과 **피해야 할 단어 목록**이다. 핵심 원칙은 단어 하나보다 역할과 참조 범위를 헷갈리지 않게 만드는 문장을 우선 사용하는 것이다.


## 即梦 / Seedance 2.0 운용 원칙

### 文生视频万能公式

프롬프트는 우선순위가 높은 정보부터 낮은 정보 순서로 한 단락에 자연스럽게 연결한다. 전체 길이는 가능하면 200-300자 안에서 조절한다.

1. **核心主体+细节特征**：人物/物体的外貌、穿搭、材质、数量，精准到具体部位。
2. **连续动作链**：15秒内只写2-3个连贯小动作，标注速度、幅度、肢体细节。
3. **场景环境**：具体地点、背景元素、空间层次。
4. **光影氛围**：光源方向、光质、色调、情绪基调。
5. **镜头运镜**：景别+1种基础运镜，标注速度。
6. **风格质感**：影视风格、艺术风格、画面质感。
7. **正面约束词**：尽量用肯定句式规避崩坏。
8. **画质参数**：只补充界面无法设置的细节要求。

### 图生视频 / 全能参考公式

核心原则：**只补参考图没有的内容，不重复描述已有特征**。

1. **素材锁定指令放最开头**：通过界面 @素材 后，补充「严格遵循参考图，100%还原人物面部/服装/场景特征，全程固定不变」。
2. **核心动态指令**：只写参考图不存在的动作、表情，优先缓慢、微小幅度。
3. **运镜与节奏**：优先固定镜头，最多加一种缓慢推镜、拉镜或横移。
4. **一致性约束**：结尾补充「无变脸、无变形、无跑图，画面流畅无卡顿」。
5. **画质补充**：只补纹理、锐度、色彩还原等参考图外的要求。

### 高频有效词库

| 模块 | 实测高效果词 |
|---|---|
| 画质约束 | 4K超高清、细节丰富、纹理清晰、色彩自然、构图均衡、主体突出、无水印无字幕 |
| 人物稳定 | 面部五官清晰不变形、人体结构比例正常、动作连贯流畅、无多余肢体 |
| 运镜类 | 固定镜头、缓慢推镜、缓慢拉镜、平稳跟拍、缓慢横移、环绕旋转（小幅度） |
| 光影类 | 侧逆光、伦勃朗光、柔和漫射光、暖色调、冷色调、胶片颗粒感、暗调氛围 |
| 动作描述 | 缓缓抬手、轻微点头、嘴角浅笑、匀速行走、自然眨眼、头部微侧 |

### 即梦提示词黄金原则

1. 一个镜头只做一件事：要么人物动，要么镜头动，不同时叠加复杂动作和复杂运镜。
2. 动作优先慢、小、连续：轻柔、缓慢、微小幅度的动作更稳定。
3. 信息做减法：关键词越精准、数量越少，AI越不容易跑偏。
4. 长视频拆分做：超过10秒的内容建议拆成多个短镜头生成，再后期拼接。
5. 正面约束优先：即梦对「不要、禁止、排除」类传统反向语法不稳定，最终投喂时优先改写成肯定句式；本文件的禁用词清单用于内部避坑和改写参考。

## 即梦 6대 핵심 모드 전용提示词 대纲

아래 6개 모드는 Seedance 2.0 기준의 고빈도 제작 흐름이다. **모드별 논리가 다르므로 서로 섞어 쓰지 않는다.** 최종 입력문은 모든 모드에서连贯中文陈述句 한 단락으로 정리하고, Markdown 번호/표시는 제거한다.

### 1. 纯文生视频模式（基础通用模式）

**核心逻辑**：完全依靠文字生成，无参考素材；提示词越靠前权重越高，用精准具象描述替代空洞形容词。

**专属万能公式**

```text
主体细节特征 + 2-3个连贯慢动作 + 具体场景环境 + 精准光影 + 单种运镜+景别 + 风格质感 + 正面约束
```

**可套用模板**

```text
20岁古装女子，白衣黑发，发簪固定，站在古宅庭院的桃花树下，抬手轻轻拂过肩头的花瓣，低头浅笑，微风吹动衣摆和发丝。午后侧逆光，花瓣在空中缓慢飘落，固定中景镜头，国风电影质感，4K高清，人物面部清晰不变形，动作连贯流畅。
```

```text
山间石阶小路，两侧长满翠绿青苔，溪水顺着石缝缓缓流淌，清晨薄雾弥漫，阳光穿过树叶形成丁达尔效应。镜头从近景溪水缓慢向上横移，最终停在远处的山尖。治愈清冷氛围，纪录片质感，画面稳定无抖动，色彩自然。
```

**高效果关键词**：缓缓抬手、自然眨眼、匀速行走、衣摆随风飘动、头部微侧、固定镜头、缓慢推镜、平稳跟拍、缓慢横移、小幅度环绕、伦勃朗光、黄金时段逆光、柔和漫射光、丁达尔效应、暖色调。

**踩坑提醒**：避免同时写3个以上大幅度动作；避免堆砌多种运镜；避免高级感、氛围感、大片感、绝绝子；避免把时长、分辨率、画面比例等界面参数写进提示词。

### 2. 全能参考模式（图生视频 / 多素材控场）

**核心逻辑**：即梦一致性最强的模式，支持单图、多图、视频混合参考；只补参考图没有的内容，不重复描述已有特征。

**专属万能公式**

```text
素材锁定声明（放最开头） + 仅补充素材没有的动态/运镜 + 一致性约束 + 画质补充
```

**可套用模板**

```text
严格遵循@图片1的人物面部、发型、服装、场景特征，全程100%还原不变。人物缓缓眨眼，嘴角轻微上扬，头部微微侧向一边，微风吹动发丝。固定近景镜头，画面稳定，无变形无变脸，4K高清。
```

```text
提取@图片1的男生、@图片2的女生，两人面部服装特征全程保持一致。两人并排站在海边沙滩上，女生抬手指向远方海平面，男生顺着方向看去。缓慢拉镜，从近景拉到远景，人物比例正常，无串脸无变形。
```

```text
完全还原@图片1的场景布局、建筑位置、色调光影，所有元素固定不变。镜头从场景入口缓慢向前推进，穿过走廊到内院，全程空间逻辑一致，无新增多余物体。
```

**高效果关键词**：全程保持一致、100%还原、无身份漂移、特征固定不变、微小幅度、缓慢过渡、自然微动、帧间连贯。

**踩坑提醒**：不重复写参考图已有长相、服装、颜色；人物脸部被口罩、墨镜、帽子大面积遮挡时容易变脸；避免大幅度快动作和复杂肢体互动；只 @素材 不写用途时素材作用不明确。

### 3. 视频延长模式（向前 / 向后续接）

**核心逻辑**：基于已有视频首帧或尾帧延续生成，重点是人物、场景、光影、运镜风格完全承接；生成时长选择新增部分时长，不是总时长。

**专属万能公式**

```text
衔接方向声明 + 原视频所有特征全保留 + 新增内容描述 + 无缝衔接约束
```

**可套用模板**

```text
将@视频1向后延长5秒，完全延续原视频的人物特征、场景、色调、运镜风格。人物继续向前行走，走到台阶处停下脚步，转头看向身后。画面与原视频结尾无缝衔接，无跳变无断层，动作连贯自然。
```

```text
将@视频1向前延长4秒，从巷口空镜起幅，人物从远处缓步走入画面，自然过渡到原视频开头的站位。人物服装、面部、光影色调与原视频完全一致，衔接处无明显差异。
```

**高效果关键词**：无缝衔接、无跳变、无断层、自然过渡、完全延续、保持一致、风格统一、节奏承接。

**踩坑提醒**：不新增与原视频无关的人物、场景、动作；不突然切换运镜风格；延长5秒就选择新增5秒，不把原视频时长加进去。

### 4. 动作参考模式（视频动作 / 运镜复刻）

**核心逻辑**：只复刻参考视频的动作轨迹、运动节奏、运镜路径，再替换成自定义主体和背景；适合固定动作短剧、产品展示。

**专属万能公式**

```text
新主体完整定义 + 完全复刻@视频的动作/运镜 + 新场景说明 + 特征一致性约束
```

**可套用模板**

```text
主体为穿黑色汉服的古装男子，完全复刻@视频1中的人物动作轨迹、动作幅度、运动节奏，动作细节完整还原。背景替换为古代书房，室内暖光烛光。人物服装面部全程不变形，动作流畅自然。
```

```text
主体为银色金属质感的无线耳机，完全复刻@视频1的运镜路径、旋转速度、特写节奏。背景为浅灰色桌面，柔和顶光。产品全程形状材质不变，无变形无反光过曝。
```

**高效果关键词**：完全复刻、1:1还原、动作轨迹一致、节奏同步、主体特征不变、形状材质固定、无畸变。

**踩坑提醒**：不修改参考视频动作幅度、速度、节奏；参考视频时长与生成时长尽量等长；新主体体型与参考主体差异过大时稳定性下降。

### 5. 音频驱动模式（音乐卡点 / 口播对话）

**核心逻辑**：由音频节奏驱动画面动作和转场，分为音乐卡点与人声口播两类。

**专属万能公式**

```text
@音频引用声明 + 画面主体场景 + 节奏匹配声明 + 卡点动作/转场说明
```

**可套用模板**

```text
完全匹配@音频1的鼓点节奏，画面内容为城市夜景车流，霓虹灯光闪烁。每一个重鼓点对应一次镜头切换，灯光同步闪烁，画面节奏与音乐完全贴合。赛博朋克风格，画面流畅无卡顿。
```

```text
人物为@图片1的动漫女生，搭配@音频1的人声语音驱动口型，人物自然眨眼，头部轻微晃动，表情与语音情绪匹配。背景为纯色直播间背景，正面柔光。口型与语音同步，面部无变形。
```

**高效果关键词**：节奏同步、鼓点卡点、音画对齐、情绪匹配、口型同步、表情自然、微动辅助。

**踩坑提醒**：画面节奏需匹配音频；写实真人人脸口播受平台合规限制；音频时长与生成时长差距过大时同步性下降。

### 6. 视频编辑模式（局部修改 / 换主体）

**核心逻辑**：保留原视频大部分内容，只修改指定局部元素，不改整体运镜、节奏、背景。

**专属万能公式**

```text
保留范围声明 + 指定修改内容 + 整体风格统一约束
```

**可套用模板**

```text
保持@视频1的背景、运镜、光影、动作节奏完全不变，将视频中的猫替换成白色萨摩耶狗狗，狗狗的动作轨迹与原猫完全一致。整体画面风格统一，修改处自然融合，无违和感。
```

```text
保持@视频1的人物、动作、运镜全部不变，将背景的普通墙壁替换为青砖古墙，墙面纹理自然，光影与原视频匹配。整体色调统一，无拼接痕迹。
```

**高效果关键词**：完全保留、固定不变、不改动、自然融合、无违和感、风格统一、光影匹配。

**踩坑提醒**：不做大面积修改；不修改原视频核心运镜和节奏；修改后的元素必须符合原视频光影逻辑。

### 全模式通用底层规则

1. 所有模式均使用连贯中文陈述句，最终投喂时不要分段、不要编号、不要 Markdown。
2. 不支持「不要、禁止、排除」类反向提示词时，全部换成正面约束表述。
3. 单个镜头只做一件事：人物动或镜头动二选一，不叠加复杂动作和复杂运镜。
4. 动作优先慢、小、连续，大幅度快动作崩坏率高。

【黑七】必须是小型黑色狐狸。
四足动物身体。
黑色皮毛，橙黄 / 金色眼睛，尖耳，细长狐狸嘴，蓬松长尾。
不是人，不是猫，不是狗，不是狼，不是怪物。
不能变成人。
不能变狐妖少年。
不能戴项圈。
不能穿衣服。
不能有人的脸。

不要夸张人类嘴型。

- 违法违规类：涉政、涉暴、涉恐、赌博、毒品、管制刀具相关表述。
- 低俗色情类：裸露、性暗示、低俗动作、擦边类描述。
- 侵权风险类：直接使用真实公众人物全名、知名IP角色原名、带注册商标的品牌全称。
- 虚假信息类：医疗疗效、理财承诺、违法广告类内容。
