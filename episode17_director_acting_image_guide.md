# CUT-01｜黑七｜12秒

【参考素材】
角色参考：陆沉舟三面图；黑七黑狐三面图。
场景参考：空巷、屋顶、低光巷子。
导演参考：只参考陆沉舟停步、低声叫名、黑七从屋顶落地的动作节奏。
上一CUT：承接第十六集“有人来过。留了遮气阵”的追踪线索。
下一CUT：黑七落地后，在陆沉舟面前坐下并接话。

【角色锁定】
陆沉舟保持黑色短款皮夹克、黑T、深色长裤、黑军靴；黑七必须是黑狐本体，四足，橙黄色狐瞳，蓬松狐尾。不得出现人形黑七。

【场景】
无人巷子，屋顶可见，风声极细，光线压低。

【画面主体】
陆沉舟在巷子里停下，低声叫“黑七”。屋顶上黑七轻轻跳下，落地无声，若无其事。

【导演参考图使用规则】
只参考动作顺序、镜头从全景到近景、黑七落地节奏；不参考临时人物脸、临时服装、箭头、编号、黑白画风。

【摄影机】
全景转近景。前3秒固定全景，3秒后轻微压近陆沉舟，6秒后跟随黑七落地。

【时间分割】
【0-3秒】
画面：巷子里，陆沉舟停下来，四周无人。
人物位置：陆沉舟在巷子中央偏前景。
动作：他停步，不回头。
眼神：视线微微上抬，像已知道屋顶有人。
表情：冷静，目的明确。
镜头：全景，低光。
声音：微风声细。

【3-6秒】
画面：陆沉舟低声开口。
人物位置：陆沉舟仍在原位。
动作：嘴唇只动一下。
眼神：看向屋顶方向。
表情：不惊讶。
镜头：从全景压到近景。
声音：陆沉舟低声：“黑七。”

【6-12秒】
画面：屋顶上黑七轻轻跳下来，落地无声，若无其事。
人物位置：黑七从屋顶落到陆沉舟前方几步。
动作：四足落地，尾巴轻摆。
眼神：橙黄色狐瞳看向陆沉舟。
表情：漫不经心。
镜头：近景捕捉落地。
声音：狐狸极轻落地声后完全静默。

【对白 / 台词】
陆沉舟（低，简短）：黑七。

【表演要求】
陆沉舟不解释、不喊；黑七落地要轻，像本来就跟着他们，不要卖萌。

【声音设计】
BGM为古风低沉弦乐单音线，音量极低；微风声→狐狸极轻落地声→完全静默。

【连接逻辑】
承接第十六集遮气阵线索；结束在黑七落地坐定前，下一CUT直接接黑七坐下开口。

【关键限制】
禁止黑七人形化；禁止新增路人；禁止换陆沉舟衣服；禁止夸张术法特效。

【建议时长】
12秒。

# CUT-02｜鼻子最好使｜14秒

【参考素材】
角色参考：陆沉舟三面图；黑七黑狐三面图。
场景参考：同一巷子。
导演参考：一人一狐对视、黑七嘴上反驳身体站起的节奏。
上一CUT：黑七从屋顶落地。
下一CUT：接“鼻子最好使”后黑七继续嘴硬，但身体已开始行动。

【角色锁定】
陆沉舟现代黑衣不变；黑七黑狐本体不变。不得新增人物。

【场景】
同一无人巷子，低光，环境底噪极轻。

【画面主体】
黑七坐在陆沉舟面前，短促交谈。陆沉舟交代任务，黑七反驳，但身体已经站起来。

【导演参考图使用规则】
只参考对视、起身、对话节奏；不参考临时脸和临时服装。

【摄影机】
近景，一人一狐对视。镜头不大动，靠台词节奏推进。

【时间分割】
【0-3秒】
画面：黑七在陆沉舟面前坐下，漫不经心。
人物位置：陆沉舟前景侧面，黑七在地面中景。
动作：黑七坐下，尾巴随意扫一下。
眼神：黑七抬眼看陆沉舟。
表情：不耐烦但不紧张。
镜头：近景。
声音：巷子环境底噪。

【3-8秒】
画面：陆沉舟交代遮气阵与查气息。
人物位置：陆沉舟站着，黑七坐着。
动作：陆沉舟不动，只说话。
眼神：陆沉舟看黑七。
表情：平静陈述。
镜头：一人一狐对视。
声音：低沉弦乐极低。

【8-14秒】
画面：黑七反驳，身体却已经站起来。
人物位置：黑七从坐姿变站姿。
动作：黑七嘴上反驳，四足站起。
眼神：斜看陆沉舟。
表情：嘴硬。
镜头：近景保持。
声音：环境底噪保留。

【对白 / 台词】
黑七（漫不经心）：干嘛。
陆沉舟（平，陈述）：衣服店的墙上有遮气阵，不是普通的。顺着气息查过去。
黑七（反驳）：凭什么是我。
陆沉舟：你鼻子最好使。

【表演要求】
陆沉舟两句话交代完，不拖；黑七反驳要真实，但身体已经站起来，为下一CUT嘴硬心软做铺垫。

【声音设计】
古风低沉弦乐单音线维持，极低音量；仅保留巷子环境底噪。

【连接逻辑】
接CUT-01黑七落地；结束在“你鼻子最好使”，下一CUT黑七接话“少给我戴高帽”。

【关键限制】
禁止黑七变成人；禁止让陆沉舟解释过多；禁止新增角色。

【建议时长】
14秒。

# CUT-03｜嘴硬心软｜12秒

【参考素材】
角色参考：黑七黑狐三面图；陆沉舟三面图。
场景参考：同一巷子通往入口方向。
导演参考：黑七边走边问、嘴硬心软。
上一CUT：陆沉舟说“你鼻子最好使”。
下一CUT：黑七到达巷子入口，看到鸽子。

【角色锁定】
黑七仍为黑狐本体；陆沉舟黑衣不变。

【场景】
巷子延伸到入口方向，环境低光。

【画面主体】
黑七嫌弃陆沉舟，但身体已经行动，边走边问后续。

【导演参考图使用规则】
只参考黑七起身、走远、回头问话的动作节奏。

【摄影机】
近景跟拍黑七，陆沉舟可在背景或画外。

【时间分割】
【0-4秒】
画面：黑七看了陆沉舟一眼。
人物位置：黑七前景，陆沉舟后方。
动作：黑七转头，四足开始迈步。
眼神：嫌弃地瞥一眼。
表情：狐嘴微紧，耳朵轻压。
镜头：近景。
声音：极轻脚步声。

【4-8秒】
画面：黑七边走边问。
人物位置：黑七朝巷子入口走。
动作：身体已行动，嘴上仍嫌弃。
眼神：不看陆沉舟。
表情：嘴硬。
镜头：轻微跟拍。
声音：低弦维持。

【8-12秒】
画面：黑七走远嘟囔。
人物位置：黑七渐远，陆沉舟可保持原位。
动作：黑七尾巴扫一下。
眼神：看向巷子入口。
表情：不情愿。
镜头：近景拉出一点。
声音：脚步极轻。

【对白 / 台词】
黑七（嫌弃，但已经站起来）：少给我戴高帽。
黑七（边走边问）：查到了然后呢。
陆沉舟：告诉我。
黑七：就这？
陆沉舟：查仔细点。
黑七（走远，嘟囔）：……烦死了。

【表演要求】
黑七嘴上嫌弃，身体先动。重点是嘴硬心软，不要演成纯搞笑。

【声音设计】
古风弦乐维持极低；脚步声极轻。

【连接逻辑】
接CUT-02“鼻子最好使”；结束在黑七走向巷子入口，下一CUT切巷子入口鸽子。

【关键限制】
禁止黑七人形化；禁止新增旁观路人。

【建议时长】
12秒。

# CUT-04｜鸽子眼睛｜12秒

【参考素材】
角色参考：黑七黑狐三面图；鸽子动物参考可为普通城市鸽。
场景参考：巷子入口。
导演参考：术法静音、鸽子眼睛涣散。
上一CUT：黑七走向巷子入口。
下一CUT：鸽子眼睛涣散后，黑七低声问话。

【角色锁定】
黑七必须保持黑狐本体；鸽子只是动物，不得拟人化成人。

【场景】
巷子入口，地面散落一群鸽子，环境底噪少。

【画面主体】
黑七走进画面，施术。鸽子定住，眼睛涣散。

【导演参考图使用规则】
只参考术法节奏、眼神变化、尾巴动作；不参考夸张魔法特效。

【摄影机】
全景转极近鸽子眼睛。

【时间分割】
【0-4秒】
画面：巷子入口，一群鸽子散在地上，黑七走进画面。
人物位置：黑七从画面边缘进入。
动作：四足慢步，尾巴放低。
眼神：黑七扫视鸽子群。
表情：专注。
镜头：全景。
声音：鸽子咕咕声，BGM静音。

【4-8秒】
画面：黑七眼神变了，尾巴缓缓扫一下。
人物位置：黑七停在鸽子前。
动作：尾巴缓慢扫动。
眼神：狐瞳收紧。
表情：施术时冷静。
镜头：靠近黑七眼睛再切鸽子。
声音：施术瞬间极细高频音。

【8-12秒】
画面：一只鸽子定住，眼睛涣散。
人物位置：鸽子成为主体。
动作：鸽子身体停住。
眼神：鸽子眼睛涣散。
表情：动物无表情。
镜头：极近鸽子眼睛。
声音：完全静默。

【对白 / 台词】
无台词。

【表演要求】
黑七能力要用无声和眼神表现，不要夸张光效。

【声音设计】
BGM完全抽空；鸽子咕咕声→极细高频音→完全静默。

【连接逻辑】
接CUT-03黑七到达入口；结束在鸽子眼睛涣散，下一CUT黑七开口问话。

【关键限制】
禁止把鸽子拟人化；禁止魔法大光效；禁止黑七人形化。

【建议时长】
12秒。

# CUT-05｜关你屁事｜14秒

【参考素材】
角色参考：黑七黑狐三面图；鸽子。
场景参考：巷子入口。
导演参考：黑七郑重问话、鸽子嚣张顶嘴。
上一CUT：鸽子眼睛涣散。
下一CUT：黑七威胁后鸽子招供。

【角色锁定】
黑七为黑狐本体；鸽子只是被术法控制的动物。

【场景】
巷子入口静默状态。

【画面主体】
黑七低声问鸽子，鸽子嚣张顶嘴，黑七眯眼威胁。

【导演参考图使用规则】
只参考对话节奏、眼神和黑七眯眼，不参考喜剧夸张表情。

【摄影机】
极近鸽子眼睛转黑七近景。

【时间分割】
【0-4秒】
画面：黑七对着鸽子低声清晰开口。
人物位置：黑七近景，鸽子在前景。
动作：黑七头微低。
眼神：狐瞳盯鸽子。
表情：郑重。
镜头：极近转近景。
声音：静默，只留黑七声音。

【4-8秒】
画面：鸽子回答，态度嚣张。
人物位置：鸽子主体。
动作：鸽子身体仍定住。
眼神：涣散。
表情：动物无表情，声音嚣张。
镜头：鸽子极近。
声音：鸽子声音带极轻回声，仅黑七术法中可听见。

【8-14秒】
画面：黑七眯眼，再开口。
人物位置：黑七重新成为主体。
动作：黑七眼睛眯起来，耳朵微压。
眼神：危险。
表情：认真。
镜头：近景。
声音：其余静默。

【对白 / 台词】
黑七（低，清晰）：三天内，这里来过什么陌生气息。说。
鸽子（嚣张，仅黑七可听见）：关你屁事。
黑七（眯眼）：再说一遍试试。

【表演要求】
黑七越认真越好笑，但不要演成搞笑；鸽子敢顶嘴是笑点。

【声音设计】
BGM静音；仅黑七声音；鸽子说话带极轻回声。

【连接逻辑】
接CUT-04鸽子眼睛涣散；结束在黑七威胁，下一CUT鸽子招供。

【关键限制】
禁止鸽子变成人；禁止黑七夸张表情包化。

【建议时长】
14秒。

# CUT-06｜面包屑｜12秒

【参考素材】
角色参考：黑七黑狐三面图；陆沉舟三面图；鸽子。
场景参考：巷子入口。
导演参考：鸽子快速招供、术法解除、黑七转身汇报。
上一CUT：黑七威胁鸽子。
