# 《棺中檀色》第十六集｜出棺第一天 — 影视制作导演分镜图像提示表

## 使用目的

本文件根据第十六集分镜资料整理为 **图像生成用导演提示表**。全篇统一使用中文，保留原始台词、时间轴、画面、BGM、音效与“为什么”说明。

- 总CUT：12个。
- 总时长：约2分22秒。
- 所有CUT均不超过15秒。
- 基调：前半段是“日常 vs 千年”的轻喜剧，CUT-10开始在喜剧中突然切入悬疑冷意。
- 生成方式：每个CUT拆成“开始图 / 过程图 / 结束图”，一张一张连续生成。

```text
严格以随附角色三视图为最高优先级。人物脸型、五官、发型、服装结构、鞋子、身体比例必须与三视图一致。本镜头只允许改变姿势、表情、湿润程度、光影和镜头角度，不允许重新设计人物。
```

## 实际生成顺序

1. 先放本CUT出现人物的三视图。
2. 再放上一张生成图作为连续性参考。
3. 复制对应CUT的生成提示词。
4. 每次只生成一张图。
5. 检查：檀缺白衣是否与三视图一致、白清檀是否仍是黑衣、韩知玄制服是否正确、陆沉舟是否现代黑衣、黑七是否黑狐本体。
6. 前9个CUT保持轻喜剧日常；CUT-10开始降低喜剧感，切入悬疑。
