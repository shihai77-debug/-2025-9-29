# 《棺中檀色》폐기설정 충돌 방지 문서

> 목적: 본 문서는 《棺中檀色》 작업 중 폐기되었거나 오래되어 더 이상 사용하면 안 되는 설정을 따로 보관해, 최신 설정과 충돌하거나 실수로 재사용되는 일을 막기 위한 문서다.

---

## 기본 원칙

- 본 문서의 내용은 **활성 스토리 설정이 아니다**.
- 본 문서에 적힌 항목은 소설 본문, 숏드라마 대본, 콘티, 이미지 프롬프트에 다시 반영하지 않는다.
- 캐릭터 설정이 충돌할 경우 `02_캐릭터_최종설정.md`를 최우선 기준으로 삼는다.
- 본 문서는 폐기/충돌 방지용 목록이며, 새로운 장면이나 사건을 만들기 위한 자료가 아니다.

---

## 1. 삭제된 캐릭터 설정

### 陆沉舟 관련 삭제 설정

- 陆沉舟를 장포 차림의 고장풍 인물로 쓰지 않는다.
- 陆沉舟를 암청색 긴 코트 이미지로 쓰지 않는다.
- 陆沉舟를 갑옷 입은 무장으로 쓰지 않는다.
- 陆沉舟를 수선자, 선협풍 인물, 고대 도사 이미지로 쓰지 않는다.
- 陆沉舟를 玄衡门 본산/분문파의 정식 제자 출신으로 쓰지 않는다.
- 최신 기준은 아버지에게 배운 돈이 필요한 父传 실전형 风水师다.

### 白清檀 관련 삭제 설정

- 白清檀을 백의 여인으로 쓰지 않는다.
- 白清檀을 흰색 넓은 소매 고장복 인물로 쓰지 않는다.
- 白清檀을 여귀, 반투명 혼령, 귀신 이미지로 쓰지 않는다.
- 白清檀을 요괴화된 존재로 쓰지 않는다.
- 최신 기준은 昆仑山과 연결된 上古族群 마지막 여왕이며, 검은 긴 치마/검은 긴 드레스 계열 외형이다.

### 黑七 관련 삭제 설정

- 黑七을 인간형으로 쓰지 않는다.
- 黑七을 여우요괴 청년으로 쓰지 않는다.
- 黑七을 늑대인간형으로 쓰지 않는다.
- 黑七을 직립한 성인 캐릭터로 쓰지 않는다.
- 최신 기준은 작은 검은 여우 본체이며, 봉인은 이미 풀렸고 갈 데가 없어 陆沉舟 곁에 남는다.

### 韩知玄 관련 삭제 설정

- 韩知玄을 경갑 차림으로 쓰지 않는다.
- 韩知玄을 수선복 차림으로 쓰지 않는다.
- 韩知玄을 고대 갑옷 차림으로 쓰지 않는다.
- 韩知玄의 `镇灵司` 표식을 삭제하거나 흐리게 처리하지 않는다.
- 최신 기준은 정부 계열 특수조사 조직 `镇灵司` 소속 현장 조사원이며, 의상에 `镇灵司` 표식/배지/문장이 있다.

### 檀缺 관련 삭제 설정

- 檀缺을 이름만 있는 미확정 인물로 방치하지 않는다.
- 檀缺을 단순 설명캐로 쓰지 않는다.
- 檀缺과 陆沉舟의 연애를 현재 확정하지 않는다.
- 현대사회를 모른다는 이유로 바보처럼 쓰지 않는다.

### 沈泊川 관련 삭제 설정

- 沈泊川을 숨은 고수로 만들지 않는다.
- 沈泊川을 문파 장로, 镇灵司 요원, 无相会 간부, 昆仑山 혈통으로 만들지 않는다.
- 沈泊川에게 陆沉舟보다 강한 술법 능력을 주지 않는다.
- 沈泊川을 사건을 직접 해결하는 주역으로 만들지 않는다.
- 최신 기준은 陆沉舟의 외가 쪽 사촌형이자 泊川整理事务所를 운영하는 현실형 도시 사건 연결자다.

### 昆仑山 충신 장군 관련 삭제 설정

- 昆仑山 충신 장군을 악역으로 확정해 쓰지 않는다.
- 정치적으로 교활한 인물로 만들지 않는다.
- 白清檀 아버지의 충신이라는 최신 기준과 충돌하는 설정은 쓰지 않는다.

### 白清檀 아버지 관련 삭제 설정

- 白清檀 아버지를 완벽한 왕으로 만들지 않는다.
- 昆仑山 上古族群 전대 왕이라는 최신 기준과 충돌하는 신분을 쓰지 않는다.
- 내부 배신을 막지 못한 책임을 지우지 않는다.

### 白清檀 어머니 관련 삭제 설정

- 白清檀 어머니를 단순히 죽은 어머니로만 쓰지 않는다.
- 檀氏 혈통, 祭司 계열, 檀骨/생존 비밀과의 연결을 지우지 않는다.
- 檀缺을 남긴 보호자 역할과 충돌하는 설정은 쓰지 않는다.

## 2. 충돌나는 옛 설정

### 캐릭터 외형 충돌

- 최신 캐릭터 바이블의 현대/검은 의상 기준과 충돌하는 고장풍 의상 설정은 사용하지 않는다.
- 白清檀을 흰색 계열 영적 존재처럼 보이게 하는 설정은 최신 설정과 충돌한다.
- 黑七을 사람으로 바꾸는 설정은 최신 설정과 충돌한다.
- 韩知玄에게서 `镇灵司` 소속 표식을 제거하는 설정은 최신 설정과 충돌한다.

### 캐릭터 기능 충돌

- 陆沉舟를 말없는 李阳식 침묵 남주로 만드는 설정은 최신 방향과 충돌한다.
- 陆沉舟를 돈 이야기를 피하는 성인군자로 만드는 설정은 최신 방향과 충돌한다.
- 白清檀을 약하기만 하거나 계속 보호만 받는 여주로 쓰는 설정은 최신 방향과 충돌한다.
- 黑七이 봉인 후유증 때문에 陆沉舟 곁에 있어야 한다는 설정은 최신 방향과 충돌한다.
- 韩知玄을 真灵寺 소속으로 쓰는 설정은 최신 방향과 충돌한다.
- 팀 역할 분담을 현재 확정하는 설정은 최신 방향과 충돌한다.

### 관계 설정 충돌

- 白清檀이 昆仑山 및 上古族群 마지막 여왕 설정과 무관하다는 설정은 사용하지 않는다.
- 黑七의 봉인이 아직 풀리지 않았거나 봉인 자국이 계속 남아 있다는 설정은 사용하지 않는다.
- 檀缺이 白清檀 어머니가 남긴 보호자라는 설정과 충돌하는 관계 설정은 사용하지 않는다.
- 韩知玄이 부모 빽으로 들어온 낙하산이라는 설정은 사용하지 않는다.

## 3. 다시 쓰면 안 되는 설정

- 真灵寺 설정.
- 陆沉舟를 말없는 李阳식 캐릭터로 만드는 방향.
- 陆沉舟가 돈을 필요로 하지 않거나, 반대로 돈 때문에 사람을 해치는 방향.
- 陆沉舟를 玄衡门 정식 제자/문파형 풍수사로 설정하는 방향.
- 陆沉舟가 借运, 夺运, 断人祖坟, 养鬼, 拿活人做局, 害人风水를 하는 방향.
- 白清檀을 약하기만 한 여주 또는 계속 보호만 받는 여주로 만드는 방향.
- 黑七이 봉인 후유증 때문에 陆沉舟 곁에 있어야 한다는 설정.
- 黑七 봉인 자국이 계속 남아 있다는 설정.
- 黑七 인간화, 늑대형, 괴물형.
- 韩知玄을 부모 빽으로 들어온 낙하산으로 만드는 설정.
- 팀 역할 분담을 현재 확정하는 내용.
- 檀缺과 陆沉舟의 연애 확정.
- 沈泊川을 숨은 고수, 문파 장로, 镇灵司 요원, 无相会 간부, 昆仑山 혈통으로 만드는 설정.
- 손님/의뢰인/회차별 사건 인물을 캐릭터 고정 문서에서 미리 확정하는 방식.

## 4. 회차별 인물 보류 / 금지 설정

- 손님, 의뢰인, 회차별 사건 인물은 캐릭터 고정 문서에 미리 만들지 않는다.
- 손님 이름, 손님 목소리, 집 구조, 사건 원인, 단서, 결말은 해당 회차 스토리 작업에서 새로 설계한다.
- 고정 문서에는 陆沉舟, 白清檀, 黑七, 韩知玄, 檀缺, 沈泊川 같은 주요/반복 등장 캐릭터 기준만 둔다.
- 회차별 손님과 사건 세부를 폐기설정이나 이전 보조안에서 끌어오지 않는다.

## 5. 최신 설정 우선순위

설정 충돌이 발생할 경우 아래 순서대로 판단한다.

1. **최신 확정 설정**
   - `02_캐릭터_최종설정.md`에 정리된 현재 캐릭터 바이블을 최우선으로 따른다.
2. **사용자가 직접 수정한 설정**
   - 사용자가 이후 직접 수정하거나 명시한 설정은 이전 보조안보다 우선한다.
3. **이전 보조안**
   - 과거 작업 중 제안되었으나 최신 확정 문서에 반영되지 않은 내용은 보조 참고로만 본다.
4. **폐기설정**
   - 본 문서에 들어온 폐기설정은 재사용하지 않는다.

정리하면 다음과 같다.

> **최신 확정 설정 > 사용자가 직접 수정한 설정 > 이전 보조안 > 폐기설정**

## 6. 작업 시 주의사항

- 새 장면을 쓸 때는 먼저 `02_캐릭터_최종설정.md`를 확인한다.
- 본 문서의 폐기 항목은 “다시 쓰지 말아야 할 목록”으로만 사용한다.
- 폐기설정이 매력적으로 보여도 활성 스토리에 되살리지 않는다.
- 외형 프롬프트 작성 시 폐기된 의상, 종족화, 인간화, 귀신화 표현을 넣지 않는다.
- `02_캐릭터_최종설정.md`에 새로 확정된 가족사, 세력, 신분, 능력은 최신 기준으로 따른다.
- 충돌 여부가 애매하면 최신 캐릭터 바이블을 우선하고, 불명확한 내용은 “미확정”으로 남긴다.
- 기존 대본, 스토리보드, 에피소드 파일을 수정하지 않는다.
