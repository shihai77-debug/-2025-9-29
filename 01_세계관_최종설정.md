# 《棺中檀色》세계관 설정 고정본

## 스토리 비간섭 버전

## 사용 목적

이 문서는 《棺中檀色》의 세계관 기본 구조를 고정하기 위한 자료다.

이 문서는 스토리 전개를 정하지 않는다.

```text id="px7hyd"
회차 사건 설정 금지
손님 / 의뢰인 설정 금지
사건 원인 설정 금지
집 구조 설정 금지
결말 설정 금지
연애 전개 강제 금지
최종 흑막 정체 확정 금지
```

이 문서는 오직 세계관의 배경, 세력, 조직 관계, 금지 규칙, 공통 운영 방식만 다룬다.

---

# 1. 세계 구조

작품의 세계는 두 층으로 나뉜다.

```text id="gk2u8w"
表世界 = 일반인이 사는 현대 도시
里世界 = 풍수, 문파, 妖物, 古阵, 禁术이 숨은 세계
```

일반인은 대부분 里世界의 존재를 모른다.
하지만 오래된 집, 유품, 흉가, 풍수 문제, 이상한 물건 등을 통해 가끔 里世界와 접촉할 수 있다.

---

# 2. 里世界 기본 원칙

里世界의 가장 기본 원칙은 **일반인 앞에서 대놓고 드러나면 안 된다**는 것이다.

```text id="c7qvdq"
술법 공개 금지
妖物 정체 공개 금지
문파 충돌 공개 금지
古阵 존재 공개 금지
禁术 흔적 방치 금지
昆仑山 정보 공개 금지
```

일반인에게 노출되면 사건은 커지고, 镇灵司가 개입한다.

---

# 3. 증거 규칙

현대 배경이므로 CCTV, 휴대폰, 사진, 녹음이 존재한다.

하지만 里世界 사건은 증거가 완벽하게 남지 않는다.

```text id="rv3bwc"
화면이 깨짐
소리가 잡음으로 변함
중요 순간만 누락됨
그림자가 이상하게 찍힘
妖物은 흐릿하게 보임
古阵은 사진에 잘 안 남음
```

이 규칙은 세계관 유지용이다.
스토리 사건을 강제로 만들기 위한 규칙이 아니다.

---

# 4. 镇灵司

## 4-1. 기본 정체

镇灵司는 정부 측 비밀 관리 조직이다.

```text id="oidnql"
일반인 보호
사건 은폐
문파 감시
禁术 추적
위험 물건 회수
妖物 관련 사건 정리
里世界와 表世界의 충돌 조정
```

镇灵司는 싸움만 하는 조직이 아니다.
주 역할은 감시, 조사, 은폐, 조정이다.

---

## 4-2. 한계

镇灵司는 강하지만 모든 세력을 완전히 통제하지 못한다.

```text id="ju5lvt"
오래된 문파를 완전히 지배하지 못함
昆仑山 내부를 통제하지 못함
无相会의 배후를 완전히 파악하지 못함
白鹿阁의 정보망을 완전히 장악하지 못함
沧水楼의 거래망을 모두 막지 못함
```

镇灵司는 里世界의 질서를 유지하려 하지만, 절대 권력은 아니다.

---

# 5. 昆仑山

## 5-1. 기본 정체

昆仑山은 白清檀의 上古王域이다.

```text id="n33x41"
昆仑山 = 上古族群의 王域
白清檀 = 王域의 女王
```

昆仑山은 일반 산이 아니다.
오래된 왕역이며, 외부 세력이 쉽게 들어갈 수 없는 금지 영역이다.

---

## 5-2. 접근 규칙

```text id="xjlkba"
일반인 접근 불가
문파도 마음대로 진입 불가
镇灵司도 내부 통제 불가
세력들은 탐내지만 직접 진입하기 어려움
과거 침입 시도는 대부분 실패 또는 실종
```

白清檀은 女王이기 때문에 昆仑山에 들어갈 수 있다.
女王印은 입장키가 아니라 왕권과 기억 회복의 상징이다.

---

## 5-3. 세력들이 탐내는 이유

여러 세력은 昆仑山을 탐낸다.

```text id="izkdig"
古阵
희귀 약재
상고 왕실 기록
龙脉
봉인된 물건
상고族群의 비밀
왕역 내부 자원
```

하지만 구체적인 내부 구조와 비밀은 지금 확정하지 않는다.
필요할 때만 추가한다.

---

# 6. 玄衡门

## 6-1. 기본 정체

玄衡门은 현대식 풍수 문파다.

겉으로는 풍수 컨설팅 회사, 지리 자문, 부동산 풍수, 기업 풍수 등을 하는 합법 사업처럼 보인다.

```text id="4h76et"
본산
도시 분당
풍수 컨설팅
고액 의뢰
문파 체계
罗盘과 규칙 중심
```

---

## 6-2. 조직 구조

```text id="tzfyrk"
门主
长老
分堂主
执事
弟子 / 勘宅师
跑堂弟子
```

본산은 玄衡山에 있다.
도시마다 분당 또는 지부를 둘 수 있다.

---

## 6-3. 성격

玄衡门은 체계와 규칙을 중시한다.

```text id="yxp7le"
문파 명예 중시
돈 되는 의뢰 중시
罗盘과 이론 중시
정식 절차 중시
외부 풍수사에 대한 견제 가능
```

陆沉舟와 대비되는 세력이다.

```text id="el4y7e"
玄衡门 = 체계, 罗盘, 문파 규칙
陆沉舟 = 현장, 직감, 사람 기색, 빠른 판단
```

---

# 7. 白鹿阁

## 7-1. 기본 정체

白鹿阁은 정보 조직이다.

```text id="c543u6"
정보 수집
정보 검증
정보 판매
비밀 기록 관리
문파 동향 추적
禁术 흔적 추적
妖物 정보 수집
```

白鹿阁은 선악이 뚜렷한 조직이 아니다.
중립을 표방하지만 정보로 이익을 얻는다.

---

## 7-2. 정보 등급

```text id="capfh5"
流言 = 확인 안 된 소문
准信 = 어느 정도 확인된 정보
铁信 = 白鹿阁이 보증하는 확실한 정보
禁信 = 아무에게나 팔지 않는 금지 정보
```

白鹿阁의 정보는 비싸지만 신뢰도가 높다.

---

## 7-3. 관계

```text id="ycym55"
镇灵司는 白鹿阁 정보를 참고하지만 완전히 믿지 않음
玄衡门은 白鹿阁에서 세력 동향 정보를 살 수 있음
沧水楼은 물건 거래를 위해 정보가 필요함
无相会 관련 정보는 禁信 취급
昆仑山 내부 정보는 거의 없음
```

---

# 8. 沧水楼

## 8-1. 기본 정체

沧水楼은 里世界의 물건 거래 조직이다.

```text id="br7c92"
약재
符材
古器
罗盘
镇宅物
고서
봉인물
이상한 유품
위험한 물건
```

白鹿阁이 정보를 다룬다면, 沧水楼은 물건을 다룬다.

```text id="c2hhb9"
白鹿阁 = 정보
沧水楼 = 물건
```

---

## 8-2. 물건 등급

```text id="jlrvp5"
正货 = 출처가 깨끗한 물건
灰货 = 출처가 애매한 물건
禁货 = 거래 제한 물건
死货 = 팔면 안 되는 위험 물건
```

沧水楼은 겉으로는 거래 질서를 지키지만, 일부 지부나 개인 거래는 위험할 수 있다.

---

## 8-3. 관계

```text id="cqzfj0"
玄衡门은 沧水楼의 주요 고객이 될 수 있음
白鹿阁과는 정보와 물건으로 협력 가능
镇灵司는 沧水楼을 감시함
无相会와 공식 거래는 금지
昆仑山 자원에는 관심 있음
```

# 9. 无相会

## 9-1. 기본 정체

无相会는 里世界의 어둠 조직이다.

기존 赤霄门 설정은 사용하지 않는다.
최종 이름은 无相会로 고정한다.

```text id="j8m2ps"

风水 오염
夺运
断脉
养鬼
魂瓶
鬼契
污阵
비밀 자금
가짜 신분
```

---

## 9-2. 기원

无相会는 과거 외부 침략 시대의 잔재와 관련이 있다.

하지만 실제 국가명은 직접 쓰지 않는다.

사용 가능한 표현:

```text id="wcgnav"
外境旧势力
旧侵势力残部
海外黑资
跨境黑财团
无国界秘社
旧战时期留下的黑影
```

실제 국가명을 직접 언급하지 않는다.

---

## 9-3. 활동 방식

无相会는 정면으로 드러나지 않는다.

```text id="uo6l19"
가짜 회사
해외 재단
민속 연구소
고서 거래상
부동산 개발사
장례 관련 업체
생물 연구소
古物 수집 조직
```

겉으로는 정상 사업처럼 보일 수 있다.

---

## 9-4. 사용하는 禁术

```text id="fhxvid"

养鬼术 = 鬼物을 기름

污阵 = 정상 阵法을 오염시킴

```

---

## 9-5. 세력 관계

```text id="noecvf"
镇灵司 = 추적하지만 완전히 잡지 못함
玄衡门 = 공식적으로는 적대
白鹿阁 = 정보를 추적함

昆仑山 = 탐내지만 직접 진입 불가
陆沉舟 = 风水 윤리상 정반대
```

# 11. 里世界 공통 사건 등급

사건은 위험도에 따라 나뉠 수 있다.

이것은 스토리 전개를 정하는 것이 아니라, 세계관 내부 위험 분류 기준이다.

```text id="k4the4"
一级 = 단순 흉가 / 민간 금기 / 이상한 물건
二级 = 풍수 문제 / 路冲 / 穿堂煞 / 집안 운 문제
三级 = 鬼物 / 妖气 / 사망자 유품 이상 현상
四级 = 人为布局 / 夺运 / 借运 / 污阵 / 鬼契
五级 = 古阵 / 棺局 / 昆仑山 관련 / 上古族群 흔적
```

이 등급은 강제 전개용이 아니라, 사건 위험도를 설명할 때만 사용한다.

# 12. 돈과 거래 규칙

里世界도 결국 돈과 거래로 움직인다.

```text id="a7kt5f"
일반 의뢰비 = 현금 / 계좌 / 계약서
白鹿阁 정보비 = 돈 또는 정보 교환
沧水楼 물건값 = 돈, 물건, 빚, 인맥
玄衡门 의뢰비 = 고액 컨설팅 비용
镇灵司 = 공식 예산과 비밀 예산
无相会 = 해외 흑자금 / 가짜 회사 / 불법 자금
```

특별한 화폐를 만들지 않는다.
현대 도시 배경이므로 돈, 계약, 인맥, 빚으로 충분하다.

# 14. 세계관 최종 요약

```text id="wlr0l4"
表世界와 里世界는 분리되어 있다.
里世界는 일반인 앞에 드러나면 안 된다.
镇灵司는 사건 은폐와 질서 유지를 담당한다.
玄衡门은 현대식 풍수 문파다.
白鹿阁은 정보 조직이다.

无相会는 禁术과 풍수 오염을 쓰는 어둠 조직이다.
无相会 뒤의 더 큰 배후는 아직 정체를 확정하지 않는다.
昆仑山은 白清檀의 上古王域이며 아무나 들어갈 수 없다.

```
