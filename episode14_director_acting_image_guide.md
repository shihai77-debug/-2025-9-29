# 《棺中檀色》第十四集｜倒棺之路 — 감독형 연기 지도 이미지 분할표

## 사용 목적

이 문서는 사용자가 제공한 제14화 분진 자료를 **장면·대사·음향·감정·도구를 정확히 유지**한 상태로, 이미지 생성용 감독형 연기 지도 프롬프트로 분할한 제작표다.

- 15秒限制修正版 기준으로 총시각을 **约5分00秒 / 300秒**로 재정렬하고, CUT-10 이후 초과 컷은 A/B로 분할한다.
- 원본 대사는 한 글자도 임의로 바꾸지 않는다.
- 원본 장면 묘사, 도구, 소품, 음향, BGM 유무, 정적, 정지 시간, 압흑 지시를 모두 기억하되, 플랫폼 제한 때문에 긴 컷은 내용 삭제 없이 분할한다.
- 매 CUT은 **开始图 / 过程图 / 结束图**로 나누어 연기 지도 이미지를 만든다.
- 각 CUT의 마지막 분할은 다음 CUT 첫 이미지의 시작 상태로 이어지게 한다.
- 이미지 생성은 한 번에 전체를 넣지 말고, `CUT-01-A → CUT-01-B → CUT-01-C → CUT-02-A` 순서로 한 장씩 만든다.

## 인물 삼면도 절대 준수 — 얼굴·옷 100% 고정

이미지 생성 시 인물이 나오는 모든 분할은 **반드시 제공한 인물 삼면도와 같은 얼굴, 같은 머리, 같은 의상, 같은 신발, 같은 체형 비율**로 만들어야 한다. “비슷한 분위기”가 아니라 **같은 인물**이어야 한다.

```text
角色三视图是最高优先级参考。必须一比一保持三视图中的脸型、五官比例、眼睛形状、发型、发量、服装结构、衣领、袖口、腰带、裤裙长度、鞋子、身体比例。只允许改变动作、表情、视线、光影、湿润程度和镜头角度。禁止换脸，禁止换发型，禁止换衣服，禁止换鞋，禁止改变年龄，禁止改变体型，禁止重新设计角色。
```

实际输入顺序：

```text
1. 对应人物三视图
2. 上一张连续性参考图
3. 本CUT分割提示词
4. 禁止换脸/换衣/换发型/换鞋/改变体型
```

## 절대 제작 원칙

```text
分镜资料必须准确表达：台词准确、场景描写准确、道具准确、人物关系准确、镜头角度准确、停顿准确。不要擅自删台词，不要改台词，不要改道具，不要改人物服装，不要改场景逻辑。
```

1. **开头6秒只用旁白**: CUT-01에는 말하는 얼굴을 만들지 않는다. 黑门闭合과 黑场만 만든다.
2. **中段不用旁白**: CUT-02 이후에는 원본에 있는 대사와 음향만 쓴다.
3. **核心感情靠演员**: 白清檀은 대성통곡 금지. 손가락, 눈, 호흡, 멈춤으로 표현한다.
4. **陆沉舟全程不喊**: 모든 선언은 낮고 평평하게,越平越强.
5. **黑七保持黑狐本体**: 사람형, 狐妖青年, 狼人 금지. 말은 狐声旁白式发声으로 처리한다.
6. **最后一只手极简**: CUT-26의 苍白手는 공격하지 않고, 특수효과·BGM·음향 없이 棺沿만 붙든다.

## 15秒限制修正说明

- 기준：每个CUT最大15秒。
- 처리：15秒 초과 컷은 A/B로 분할。
- 총 길이：约5分00秒。
- 안전 기준：所有CUT均不超过15秒，最长CUT为12秒。

## 15秒限制最终确认

- 总时长：约5分00秒。
- 所有CUT均不超过15秒。
- 最长CUT：12秒。
- CUT-11之后实际生成CUT数：23个。
- 如果平台最大15秒，所有18秒、19秒、20秒CUT必须拆开，不要硬塞。
- 硬塞会导致：台词赶、表情不到位、Lip sync乱、镜头动作断、情绪没有重量。
