# 《棺中檀色》第十五集 — 영상 제작용 감독형 연기 지도 이미지 분할표

## 사용 목적

이 문서는 사용자가 제공한 제15집 분진 컷을 기준으로, **모든 CUT 15초 이하 / 총 17컷 / 약 3분19초** 구조를 그대로 유지한 이미지 생성용 감독형 제작표다.

- 원본 대사, 시간축, 감정 이유, BGM, 음향, 화면 묘사를 임의로 바꾸지 않는다.
- 각 CUT은 플랫폼 15초 제한에 맞춰 그대로 사용한다.
- 각 CUT 내부는 **开始图 / 过程图 / 结束图**로 분할해 한 장씩 생성한다.
- 이전 CUT 마지막 이미지가 다음 CUT 첫 이미지가 되도록 인물 위치, 시선, 조명, 도구, 공간을 이어간다.
- 명대사만 크게 만들지 말고, 사용자가 지정한 “사람 냄새” — 기침, 농담, 회피, 손사래, 어색한 걷기 — 를 반드시 살린다.

## 고정 지시 적용

이 문서는 반드시 [`FIXED_DIRECTOR_GENERATION_RULES.md`](FIXED_DIRECTOR_GENERATION_RULES.md)의 **角色三面图锁定 + 连续性演技分镜生成规则**을 최상위 규칙으로 따른다.

- 역할 삼면도 / 정장도 우선순위가 가장 높다.
- 감독 참고도는 동작, 카메라, 표정, 리듬만 참고한다.
- 감독 참고도나 배경 참고도의 임시 인물 얼굴·옷은 최종 캐릭터 외형으로 쓰지 않는다.
- 모든 CUT은 이전 CUT 종료 상태와 다음 CUT 연결 상태를 명확히 유지해야 한다.

## 인물 삼면도 절대 준수 — 얼굴·옷 100% 고정

이미지 생성 시 인물이 나오는 모든 분할은 **반드시 제공한 인물 삼면도와 같은 얼굴, 같은 머리, 같은 의상, 같은 신발, 같은 체형 비율**로 만들어야 한다. “비슷한 분위기”가 아니라 **같은 인물**이어야 한다. 檀缺은 특히 白清檀과 섞이면 안 된다.

```text
角色三视图是最高优先级参考。必须一比一保持三视图中的脸型、五官比例、眼睛形状、发型、发量、服装结构、衣领、袖口、腰带、裤裙长度、鞋子、身体比例。只允许改变动作、表情、视线、光影、湿润程度和镜头角度。禁止换脸，禁止换发型，禁止换衣服，禁止换鞋，禁止改变年龄，禁止改变体型，禁止重新设计角色。檀缺和白清檀必须保持两套完全不同的脸与服装。
```

实际输入顺序：

```text
1. 对应人物三视图
2. 上一张连续性参考图
3. 本CUT分割提示词
4. 禁止换脸/换衣/换发型/换鞋/改变体型
```

## 15초 제한 제작 원칙

```text
每个CUT最大15秒。不要压缩台词，不要删减人物反应，不要硬塞。剧情不断，CUT拆开；情绪不断，镜头拆开；一句话一个重心；一个CUT只完成一个主要情绪。
```

- 총 길이：约3分19秒。
- 총 CUT：17컷。
- 모든 CUT：15秒 이하。
- 핵심 방식：명대사 앞뒤에 약한 말, 숨, 회피 동작, 일상 대사를 넣어서 카피 나열 느낌을 없앤다.

## 삼면도 최우선 인물 고정

이미지 생성 시에는 해당 인물 삼면도와 직전 이미지를 함께 넣는다. 삼면도는 인물 고정용이고, 직전 이미지는 연결용이다.

```text
严格以随附角色三视图为最高优先级。人物脸型、五官、发型、服装结构、鞋子、身体比例必须与三视图一致。本镜头只允许改变姿势、表情、湿润程度、光影和镜头角度，不允许重新设计人物。
```

### 陆沉舟

```text
陆沉舟必须与三视图一致：年轻男性，黑色微乱短发，黑色短款皮夹克，黑色T恤，深色长裤，黑色军靴，冷静克制，低声平稳。禁止暗青色长风衣、长袍、甲胄、修仙化、古装化、换脸、换发型。
```

### 白清檀

```text
白清檀必须与三视图一致：年轻女性，长黑发，黑色旗袍式长裙或黑色长裙，黑色低跟鞋，清冷疲惫但逐渐有活人重量。禁止白衣、白色广袖古装、女鬼化、半透明幽魂化、大哭、夸张崩溃。
```

### 檀缺

```text
檀缺必须严格参考本轮上传的檀缺三视图：年轻女性，苍白清冷的脸，灰色眼睛，长黑发，半披半束的高马尾/半扎发，额前与脸侧有自然碎发，白色交领内衫，白色短外衫，浅灰白腰封与垂带，白色层叠长裙，浅灰白短靴，整体是久困棺中后刚苏醒的苍白感与疲惫感。她可以穿白衣，因为她是新出现的棺中女子，不要与白清檀混淆。禁止把檀缺画成白清檀本人，禁止换成黑裙，禁止女鬼化飘浮，禁止半透明幽魂化，禁止妖化，禁止过度仙气，禁止改成现代装。
```

## 实际生成顺序

1. 先放对应角色三面图：本CUT出现谁就放谁。
2. 再放上一张生成图作为连续性参考。
3. 复制该CUT对应分割的 **生成提示词**。
4. 一次只生成一个分割图。
5. 检查：人物是否与三面图一致、檀缺和白清檀是否区分清楚、黑七是否仍是黑狐、道具是否正确、台词情绪是否不抢。
6. 用通过检查的图继续下一分割。

# CUT-01｜00:00–00:14｜全景→极近｜棺开，檀缺第一句话

**原始资料锁定**：0-6秒黑链绷直→崩断。倒悬的棺缓缓翻正，棺盖打开。6-10秒里面：白衣女子，灰眼。她缓缓抬头，看向白清檀。10-14秒对视，无声。BGM低频爆发→棺开时戛然停。音效锁链崩断→棺翻转沉响→死寂。

**台词**

```text
檀缺（沙哑、慢，先咳了一下才出声）：（咳）……终于。
檀缺（缓过来，自言自语）：嗓子……都锈了。
```

**为什么**：千年第一句不该是金句。先是一声咳，一句废话。人味就出来了。

# CUT-02｜00:14–00:25｜极近，两张脸｜你被丢掉的那一半

**原始资料锁定**：0-4秒白清檀盯着她，戒备。4-11秒檀缺打量她，像在看一件旧东西。BGM低弦一根线。音效安静。

**台词**

```text
白清檀（绷着）：你是谁。
檀缺（看了她半天，才慢慢说）：你被丢掉的……那一半。
```

**为什么**：檀缺先「看半天」再答。不抢着甩金句，让那句话慢下来。

# CUT-03｜00:25–00:37｜近景，过肩｜夜里不冷？

**原始资料锁定**：0-5秒白清檀皱眉，想反驳。5-12秒檀缺没等她，先开口，语气很随意。BGM低线添暖，音效安静。

**台词**

```text
白清檀：我没缺什么。
檀缺（随口，不强调）：是吗。那你夜里……不冷？
```

**为什么**：不说狠话「那是我替你冷的」。改成随口一问「不冷？」。轻，反而扎。

# CUT-04｜00:37–00:48｜近景｜还你，会疼，忍着点

**原始资料锁定**：0-4秒白清檀没接话——她确实冷。4-8秒檀缺抬手，掌心一点白光浮起。8-11秒她看着光，语气平常。BGM暖线延续，白光极轻嗡鸣。

**台词**

```text
檀缺（像在说一件家常事）：还你。会疼，忍着点。
```

**为什么**：「忍着点」是大白话。比「会很疼」更像活人说的。

# CUT-05｜00:48–00:59｜近景｜你呢？

**原始资料锁定**：0-5秒白清檀盯着檀缺的脸，不看光。5-11秒她问。檀缺笑意淡下去。BGM暖线，音效安静。

**台词**

```text
白清檀：你呢？
檀缺（顿，避开她眼睛）：……我？无所谓。散了就散了。
```

**为什么**：檀缺「避开眼睛」+「无所谓」。嘴上轻，身体在躲。这就是人。

# CUT-06｜00:59–01:09｜极近，白清檀脸｜我不想让你散

**原始资料锁定**：0-4秒白清檀沉默。4-7秒她开口，没有很用力。7-10秒切檀缺——怔住。BGM这句前抽掉半拍；檀缺怔住时轻轻接回。音效半拍静默。

**台词**

```text
白清檀（很平，不煽）：我不想让你散。
```

**为什么**：这一句独占镜头。前后不许有别的台词压它。它本身够强，不用包装。
